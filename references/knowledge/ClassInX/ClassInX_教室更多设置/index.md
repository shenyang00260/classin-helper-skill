# ClassInX 教室更多设置


# 一、显示/隐藏座位席区域


## 使用说明


座位席区域是指黑板中最上方的一行区域，该区域仅用于展示师生的摄像头画面，不支持书写。


除了在创建课堂时设置是否显示座位席区域，进入教室后，授课教师或联席教师还可以在教室的更多设置中选择显示或隐藏座位席区域。


另外只有座位席区域显示时，教师设置的座位席轮播才生效。当教师需要全屏书写时，可隐藏座位席区域。


## 具体操作步骤


在屏幕中央底部点击或上滑进入dock栏


点击最右侧的 设置


点击 更多设置


点击 常规


选择 显示 或 隐藏 座位席区域

# 二、专注模式


## 适用人员


授课教师、联席教师


避免学生上课时分心，建议使用专注模式功能。后台管理人员可以将勾选【开启专注模式】功能的皮肤应用到班级，该班级下的正式教室将默认开启专注模式；除此之外，授课教师或联席教师进入教室后，也可在教室的更多设置中开启或关闭专注模式。


开启专注模式后：若学生使用 Windows 电脑进入教室，教室窗口将保持最大化并置顶，不被其他窗口遮挡。若学生使用手机或平板进入教室，切换到其他应用时会自动退出教室。


授课教师或联席教师在教室内设置的专注模式仅对当前教室有效。


## 注意事项


专注模式对使用 Mac OS设备的学生不生效。


专注模式对旁听生不生效。


若学生设备支持分屏或类似功能，专注模式也可能无法生效。


开启 或 关闭 专注模式

# 三、举手声音提示


## 适用场景


当学生在教室内举手时，教师设备会自动播放举手提示音，便于教师在授课、板书或查看文件时，及时发现学生的互动请求，不错过课堂互动。


进入教室后，授课教师和联席教师可分别设置自己的学生举手提示音，设置互不影响。


在举手提示音设置中，教师可选择无提示音，或选用叮音/滴答音/叮咚音作为提示音。


提示音仅教师本人可听见，教室内学生不会听到。


选择 举手提示音

# 四、进出教室通知


进入教室后，教师在教室的更多设置中开启进出教室通知后，每当有教室的学生进入或离开教室时，教师的屏幕底部都会弹出提醒告知，便于教师实时掌握学生出勤状态。


授课教师和联席教师可分别设置自己的进出教室通知，设置互不影响。


在授课教师开启该功能的前提下，联席教师的进入或离开，授课教师也会收到提醒。


开启 或 关闭 进出教室通知

# 五、教室背景


教室背景可以提前在后台管理系统中设置，进入教室后，教师也可以在教室的更多设置中自行设置。教师在教室中更换完教室背景后，新换的教室背景会立即生效。


教师可选择系统自带的背景图，或上传本地图片作为教室背景图片。上传图片须为 JPG 格式，且大小不超过 1 MB。


教师在教室更多设置中更换的教室背景仅对当前教室生效。


点击 教室背景


选择 或 上传 教室背景图片

# 六、教室窗口布局


将摄像头、课件及小工具拖动至预设位置并自动调整大小，可帮助教师快速整理教室黑板界面、高效切换教学场景，更专注于授课与互动。


进入教室后，教师可通过教室的更多设置 > 其他 > 窗口布局的路径，选择已有布局模板。也支持自定义添加布局模板，添加布局模板中支持添加平铺模式（窗格尺寸和位置都是固定的无法自由调整），或自由窗格模式（自由调整窗格尺寸和位置）。


选定模板后，教师如果使用大屏可以用单指拖动摄像头、课件或小工具，停留 0.5 秒即可进入布局模式下；如果使用电脑，用鼠标拖动的同时按住键盘的Shift键，即可进入布局模式下。


点击 其他


在布局设置下点击 窗口布局


选择 或 添加 布局模板


单指拖动摄像头 / 课件 / 小工具时，停留 0.5 秒即进入布局模式


（如果使用电脑，拖动的同时按住shift键）

# 七、教室求助


授课教师、联席教师、课堂学生（除旁听生）


教室中的老师或学生遇到无法自行解决的问题时，可向后台教务管理老师求助。


进入教室后，教师可通过教室的更多设置 > 其他 > 求助的路径，进入求助窗口。在求助窗口中选择问题用户、描述问题后点击确定即可。后台教务管理老师可在www.eeo.cn的教务管理 > 监课管理中查看课堂求助信息，信息包含问题描述与系统截取的当前教室内所有用户的教室截图。教务老师回复后，教室内教师界面会自动弹出回复窗口。


后台管理人员将勾选【允许学生向学校后台发送求助】功能的皮肤应用到班级，学生使用电脑或ipad进入班级下的教室后，才可以使用求助功能。在求助窗口中，学生描述问题后点击确定即可。后台教务管理老师可在www.eeo.cn的教务管理 > 监课管理中查看课堂求助信息，信息包含问题描述与系统截取的该学生的教室截图。教务老师回复后，教室内该学生界面会自动弹出回复窗口。


学生（旁听生除外）使用电脑或ipad通过ClassInAPP进入勾选【允许学生向学校后台发送求助】功能的教室，才可以使用求助功能。


在快速求助下点击 求助


教师：选择问题用户，描述问题后点击确定


学生：描述问题后点击确定

# 九、教室设置手写笔


在 ClassInX中使用手写笔在黑板上书写时，如果手写笔仅用于书写，那么建议在屏幕正下方的dock栏 > 设置 > 更多设置 >硬件中开启【手写笔仅用于手写】功能。


启用后，手写笔仅用于书写，手指的操作将与圆盘内选择的工具一致（倾斜笔尖可能导致识别为手指）。


点击 硬件


开启 或 关闭 手写笔仅用于手写

# 十、教室触控锁设置


为防止线下学生在大屏的屏幕上误触，教师可在课间休息时在ClassInX软件的屏幕正下方的dock栏 > 设置 > 更多设置 >硬件中开启【锁定屏幕触控功能】。


触控锁打开的情况下，用户可以点击屏幕下方的触控锁按钮或长按翻页笔上一页即可关闭触控锁。


触控锁关闭的情况下，用户也可以长按翻页笔上一页打开触控锁。


开启 或 关闭 锁定屏幕触控功能

# 十一、大可语音助手


大可语音助手可帮助用户通过语音指令在 ClassInX 2.0 大屏上更便捷地完成 ClassIn 相关操作。


语音助手功能仅支持 ClassInX 2.0 大屏设备，使用时需在 ClassInX 软件中开启麦克风权限。


在ClassInX软件的屏幕正下方的dock栏 > 设置 > 语音助手中可以对【大可语音助手】功能开启或关闭。


开启大可语音助手后，还可以设置【免唤醒】（无需说大可大可唤醒，可直接说指令）、【文字反馈】（语音识别以文字形式在大屏幕上显示）、【音效反馈】（大可发出叮进行回应）功能的开启或关闭，这些功能也可通过点击教室或黑板右下角的大可图标，进行开启或关闭。


此外，在正式教室中，授课教师、联席教师及分组组长在满足条件时也可使用大可语音助手功能，符合条件的学生则无此功能权限。


点击 语音助手


开启 或 关闭 大可语音助手
