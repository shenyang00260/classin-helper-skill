# ClassInX 其他常用设置


# 一、触屏大小尺寸


## 适用人员


所有人


## 使用说明


用户在大屏上使用ClassInX，未选中橡皮擦工具，单指操作或双指滑动可能会被误识别为橡皮擦。


出现此情况时，建议用户在教室屏幕正下方的dock栏中进入ClassIn >点击左下角头像 >进入教室设置，找到【触屏大小尺寸】，将其尺寸（实际大屏尺寸相比）调小一些即可。


## 具体操作步骤


在屏幕中央底部点击或上滑进入dock栏


点击 ClassIn


点击左下角 头像


点击 教室设置


进入 触屏大小尺寸，调整尺寸

# 二、开机自动运行


如果用户需要每次启动大屏后，可以自动打开ClassInX软件，那么建议用户在教室屏幕正下方的dock栏中进入ClassIn >点击左下角头像 >进入系统设置，找到【开机自动运行】将其开启即可。反之将其关闭。


点击 教系统设置


开启 或 关闭 开机自动运行

# 三、触控屏自动唤起软键盘


用户在大屏上使用ClassInX进入黑板或教室，如果用户需要打字时可以使用系统自带的软键盘，那么建议用户在屏幕正下方的dock栏中进入ClassIn >点击左下角头像 >进入系统设置，找到【触控屏自动唤起软键盘】将其开启即可。


点击 系统设置


开启 或 关闭 触控屏自动唤起软键盘

# 四、ClassInX 软件日志上报


用户在使用ClassInX软件遇到无法自己解决的难题时，可向 ClassIn 客服提供问题描述、问题账号、发生的时间等信息，并在屏幕正下方的dock栏中进入ClassIn >点击左下角头像 >进入系统设置，在【软件日志上报】后点击上报，以便技术人员高效快速定位问题。


在软件日志上报后，点击 上报

# 五、上报软件助手录音


用户在使用ClassInX 2.0大屏时，使用大可语音助手遇到语音识别异常等问题，，可向 ClassIn 客服提供问题描述、问题账号、发生的时间等信息，并在屏幕正下方的dock栏中进入ClassIn >点击左下角头像 >进入系统设置，在【上报软件助手录音】后点击上报，以便技术人员高效快速定位问题。


在上报软件助手录音后，点击 上报

# 六、ClassInX 清除缓存文件


若用户在ClassInX中打开某个功能页面时出现加载时间过长无法正常显示内容的情况，可在屏幕正下方的dock栏中进入ClassIn >点击左下角头像 >进入文件传输，在【缓存文件】后点击清除，操作清除后点击立即重启软件即可。


点击 文件传输


在缓存文件后，点击 清除


点击 立即重启

# 七、ClassInX 清除数据文件


若用户在ClassInX中，确保排课正常且登录账号正确的情况下，无法查看教室入口，可在屏幕正下方的dock栏中进入ClassIn >点击左下角头像 >进入文件传输，在【数据文件】后点击清除，操作清除后点击立即重启软件即可。


在数据文件后，点击 清除


点击 立即重启 软件
