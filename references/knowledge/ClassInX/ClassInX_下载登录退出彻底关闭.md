# ClassInX 下载/登录/退出/彻底关闭


# 一、下载ClassInX 软件


## 使用说明


ClassInX 软件是一款面向线上、线下、混合式教学的一体化教育工具，核心覆盖三大场景，适配 K12、高等教育、教培机构等多类用户，适用于教师使用。


ClassInX 软件可在官方网站www.eeo.cn下载，支持 Windows 7 及以上操作系统，不兼容其他系统。


## 具体操作步骤


在浏览器中访问 www.eeo.cn


点击进入 下载中心


下滑页面右侧的滚动条，在ClassInX（大屏版）下点击 Windows版下载


下载完成后，双击打开ClassInX的安装包


用户账户控制提示是否允许此应用对你的设备进行更改，点击 是


选择安装使用的语言后，点击 确定


勾选协议后，点击 快速安装 或自定义安装


安装进度完成后，系统会自动进入ClassInX软件中

# 二、登录ClassInX 账号


在ClassInX 中进入课堂、使用云盘文件及创建班级等操作均需登录ClassIn账号才可以使用。


启用ClassInX后，用户可以点击或向上滑动屏幕正下方进入dock栏中，点击左侧的头像进行登录。


登录的方式包括：用手机或平板登录ClassIn后扫码登录ClassInX；输入ClassIn账号（手机号/邮箱）密码登录。


进入教室正下方dock栏中


点击左侧的 头像


扫码或输入ClassIn账号密码 登录

点击 退出登录

# 五、彻底关闭ClassInX 软件


退出登录不代表关闭了ClassInX软件。如果需要关闭ClassInX软件，需要在下方dock栏中点击去桌面，在桌面的任务栏中长按ClassInX窗口来关闭软件。


点击 去桌面


在桌面的下方任务栏中长按ClassInX窗口，点击 关闭窗口
