# 【其他】硬件使用问题


# 1. 翻页笔和接收器重新配对


现象描述：翻页笔无法使用，需要重新配对。


解决方案：对码 长按1键不松手，连续点击2键5次，指示灯闪烁进入对码模式。松开按键1对码结束。

# 2. 翻页笔翻页距离不够远


现象描述：翻页笔翻页距离太近，使用不便。


解决方案：翻页笔的接收器尽量插在大屏侧面USB口上，已确保接收器无遮挡。
