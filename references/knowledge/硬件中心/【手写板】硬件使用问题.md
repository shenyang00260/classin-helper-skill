# 【手写板】硬件使用问题


# 1. 写字异常


现象描述：手写板写字异常。


解决方案：尝试下方任意一种方法以解决此问题。


确认驱动安装：确保已正确安装classintablet驱动。驱动下载地址：https://www.eeo.cn/cn/download （安装完成后，请确保驱动程序在后台最小化运行，切勿关闭）


检查驱动冲突：确认电脑未安装其他手写板驱动程序，例如高漫（Gaomon）、Wacom等。如有安装，请卸载这些驱动以避免冲突


数据线检查：请使用原装数据线或兼容的华为手机数据线进行连接（充电线不可用）

# 3. 驱动显示未连接


现象描述：手写板驱动显示未连接。


尝试更换一个USB接口进行连接


尝试更换一根USB数据线进行连接


如果用户使用的是Mac电脑，可以尝试更换扩展坞来进行连接


# 4. 书写卡顿


现象描述：手写板在书写过程中出现卡顿现象。


解决方案：请下载并安装360驱动大师，使用该工具更新显卡驱动程序，以改善手写板的书写体验。


# 5. 未接触板子就可以写字


现象描述：手写笔在未接触板子的情况下就能书写。


解决方案：这种情况通常是由于手写笔出现故障所致，建议联系Classin售后更换新的手写笔以解决问题。
