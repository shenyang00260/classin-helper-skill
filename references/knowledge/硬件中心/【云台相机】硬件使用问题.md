# 【云台相机】硬件使用问题


# 1. 摄像机影像模糊


现象描述：电脑连接摄像机后，显示的摄像机影像出现模糊。


解决方案：尝试下方任意一种方法以解决此问题。


检查镜头是否有灰尘或其他障碍物，如有，进行清洁


通过遥控器或相机软件调试焦距


使用遥控器或摄像机配套的软件调节摄像机的焦距。建议在查看清晰度时，不要通过ClassIn黑板界面打开摄像机，因为图像可能会被压缩。请使用ClassIn辅助摄像头功能或Windows自带的相机应用程序来查看摄像机的真实清晰度


在电脑上安装iSmart CMS软件，通过调节聚焦功能（聚焦+/-）来调整图像的清晰度

# 2. 摄像机影像有水波纹


问题描述：当电脑/大屏一体机连接摄像机后，显示的摄像机影像出现水波纹现象。


使用遥控器：


通过遥控器进入菜单


依次选择“图像” > “曝光” > “抗闪烁”


更改抗闪烁参数为50Hz或60Hz


针对大屏类型：


如果大屏非ClassinX品牌，并且为点阵屏或使用PWM（脉宽调制）调光技术的屏幕，水波纹现象可能无法完全消除


尝试调高屏幕亮度或调整摄像机的曝光参数，以改善水波纹效果

# 4. 设置预置位


问题描述：摄像机安装完成后，需要设置预置位来提前预设摄像机的位置、焦距和图像等参数。


解决方案：尝试下方方法以解决此问题。


CAM34摄像机：


设置预置位：按下数字键后再按“设预置”键


调用预置位：按下对应的数字键+“CALL”键


清除预置位：按下对应的数字键+“CLEAR”键


CAM37摄像机：


设置预置位：按下数字键后再按“SET”键


P1摄像机：


设置预置位：长按数字键，待相机指示灯闪烁一次即可完成设置

# 5. 通电后，电源指示灯不亮


问题描述：摄像机通电后，电源指示灯没有亮。


解决方案：如果摄像机在通电后没有反应，且电源指示灯不亮，可能存在硬件故障，需要联系Classin硬件售后进行维修或更换。


# 6. 复位预设位置


问题描述：需要调整摄像机预设位置。


解决方案：尝试下方任意方法以解决此问题。


直接按下对应的数字键即可将摄像机调回到预设位置


首先按下对应的数字键，然后按下CALL键，即可将摄像机调回到预设位置


# 7. 不显示画面（CAM34）


现象描述：CAM34摄像机的POWER蓝灯和STANDBY绿灯同时亮起，但屏幕无画面显示。


解决方案：当CAM34摄像机的POWER蓝灯和STANDBY绿灯同时点亮时，表示摄像机正在启动过程中。请耐心等待约20秒，直到仅POWER蓝灯亮起，这表明摄像机已成功启动。此时，画面才会正常显示。


# 8. 画面花屏（CAM37）


现象描述：CAM37摄像机画面出现花屏现象。


首先确认摄像机与电脑之间的连接线类型，通常为HDMI线或SDI线


如果使用的是HDMI线或SDI线，尝试更换一条新的HDMI线或SDI线


如果更换连接线后问题依然存在，建议使用摄像机自带的Mini USB线进行连接


# 9. 不显示画面（CAM37）


现象描述：连接CAM37摄像机后没有显示画面。


解决方案：尝试下方任意一种方法以解决此问题


检查连接线：确保CAM37摄像机与电脑之间的连接线正确且牢固连接。请确认数据线没有损坏或松动


确认电源连接：确保CAM37摄像机的电源已正确接入，并且电源指示灯正常亮起


尝试重启：使用遥控器上的开关机键，尝试重新启动摄像机


# 10. 不显示画面（CAMP1）


现象描述：CAMP1摄像机的绿灯和红灯同时亮起，但屏幕无画面显示。


解决方案：当CAMP1摄像机的绿灯和红灯同时点亮时，表示摄像机正在启动过程中。请耐心等待约20秒，直到红灯熄灭后，这表明摄像机已成功启动。此时，画面才会正常显示。


# 11. 关闭自动变焦


现象描述：需要关闭摄像机的自动变焦功能


找到摄像机的遥控器


按下遥控器上的“手动”按钮


# 12. 调整摄像机亮度（CAM34）


现象描述：需要调整摄像机的画面亮度


解决方案：调整CAM34摄像机的亮度，请按照以下步骤操作


按下CAM34摄像机遥控器上的“菜单”按钮


在电脑屏幕上会弹出摄像机的设置界面


在设置界面中，选择并进入“曝光”栏目


在“曝光”栏目内，找到并调整“亮度补偿”选项，直到达到满意的亮度效果


# 13. 调整摄像机焦距


现象描述：需要调整摄像机的焦距数值


使用遥控器中的“-”键或“+键”调整焦距大小


# 13. USB摄像机无法识别


现象描述：摄像机安装后，大屏无法识别此硬件


重新插拔USB电脑端接口


重新启动摄像机，最好断电重启下摄像机


打开电脑设备管理器，将“未知USB设备”卸载后，重新加载安装


如果以上方法都没有解决，考虑USB线材故障原因
