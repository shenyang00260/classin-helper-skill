# 【无线麦克风】硬件使用问题


# 1. 充电盒充不上电（猛犸LARK150）


现象描述：猛犸LARK150无线麦克风（小蜜蜂）充电盒无法充电。


解决方案：按照以下步骤进行复位操作。


将充电盒连接至充电器


使用针状物分别按压充电盒、麦克风和接收器上的重置小孔，长按5秒钟


将麦克风和接收器从充电盒中取出，单独对充电盒进行充电30分钟


30分钟后，将麦克风和接收器重新放回充电盒中，此时应能看到设备的电量显示，充电盒将恢复正常充电功能

# 2. 如何接线？（猛犸LARK150、M1、罗德）


现象描述：麦克风线路连接问题。


解决方案：按照以下步骤进行操作。


无线麦克风与大屏一体机连接（使用TRS转TRS线进行连接）


一端连接无线麦克风（小蜜蜂）


另一端连接大屏一体机OPS主机上的红色麦克风音频插口


无线麦克风与笔记本电脑连接（使用TRS转TRRS线进行连接）


TRS端连接无线麦克风（小蜜蜂）


TRRS端连接笔记本电脑上的3.5mm音频插口


无线麦克风与手机或平板连接（使用TRS转Type-C线进行连接）


另一端连接手机或平板上的Type-C插口


无线麦克风与USB声卡连接（使用TRS转TRS线进行连接）


另一端连接USB声卡上的红色麦克风音频插口

# 4. 听不清楚声音（猛犸LARK150、猛犸M1、罗德）


现象描述：使用猛犸LARK150、猛犸M1或罗德无线麦克风（小蜜蜂）时，发现声音较低且听不清楚。


解决方案：尝试下方任意一种方法以解决此问题。


调整设备自身音量：首先，请检查并调高您的无线麦克风设备自带的音量设置。确保设备音量处于适宜的水平


调节ClassIn教室麦克风音量：进入ClassIn教室后，找到麦克风音量设置，并适当调高麦克风音量，以确保声音清晰可闻（参考下图）

# 5. 续航时间


现象描述：了解猛犸M1的发射器、接收器、充电盒续航时间。


问题答复：


猛犸M1无线麦克风：


发射器和接收器在完全充电后均可连续使用8小时


配套的充电盒可为发射器和接收器各提供两次完整充电，确保长时间使用需求


猛犸LARK 150无线麦克风：


发射器在满电状态下可连续使用4小时


接收器在满电状态下可连续使用7小时


充电盒在充满电后，可为两个发射器和一个接收器充电2.5次


需要注意的是，和手机一样，在低温环境下使用时，设备的续航时间可能会缩短。此外，不同的使用距离也会影响发射器和接收器的续航时间

# 7. 三种工作模式的效果？（猛犸LARK150）


猛犸LARK150具有三种工作模式，每种模式的效果如下：


立体声模式：当使用两个麦克风（MIC1和MIC2）采集音频时，接收端会将MIC1的音频输出至左声道，将MIC2的音频输出至右声道。这样，用户可以清晰地分辨出两个麦克风所采集到的不同声音。


单声道模式：在此模式下，两个麦克风（MIC1和MIC2）采集的音频会被合并，接收端将合并后的音频输出至左右声道。这样，左右声道的音频内容完全一致，形成单声道音频。


安全音轨模式：此模式下，接收端会将MIC1和MIC2的音频合并处理，但左声道和右声道的音量有所不同。左声道为正常音量，右声道的音量比左声道低6dB。这样可以在录音过程中提供一个备份音轨，以防止因音量过高导致的音频失真，从而确保音频质量和用户隐私的双重保障。


# 8. 安全音轨模式有什么用？


在进行现场录音时，可能会遇到音量突然增大的情况，这种过大的声音会导致录音文件失真或无法使用。为了应对这种情况，可以使用猛犸LARK150的安全音轨模式。此模式能够在录音时生成双声道音频，其中左声道录制正常音量的音频，右声道录制较低音量的音频。这样，即使左声道的音频出现爆音或失真，右声道的音频仍然可以作为备用，从而确保录音素材不至于完全不可用。


# 9. 发射器和接收器配对


启动发射器配对模式：长按发射器上的黄色按钮约3秒钟，直至蓝色指示灯快速闪烁。


启动接收器配对模式：长按接收器上的“+”按钮约3秒钟，直至蓝色指示灯快速闪烁。


等待配对完成：当发射器和接收器的蓝色指示灯变为常亮状态时，表示配对成功。


猛犸LARK150无线麦克风：


使用猛犸LARK150进行配对时，只需将发射器和接收器放入充电盒并打开电源，它们会自动完成配对，无需任何额外操作。


# 10. WP1无线麦克风如何锁频和取消锁频


现象描述：需要调整WP1麦克风的频率锁定状态。


问题答复：请按照以下方式操作。


手持麦锁定：


手持麦和接收机配对后，手持端同时按“翻页上”+“翻页下”松手，屏幕显示频道闪烁按“音量加”松手，操作完成。锁定成功后屏幕上方有小锁图标显示


手持麦解除锁定：


同时按“翻页上”+“翻页下”松手，屏幕显示频道闪烁，按音量减”松手，操作完成


接收机锁定：


接收机与手持麦自动配对后，将接收机的三个“信道”拨码开关拨至ON位置，操作完成


接收机解除锁定：


调节接收机“信道”拨码开关，只要三个拨码开关不全部位于ON位置即可


注意：频率锁定设置需要手持麦与接收机都设置成频率锁定模式，否则无法链接配对。
