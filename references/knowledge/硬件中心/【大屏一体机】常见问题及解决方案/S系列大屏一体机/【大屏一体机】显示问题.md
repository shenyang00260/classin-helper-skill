# 【大屏一体机】显示问题


# 1. 调节屏幕亮度


现象描述：使用大屏一体机时，屏幕亮度不合适。


解决方案：请按照以下步骤调节屏幕亮度。


点击一体机的电源键


在安卓弹窗中，手动调整背光亮度

# 2. 屏幕出现跳点/波纹点


现象描述：用户使用大屏一体机时，触控屏幕出现跳点或波纹点现象。（如下图所示）


![](https://cofile.eeo.cn/res-store%2Ff84ad16ff7f4cfbbaa5a9cb77121eb503bf26909f74269c0d6af8a56387a9b55_240988?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1777824000;1778428800&q-key-time=1777824000;1778428800&q-header-list=host&q-url-param-list=&q-signature=7343e5b885b33dd00988c5b4a2d844e6d66b6c10)


解决方案：这个是因为一体机屏幕外凸导致触控模组信号异常，可参考下方修复方式进行操作。


可以用拳头轻轻按压屏幕玻璃中间部位，重复多次，尝试使屏幕恢复平整


若条件允许，可以将一体机底部稍微抬起，使一体机屏幕呈一定的斜角向上倾斜，可能有助于恢复正常触控功能


如果以上方法仍未能解决问题，建议联系Classin售后进行维修


# 3. 屏幕出现线条/画面闪屏


现象描述：大屏一体机屏幕出现线条或画面闪烁现象。（如下图所示）


![](https://cofile.eeo.cn/res-store%2Fa88cee69e4546e56d39acbfdcd678e32f54d1c83c860be5993fca9b0d9c5da28_1217560?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1777824000;1778428800&q-key-time=1777824000;1778428800&q-header-list=host&q-url-param-list=&q-signature=3a15aaad94e6cdaa0b95a42b8190a6c67a2bed30)


解决方案：该问题通常是由屏幕的OC（驱动电路）和TCON（时序控制电路）板故障引起的。建议联系Classin售后进行返厂维修。


# 4. 一体机屏幕出现闪烁


现象描述：开机后，屏幕闪烁，无法正常使用。


解决方案：下载360驱动大师/驱动精灵等，进行显卡驱动升级。


# 5. 一体机画面卡死黑屏


现象描述：一体机开机一直卡死在classin logo页面，或者一直黑屏无法进入到系统页面。


解决方案：此类问题大概率为电源板/主板故障，建议联系Classin售后进行维修。
