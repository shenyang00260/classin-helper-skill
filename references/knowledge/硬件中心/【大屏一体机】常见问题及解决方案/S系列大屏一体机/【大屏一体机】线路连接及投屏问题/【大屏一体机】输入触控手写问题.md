# 【大屏一体机】输入触控手写问题


# 1. 不弹出软键盘


现象描述：使用一体机时，点击系统桌面右下角的“键盘”图标无法弹出软键盘，导致无法正常使用软键盘进行输入。


解决方案：按照以下步骤进行设置。


点击桌面左下角的“Windows”图标，进入“设置”


在“设置”界面中，选择“设备”选项


选择“输入”找到并开启“不处于平板电脑模式且未连接键盘时显示触摸键盘”选项（如默认打开状态可点关闭再次打开重启下电脑）

# 2. 书写卡顿


现象描述：在大屏一体机上使用ClassIn进行录课时，遇到使用教室画笔工具在黑板上书写时出现卡顿的情况。


解决方案：尝试下方任意一种方法以解决此问题。


调整分辨率和显示比例：


将一体机的Windows系统桌面分辨率设置为1920 x 1080


将显示比例调整为150%

清理ClassIn缓存


打开ClassIn主界面，点击左下角的设置选项


进入“系统设置”界面


点击“清理缓存”按钮
