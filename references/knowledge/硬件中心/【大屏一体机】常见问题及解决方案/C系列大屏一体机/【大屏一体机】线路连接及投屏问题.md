# 【大屏一体机】线路连接及投屏问题


# 1. 大屏一体机无线投屏


现象描述：使用大屏一体机时，需要无线投屏。


解决方案：尝试按照下方步骤执行。


将大屏切换到安卓页面


将安卓系统连接到本地无线网


打开安卓自带的“连屏”软件


按页面操作（笔记本、手机、平板）连接同一个局域网，下载客户端即可投屏

# 2. 大屏一体机有线投屏


现象描述：使用大屏一体机时，需要有线投屏。


解决方案：各种连接方法请参考下方。


使用HDMI线将笔记本与大屏一体机连接（大屏一体机上有三个HDMI输入接口，分别位于前面板、左后侧面板的HDMI1和HDMI2，均可使用）


使用USB A-B触控线将笔记本与大屏一体机连接（大屏一体机上有两个触控接口，分别位于前面板和左后侧面板的TOUCH-USB接口，均可使用）


如果没有显示图像，请尝试更换大屏一体机上的HDMI接口


当通过上述有线方式连接大屏和笔记本时，请注意将麦克风和摄像头连接至笔记本，而不是大屏一体机
