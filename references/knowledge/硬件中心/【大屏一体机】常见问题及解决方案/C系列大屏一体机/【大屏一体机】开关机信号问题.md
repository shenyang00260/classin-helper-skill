# 2. 开机后无信号，无法进入win系统桌面


现象描述：大屏一体机开机后，屏幕一直显示“信号连接中”、“无信号”、“NO SIGNAL”状态。无法进入到Windows系统桌面中。


解决方案：尝试下方任意一种方法以解决此问题。


确认OPS主机是否正常通电：


观察一体机背面OPS主机上的“POWER”指示灯来判断：如果“POWER”指示灯亮起，说明电脑已正常通电并处于开机状态；如果“POWER”指示灯未亮，您可以点击下方的POWER按钮进行开机操作（如下图所示）


![](https://cofile.eeo.cn/res-store%2Fc5330ab2d65c784a01eced2109b83aaac63b46ff292cca535a63e8f3f8aee6d0_335015?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=305523f2405a4555ea96e7e48f2e3958d9c31bc0)


尝试切换OPS：


点击大屏一体机屏幕两侧任意一侧的箭头图标


进入“信号源”菜单


选择“OPS”信号源（如下图所示）


![](https://cofile.eeo.cn/res-store%2Fe885df33183296811fc0891ca8a7c7e4d2c4738a552647855c80e6cada75be55_491634?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=1cc5b2b16ebcb64279f800cbbd1dea291a3f0e13)


重启OPS主机：


在一体机背面的OPS主机上找到POWER按钮


长按POWER按钮，直至OPS主机完全关机


再次按下POWER按钮，将OPS主机重新启动（如下图所示）


![](https://cofile.eeo.cn/res-store%2Fe4aa7bd7083d68b45810aacd75542e5361e06cf8c0047915a20a2232eba0d738_339165?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=96dedae8c0304d3c3e646e391f9685e140621660)


一键还原OPS：


点开大屏一体机屏幕中两侧任意一边的箭头


进入“设置”


进入“系统设置”


点击“一键还原OPS”（如下图所示）


![](https://cofile.eeo.cn/res-store%2F9969830755991f4b88bfa5adba7b913959c563bf16ae5db73f6d3e09cfa250f7_1342860?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=fc9c211bbaf554d6f6ed7f3c67ffc59641b188ff)
