# 【大屏一体机】显示问题


# 1. 屏幕显示异常（变绿、变灰、变花）


现象描述：使用大屏一体机时，如果屏幕显示的颜色偏绿、偏灰或偏黄。


解决方案：调整图像模式为“电脑模式”。


通过点击大屏一体机屏幕两侧任意一边的箭头，调出菜单


选择“信号源”选项


进入右下角的“设置”菜单


将图像模式调整为“电脑模式”（如下图所示）

# 2. 调节屏幕亮度


现象描述：使用大屏一体机时，屏幕亮度不合适。


解决方案：请按照以下步骤调节屏幕亮度。


点击屏幕两侧任意一边的箭头图标


进入屏幕右下角的“设置”菜单


切换至“菜单模式”


关闭“自动背光”功能（如下图所示）

# 3. 屏幕出现跳点/波纹点


现象描述：用户使用大屏一体机时，触控屏幕出现跳点或波纹点现象。（如下图所示）


![](https://cofile.eeo.cn/res-store%2Ff84ad16ff7f4cfbbaa5a9cb77121eb503bf26909f74269c0d6af8a56387a9b55_240988?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=c9700c18f74f99fcd95872ccabc21a6817c99fbc)


解决方案：这个是因为一体机屏幕外凸导致触控模组信号异常，可参考下方修复方式进行操作。


可以用拳头轻轻按压屏幕玻璃中间部位，重复多次，尝试使屏幕恢复平整


若条件允许，可以将一体机底部稍微抬起，使一体机屏幕呈一定的斜角向上倾斜，可能有助于恢复正常触控功能


如果以上方法仍未能解决问题，建议联系Classin售后进行维修


# 4. 屏幕出现线条/画面闪屏


现象描述：大屏一体机屏幕出现线条或画面闪烁现象。（如下图所示）


![](https://cofile.eeo.cn/res-store%2Fa88cee69e4546e56d39acbfdcd678e32f54d1c83c860be5993fca9b0d9c5da28_1217560?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=80377e5275748c41ee5dc67e8c3013dc3da01490)


解决方案：该问题通常是由屏幕的OC（驱动电路）和TCON（时序控制电路）板故障引起的。建议联系Classin售后进行返厂维修。


# 5. 屏幕颜色轮转


现象描述：开机时，各种颜色轮流切换。（如下图所示）


![](https://cofile.eeo.cn/res-store%2F2b205c7149fabcbb09e35674db29b8541c4f8008d43925c67ed17dc48e48c0c4_326694?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=3ea5cf946fb05d27d2f68c4f323dd8dd0c953fbd)


解决方案：参考如下方案尝试。


1.依次点击 侧边栏—＞信号源—＞更多—＞菜单模式，关闭自动背光，手动调节背光


2.重启大屏


3.断电重启大屏


4.如果以上方法仍未能解决问题，建议联系Classin售后进行维修。


# 6. 一体机屏幕出现闪烁


现象描述：开机后，屏幕闪烁，无法正常使用。


解决方案：下载360驱动大师/驱动精灵等，进行显卡驱动升级。


# 7. 一体机画面卡死黑屏


现象描述：一体机开机一直卡死在classin logo页面，或者一直黑屏无法进入到系统页面。


解决方案：此类问题大概率为电源板/主板故障，建议联系Classin售后进行维修。
