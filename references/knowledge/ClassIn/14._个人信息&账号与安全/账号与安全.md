# 账号与安全


# 一、查看账号


### 适用人员


所有人


### 适用场景


如果用户在使用ClassIn时看不到上课的教室或活动，请检查一下是否登录了正确的账号。如果账号登录错误会导致无法看到上课的教室或活动。

当用户有多个账号时，可以进入账号与安全查看当前登录的账号，便于管理和使用。

# 二、绑定或修改手机号


当用户用邮箱注册ClassIn，那么完成账号注册后，需要绑定一个手机号，便于以后用手机号登录。


当用户的手机号不再使用时，用户可以在ClassIn中将当前手机号修改为新的手机号，这样做的好处是可以保留原有的上课数据，而不会因为更换手机号而丢失数据。新手机号需要没有注册过ClassIn才可以，修改中需要知道当前ClassIn登录密码进行校验。修改后原手机号下的所有上课数据都会被转移到新手机号中，这包括已经上过的课程记录、课堂回放等等，后续用户就可以登录新手机号中继续使用ClassIn了。


### 注意事项


如果用户的ClassIn账号是通过手机号注册的，或已经绑定过手机号，则不可再次绑定，如果要更换手机号，可以修改手机号。


如果用户是“后台管理系统”中的主账号，则无法自行修改，请联系ClassIn客服帮助修改，客服电话：400-077-1585。


非真实手机号不支持修改ClassIn账号。

# 三、绑定或修改邮箱


如果用户使用手机号注册的ClassIn，则支持绑定邮箱，便于后续登录使用。


当用户的邮箱不再使用时，用户可以在ClassIn中修改当前邮箱。新邮箱需要没有注册过ClassIn才可以。修改中需要知道当前ClassIn登录密码进行校验。修改后邮箱下的所有上课数据都会被转移到新邮箱中，这包括已经上过的课程记录、课堂回放等等，后续用户就可以登录新邮箱中继续使用ClassIn了。

# 五、注销账号


当用户不再继续使用ClassIn时，可以使用手机或平板登录ClassIn后注销账号。账号成功注销后，账号下原有的所有课程及其他内容将一并删除且无法找回，已被注销的账号会变成未注册状态，请您谨慎操作。


如果用户的账号是“后台管理系统”的主账号，则无法注销账号。


如果您的账号不是真实手机号，则无法注销账号。


当用户的账号下有未完成的课堂或临时教室时，该账号无法被注销。


当用户的账号处于被封禁状态下则无法被注销。


点击 注销账号

# 六、设备管理
