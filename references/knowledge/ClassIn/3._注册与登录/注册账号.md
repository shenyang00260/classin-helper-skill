# 注册账号


英文使用说明


### 前置条件


用户在注册ClassIn前，需要有一部能正常接收验证码的手机或邮箱，以确保注册ClassIn账号时能正常接收到注册验证码。


### 使用说明


注册ClassIn账号时，如果用户未接收到短信验证码，此时用户可等待60秒后点击获取“语音验证码”。


非中国的手机号也可以注册ClassIn账号，注册的时候用户需要选择正确的区号。如美国的手机号注册，那么区号的位置应该选择"+1"。


语音验证码仅支持中国大陆地区的手机号（+86 区号的手机号）。


国内用户使用邮箱注册后，需用手机号进行验证。


短信、邮箱验证码和语音验证码每日一共上限10条，超出条数后当日将不能再获取验证码。


### 具体操作步骤


#### 电脑/平板


打开Classin


点击 立即注册


输入 手机号 或 邮箱 后，点击 获取验证码 ，设置登录密码
