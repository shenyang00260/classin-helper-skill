# 二、手机一键登录


### 前置条件


手机上必须插有可正常使用的SIM电话卡才可使用“本机号码一键登录”功能。


### 注意事项


电脑和平板用户无法使用“本机号码一键登录”功能。


点击 「本机号码一键登录」即可直接登录Classin

# 三、扫码登录


### 使用说明


用户首先在手机或平板设备上安装并登录Classin，或已完成Classin和微信绑定。然后在电脑上打开Classin，使用手机或平板上Classin或微信应用中的“扫一扫”功能扫描电脑中Classin登录页面上的二维码，以完成登录过程。


电脑打开Classin，点击 扫码登录


然后在手机/平板上打开Classin或微信并登录


点击手机/平板上Classin或微信右上角 + 号，选择 扫一扫


扫描电脑中Classin的二维码

# 四、验证码登录


需确保手机或邮箱能正常接收短信或邮件才可以使用验证码登录功能。


短信、邮箱验证码和语音验证码每日上限共10条，超出条数后当日将无法再获取验证码。


打开 Classin


点击 其他方式登录


点击 验证码登录


输入 手机号 或 邮箱 后 获取验证码

# 五、微信或Apple登录


在手机或平板设备上完成Classin和微信或AppleID绑定后，即可用微信或Apple登录使用。


手机或平板打开Classin，点击 其他方式登录


点击 微信登录 或 Apple登录


完成绑定验证即可登录
