# 教学工具


# 一、点击工具


### 什么是点击工具？


“点击工具”又称鼠标模式，是教室工具中的常用工具之一。教师可以利用点击工具触发PPT课件中的动画效果，播放PPT课件中嵌入的音频和视频，还能点击浏览器中的网页内容。此外点击工具还可用于操作虚拟“化学实验室”和“物理实验室”中的设备。当教师使用“学生屏幕共享”工具时，点击工具也能够用来控制学生桌面上的文件操作。通过使用点击工具，教师可以更加灵活、高效地进行教学，从而提升教学效果。


### 适用人员


授课教师、联席教师、被授权的学生。

# 二、选择/移动


### 什么是选择/移动工具？


“选择/移动”工具是教室工具中的常用工具之一。用户可以利用该工具对黑板上的各种元素（包括图片、图形、文字等）进行单选、多选和移动操作。当选中某个黑板元素时，会在该元素下方弹出一个“二次编辑”功能栏。用户可以通过该功能栏中的快捷键，对已选中的元素进行修改和调整。


### 使用说明


使用“选择/移动”工具时，可以单选或多选教室中的图片、文本、图形等元素，并支持对这些元素进行二次编辑。【框选模式】可在黑板上画出一个方形，方形中的所有元素都将被选中。【自由选择模式】可随意画不规则套圈，套圈中的元素都会被选中。


使用“选择移动”工具中的“二次编辑”功能时，可以先点击黑板中的元素，被选中的元素下方会出现一排可对已选元素进行二次修改的功能栏。用户可以利用功能栏中提供的功能，对当前已选中的元素进行二次修改和编辑。


当黑板中的多个元素被同时选中时，可以同时修改多个元素的颜色。

# 三、画笔


### 适用场景


画笔工具是教室里常用的教学工具之一，功能丰富，能够让用户在黑板、课件以及图片上进行标记、书写和绘画。它支持绘制多种形状，包括正圆、椭圆、正方形、长方形、不规则三角形、等边三角形、直线和曲线，以及书写文字。


用户可以选择使用普通画笔或荧光画笔。普通画笔的效果类似于粉笔，而荧光画笔则呈现半透明效果，适合在文本、课件或图片上进行重点标注。此外，用户还可以选择实线或虚线效果。


画笔工具还允许用户自由调节画笔的颜色和粗细，满足不同的绘画和标注需求。对于用户来说，画笔工具将大大提升教学互动和课堂效果。


当用户选择“曲线”模式并在黑板上绘制不规则图形（如圆形、方形、横竖线、三角形等）时，系统将在用户绘制完毕并停顿约1.5秒后进行自动图形识别。如果识别成功，系统将用户绘制的不规则图形转换为规则图形。


（1）画笔模式及快捷键


曲线模式：按住Shift键可绘制直线。


直线模式：按住Shift键可绘制水平直线。


方形模式：按住Shift键可绘制正方形。


圆形模式：


按住Shift键可绘制正圆。


按住Ctrl键可以落笔点为圆心绘制圆。


按住Shift+Ctrl键可以落笔点为圆心绘制正圆。


快捷键：


按住Alt键并按数字键盘上的"1"、"2"、"3"、"4"可分别将画笔切换为“曲线”、“直线”、“圆形”、“方形”模式。


按住Alt键并按字母键"q"、"w"、"e"、"r"、"a"、"s"、"d"、"f"、"z"、"x"、"c"、"v"可更换画笔颜色。


按住Ctrl键并按Z键可撤销上一步绘画操作。


温馨提示：在不同的软件和操作系统上，快捷键和设置方法可能会有所不同。上述操作仅为一般情况，实际操作时请根据具体软件和操作系统的要求进行相应操作。（Windows电脑上的“Ctrl”键对应Mac电脑上的“Command”键）、（Windows电脑上的“Alt”键对应Mac电脑上的“Option”键）。


在荧光笔模式下，画笔默认是实线，无法切换为虚线。画笔形状默认是曲线，无法进行选择或更改。


已经在黑板中绘画的图形无法使用快捷键修改颜色，只能使用二次编辑功能修改颜色。


自动转换为规则形状的图形无法使用Ctrl+Z撤回到上一步。


教师或学生在教室中使用画笔工具绘画或书写时，教室中的所有人都可以实时看到绘画和书写的过程。

# 四、黑板擦


橡皮擦是教室中的常用教学工具之一，用户可以使用黑板擦工具擦除黑板中的画笔工具绘画书写的内容，并且黑板擦中还有“一键清屏”功能，当遇到无法擦除的黑板元素时，可以使用“一键清屏”工具清空黑板中的所有内容。


黑板擦工具无法擦除教室中“文本工具”工具输入的文字。


黑板擦无法擦除已锁定的图形元素。


学生的黑板擦工具中没有“一键清屏”功能。


使用“一键清屏”功能后，不支持撤销。


当用户在上课过程中，意外删除了黑板中的板书内容时，用户可以尝试在键盘上同时按的Ctrl+Z键来撤销上一个删除动作。


黑板擦工具无法擦除使用“画笔”工具“直线”模式下画出来的直线线条，如果需要删除直线，建议先使用“移动/选择”工具选中直线，然后操作删除。

# 八、云盘空间


“云盘空间”在ClassIn的主界面和教室中均可使用，且两者的数据是互通的。云盘空间是教室中常用的工具之一，教师和学生可以将本地的各种课件资源(如音频、视频、文档等)上传到云盘空间中进行安全可靠的存储，以便在上课时灵活调用和使用。用户在教室中可以通过“我的云盘-文档”路径快速找到自己上传的课件并进行展示。学校授权给授课教师的公共课件可以在“授权资源”路径中查找和使用，教师从资源中心获取的相关教材也会在“授权资源”路径下显示。


教室中仅显示本课堂所属学校的组织云盘文件。


个人课件：打开云盘，进入 我的云盘 ，打开本地上传的课件


学校/机构授权的课件：打开云盘，进入 组织云盘 ，进入 学校资料 ，打开授权的课件


组织课件：打开云盘，进入 组织云盘 ，打开组织课件


资源中心课件：打开云盘，进入 资源中心 ，进入 我的资源 ，打开从资源中心里获取的课件

# 九、LMS


## 适用人员


## 适用场景


教室中的学习管理系统（LMS）是教室常用的工具之一。通过LMS，教师可以在课堂上布置随堂作业、测验和答题卡，并及时批阅学生提交的内容。此外，教室内的作业、测验和答题卡数据可以与班级外的数据无缝对接，提高了教学管理的便捷性。教师还可以利用LMS布置课后作业，并在教学过程中即时批阅，提供实时反馈。这种功能大大提升了教学效率和学生的学习体验。

iPhone、安卓手机、安卓平板中没有课堂LMS功能。

# 十、课堂笔记


所有人。


课堂笔记是教室中的常用工具之一，课堂笔记主要功能是通过截图的方式记录课堂上的重要笔记和精彩瞬间，支持保存教室中课件（pdf,ppt,word等）上的标注和解析，并可对已保存的图片进行二次编辑和添加文字说明，从而更加清晰地表达和记录核心知识点。同时，课堂笔记工具还支持在教室中将笔记图片下载到本地进行保存和分享。


每个笔记与笔记之间的添加间隔是10秒。


手机和平板没有“课堂笔记”功能。


每位用户可以同时记录100张笔记。

# 十一、聊天工具


聊天工具是教室中常用的教学工具之一。聊天工具中包括两个主要模块：聊天模块和问题模块。在聊天模块中，教师和学生可以自由打字发言，并且双方都可以撤回自己已发出的消息，并支持翻译对方的消息。此外，教师还具有禁止学生发送信息、删除学生已发送的消息等权限。在问题模块中，学生可以向教师提出问题，而教师则可以查看学生的问题并给予相应的回答。


下课退出教室后，教室内聊天框内的记录也会被清空，且无法保存。
