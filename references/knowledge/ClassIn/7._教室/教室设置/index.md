# 教室设置


# 一、教室模式


## 什么是教室模式？


“教室模式”是教室中的一项常用设置。教室模式中分为两种模式，其一是“标准模式”适用于纯线上教学的用户，提供更清晰、更完整的学习体验。其二是“大屏模式”则更适用于线上与线下混合教学，或使用触屏电脑时，能够更好地满足教师使用需求，提供更直观、更便捷的操作方式。


## 适用人员


所有人


## 使用说明


ClassIn在电脑上为教室提供两种操作界面样式，分别为“标准模式”和“大屏模式”。


当用户在教室使用“标准模式”时，教学工具位于教室界面的右侧，而教室设置则位于右上角。


当用户在教室使用“大屏模式”时，教学工具将以圆盘工具的形式呈现在黑板中，同时教室设置被放置于左下角。当使用触屏电脑时，可以通过四指触控黑板中的任意位置，将圆盘工具呼叫到被触碰的位置上。


在教室中，用户可设置自己教室模式，设置后仅影响自己，不影响教室中其他用户。


用户切换教室模式后，相当于从当前教室退出重新进入。如果教师在录课中操作切换教室模式，那么录课会停止，切换完成后，教师需要重新再点击录课等操作。


除了可以在教室中切换外，用户使用电脑也可以在本地黑板/临时教室/备课室等操作切换。


## 注意事项


使用手机和平板无法修改“教室模式”。

# 二、开/关座位席区域


## 什么是教室座位席区域？


座位席区域是指黑板中最上方的一行区域，该区域仅用于展示师生的摄像头画面，不支持书写。


“授课教师”和“联席教师”


除了在创建课堂时设置是否显示座位席区域，进入教室后，授课教师或联席教师还可以在教室的更多设置中选择显示或隐藏座位席区域。


当教室中显示“座位席区域”时，教室顶部将显示一排摄像头，其中包括“教师”和“被教师邀请上台的学生”摄像头。即使“教师”和“被教师邀请上台的学生”未开启自己的摄像头，其摄像头仍将出现在摄像头排列中。


当教室中隐藏“座位席区域”时，教室顶部将不显示任何摄像头（包括授课教师的摄像头），整个界面都是黑板，可以用于书写等操作，也可以全屏展示课件。在此状态下授课教师说话教室内其他用户仍然可以听见，如果授课教师需要展示自己的摄像头时，那么授课教师可以在教室右上角快速开启自己的摄像头，开启后授课教师的摄像头会出现在教室左上角区域。如果要展示学生的摄像头，则需要教师在花名册中点击学生上台。


联席教师和授课教师都拥有隐藏座位席区域的操作权限。当联席教师或授课教师执行隐藏操作后，教室内其他老师和学生的屏幕都无法显示座位席区域。


另外只有座位席区域显示时，教师设置的座位席轮播才生效。


使用手机或平板在教室时，该功能名称为摄像头区域。

# 三、专注模式


## 什么是专注模式？


“专注模式”是教室中的一项常用设置。专注模式是ClassIn专为课堂纪律而设计的一个课堂管控功能，能够有效的防止学生从教室切屏出去玩游戏或浏览网页等行为。


后台管理人员可以将勾选【开启专注模式】功能的皮肤应用到班级，该班级下的正式教室将默认开启专注模式；除此之外，授课教师或联席教师进入教室后，也可在教室的更多设置中开启或关闭专注模式。


开启专注模式后：若学生使用 Windows 电脑进入教室，教室窗口将保持最大化并置顶，不被其他窗口遮挡。若学生使用手机或平板进入教室，切换到其他应用时会自动退出教室。


授课教师或联席教师在教室内设置的专注模式仅对当前教室有效。


教师使用电脑或ipad才可以在教室中开启或关闭专注模式。


专注模式对使用 Mac OS设备的学生不生效。


专注模式对旁听生不生效。


若学生设备支持分屏或类似功能，专注模式也可能无法生效。

# 四、举手声音提示


## 适用场景


当学生在教室中举手时，老师端会自动播放举手提示音，方便老师在专注授课、书写板书或查看课件时，也能及时感知到学生的举手请求，不错过课堂互动。


进入教室后，授课教师和联席教师可分别设置自己的学生举手提示音，设置互不影响。


在举手提示音设置中，教师可选择无提示音，或选用叮音/滴答音/叮咚音作为提示音。


提示音仅教师本人可听见，教室内学生不会听到。


教师使用电脑才可以在教室中设置“举手声音提示”。

# 五、进出教室通知


进入教室后，教师在教室的设置中开启进出教室通知后，每当有教室的学生进入或离开教室时，教师的屏幕都会弹出提醒告知，便于教师实时掌握学生出勤状态。(标准模式下，该提示会显示在右侧花名册旁；大屏模式下，该提示会显示在左下角)


授课教师和联席教师可分别设置自己的进出教室通知，设置互不影响。


在授课教师开启该功能的前提下，联席教师的进入或离开，授课教师也会收到提醒。

# 八、快速求助


“授课教师”、“联席教师”、“学生”（除旁听生）

# 九、置顶教室窗口


## 什么是置顶教室窗口？


置顶教室窗口是教室中的一项常用设置。使用电脑的用户，若想让ClassIn教室窗口始终不被其他软件挡住，用户可在教室中置顶教室窗口。


用户使用电脑进入教室，标准模式下可以在右上角更多中点击置顶，操作后教室窗口将默认展示在所有应用的最上层，再次点击可以取消置顶教室窗口。大屏模式下默认教室窗口为全屏状态，无该功能。

# 十、退出教室


用户使用电脑进入教室后，标准模式下点击教室窗口右上角的“X”即可退出当前教室，大屏模式下需要在左下角更多中点击“退出”；如果使用的是手机或平板，需要在设置按钮中点击“退出教室”。


如果是授课教师使用电脑进入教室后，标准模式下点击教室窗口右上角的“X”后，还需要点击“仅自己离开”才是退出当前教室，大屏模式下需要在左下角更多中点击“退出”；如果使用的是手机或平板，需要先点击设置按钮，再选择点击“仅自己离开”。


没有到拖堂时间退出教室后，均可再次进入教室上课。一旦到了拖堂时间无法再进入了。

# 十二、OMO站播


## 前置条件


后台管理人员在后台的学校设置-增值服务-开启OMO站播功能，然后在创建课堂时勾选「OMO站播」。


仅需要学生在教室内观看老师站播画面且不需要录课时，推荐创建课堂时勾选「OMO站播」功能。


OMO站播模式是一种高清录制现场的模式，授课教师在OMO站播设置中选择需要被展示的摄像头画面后，在教室中的线上学生可以在右下角打开OMO站播画面窗口，更清晰地观看老师站播的授课内容。另外也支持授课教师在课中关闭OMO站播功能，关闭后教室中的线上学生将无法看到其站播画面。


当录制现场模式与OMO站播模式同时开启时，两个模式无法分别选择不同的摄像头。也就是说，当授课教师在录制现场模式中选择某个摄像头时，OMO站播也会自动切换到这个摄像头，反之亦然。


录制现场与 OMO 站播功能仅支持授课教师使用电脑或大屏（Windows系统），其他设备（比如手机/平板）不支持。
