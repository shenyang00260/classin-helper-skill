# 二、学生授权


### 适用人员


授课教师、联席教师


### 适用场景


在教室中，为了让学生积极参与互动并提高教学效果，例如当学生需要在黑板上绘画、打字或触发PPT动画效果时，教师可以根据需求，单独授权学生使用教学工具，使其更加方便快捷地完成学习任务或展示自己的学习成果。


### 注意事项


使用手机和平板（安卓系统、鸿蒙系统）上课的教师，在教室摄像头下方没有快捷键，教师可以在教室中的花名册工具里授权学生使用教学工具。


不管学生是在座位席中还是在台下都可以被教师授权使用教学工具。


教室中最多同时15名学生被授权，当授权人数达到满人时，如果再次给其他学生授权，那么最开始被授权的学生将被系统自动取消授权。


如果学生在黑板上的板书上随便涂画，可能会影响老师的教学。如果老师感到不适，可以考虑取消学生在黑板上涂画的权限——即取消授权。


如果学生摄像头下方按键中没有“授权”选项，请进入“后台管理系统”中《修改学生摄像头下方按键》。


学生端无独立教学工具箱工具，老师打开工具箱内的部分工具后，学生可同步使用。

# 三、学生上台自动授权


在教室中，为促进师生互动，教师可以开启“上台自动授权”功能。启用该功能后，学生只要一上台，立刻被系统自动授权允许使用教学工具，无需等待老师手动进行授权处理；而学生下台时，系统将自动取消学生的授权。此举将极大地提升课堂互动效率，将更多时间用于知识传授和讨论。同时本功能将为教师节省授权操作的时间和精力，提高教学效率。开启“上台自动授权”后，老师摄像头右上角会显示一个金色的循环样式皇冠图标，关闭“上台自动授权”后该图标会自动消失。


安卓手机和安卓平板没有“学生上台自动授权”功能。


教师摄像头下方的按键最多显示四个。如果按键中没有“学生上台自动授权”选项，请进入“后台管理系统”中《修改教师摄像头下方按键》。


老师 点击 自己的摄像头


在摄像头下方按键中，点击 上台自动授权

# 四、台上全体学生授权


### 使用说明


当授课教师需要在教学过程中让台上所有学生共同参与黑板绘画、打字、触发PPT动画效果以及进行课堂互动时，可以利用“全体授权”功能进行操作，以便更有效地促进学生的参与与互动。


安卓手机和安卓平板没有“全体授权”功能。


教师摄像头下方的按键最多显示四个。如果按键中没有“全体授权”选项，请进入“后台管理系统”中《修改教师摄像头下方按键》。


在摄像头下方按键中，点击 全体授权 （再次点击则取消授权）

# 五、取消所有学生授权


在上课过程中发现有学生擅自在黑板中对课件或元素胡乱操作时，如果教师无法确定是哪位学生操作的，教师可以采用“取消全体授权”功能，此举将关闭教室中所有被授权学生的教学工具使用权限，包座位席上和台下的学生。这一操作能够有效地防止学生在课堂上擅自干扰授课内容。


安卓手机和安卓平板没有“取消全体授权”功能。


教师摄像头下方的按键最多显示四个。如果按键中没有“取消全体授权”选项，请进入“后台管理系统”中《修改教师摄像头下方按键》。


在摄像头下方按键中，点击 取消全体授权
