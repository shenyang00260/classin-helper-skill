# 学生奖杯


# 一、给学生发奖杯


### 适用人员


授课教师、联席教师


### 适用场景


当学生在教室中表现优异时，教师可以给学生发送奖杯予以奖励。


### 注意事项


使用安卓手机和安卓平板上课的教师，在教室摄像头下方没有快捷键，教师可以在教室中的花名册工具里给学生颁发奖杯。


如果学生摄像头下方按键中没有“奖杯”选项，请进入“后台管理系统”中《修改学生摄像头下方按键》。

《修改奖杯样式》


### 具体操作步骤


#### 电脑/iPhone/iPad


老师 点击 学生摄像头


在摄像头下方按键中，点击 奖杯

# 二、台上全体学生奖杯


安卓手机和安卓平板没有“全体学生奖杯”功能。


教师摄像头下方的按键最多显示四个。如果按键中没有“全体学生奖杯”选项，请进入“后台管理系统”中《修改教师摄像头下方按键》。


老师点击自己的摄像头


在下方弹出的按钮中，点击 全体奖杯 按钮

# 三、花名册中给学生发奖杯


### 使用说明


学生如何在教室中查看自己获得的奖杯数：


方法一：如果学生在台上，可以在自己的摄像头右下角处看到奖杯得数


方法二：如果学生在台下，可以在教室右侧打开进入花名册，找到奖杯得数那一栏查看具体奖杯得数


#### 电脑


点击教室右侧 花名册


找到学生


点击右侧 奖杯

# 四、扣除学生奖杯


手机和平板无法扣除学生奖杯。


如果学生摄像头下方按键中没有“授权”选项，请进入“后台管理系统”中《修改学生摄像头下方按键》。


老师 点击 学生的摄像头


在摄像头下方按键中，右键单击 奖杯
