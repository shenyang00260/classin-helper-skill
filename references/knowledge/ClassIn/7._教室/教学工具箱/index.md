# 教学工具箱


# 一、屏幕共享


## 1、教师屏幕共享


### 适用人员


授课教师、联席教师。


### 适用场景


“屏幕共享”是教室中教学工具箱里的一款专业教学工具。通过ClassIn教室中的“屏幕共享”功能，教师可以实时展示教学内容，让学生同步观看和学习，从而提升教学效果和互动性。此外，教师还可以远程操控学生的桌面，迅速解决学生在操作中的困难，显著提升学生的学习体验和教学效果。


在ClassIn中，屏幕共享功能分为两种模式：“教师屏幕共享”和“学生屏幕共享”。其中，“教师屏幕共享”是指教师将自己的电脑屏幕内容实时分享给学生。这种方式广泛应用于在线教育和混合式教学环境中。通过屏幕共享，教师可以展示教材、课件、演示操作步骤，或讲解复杂概念，帮助学生实时同步观看和学习。


### 使用说明


在教师进行屏幕共享时，教师可以共享自己的电脑桌面和声音，也可以选择共享特定软件的窗口。在共享过程中，教师还可以使用“花名册工具”和“聊天工具”，并且可以打开自己的摄像头和麦克风，以便更好地与学生互动。


此外，教师可以利用“画笔工具”、“点击工具”、“选择或移动工具”、“黑板擦”以及“文本”等工具在共享窗口中进行标记和注释。教师还可以切换学生的权限，具体权限包括：


不允许学生控制：学生只能观看，无法在教师共享的窗口中进行标记或涂写。


允许授权学生控制画面：被授权的学生可以与教师共同在共享窗口中进行协同写作或绘画。


允许授权学生控制电脑：被授权的学生可以使用“点击工具”控制教师的电脑。


### 注意事项


手机和平板上只有“屏幕共享”模式，仅能共享自己设备上的画面，无法让学生在自己设备上绘画等操作。


“教师屏幕共享”模式只有系统满足：Windows8；Windows10；Mac电脑时，才可以单独共享某一个软件窗口，最大共享范围无限制（超过1920 x 1080会自动压缩到1920 x 1080）。


当教室内使用屏幕共享功能云端录课后，生成的回放画面中会显示整个电脑桌面，此时回放中录制的画面不仅仅是共享时选择的窗口，还会有其他窗口或者任务状态栏。

## 2、学生屏幕共享


授课教师、联席教师、被共享屏幕的学生

# 二、投屏


所有人（仅授课教师或联席教师可以打开此工具，打开后其他人均可使用）


上课中，当有教师或学生想在教室里展示自己手机/平板或其他电脑上的画面时，可以使用教室中的投屏工具。


投屏时，投屏设备和接收投屏设备必须在同一个网络环境下。


（1）其他电脑投屏到教室中时，投屏设备登录ClassIn后在左侧导航栏中点击投屏，将接收投屏设备上的投屏码输入后点击投屏或选择接收投屏设备的设备名称即可完成投屏，投屏设备屏幕的四个绿色角代表投屏的区域，该区域不可调整，只能是全屏被投屏使用。


（2）苹果手机、平板或电脑可以用自带的镜像投屏功能投屏，此时选择接收投屏设备的设备名称即可完成投屏；也可以使用ClassIn投屏APP，在ClassIn投屏APP中将待投屏设备上的投屏码输入后点击投屏或选择接收投屏设备的设备名称即可完成投屏。


（3）安卓手机或平板需要使用ClassIn投屏APP完成投屏，在ClassIn投屏APP中将接收投屏设备上的投屏码输入后点击投屏或选择接收投屏设备的设备名称即可完成投屏。


（4）带有投屏功能的其他视频APP也可以投屏到教室中。进入视频APP后，播放视频，点击视频自带的投屏按钮选择接收投屏设备的设备名称即可完成投屏。


在使用投屏工具时，投屏工具窗口右上角可以对本次投屏进行更多设置：


共享设置：


在本设备外放投屏设备声音；共享投屏设备声音给教室其他人员


隐私设置：


允许接收所有设备投屏；投屏窗口开启后，允许接收所有设备投屏


另外，支持多个设备一起投屏到教室中，投屏成功后，老师可以在投屏工具窗口右下角切换其他的设备来观看投屏内容。


安卓手机和平板需要满足Android系统版本在6.0及以上才可下载安装ClassIn投屏软件，苹果手机和平板需要满足iOS系统版本在12.0及以上才可下载安装投屏软件。


投屏工具只能将手机/平板或其他电脑上的画面展示到黑板上，无法将其用作手写板。如果教师希望将手机或平板用作手写板，可以尝试使用“双设备进入教室”功能。


老师使用电脑上课时，才可以使用教室内投屏功能。

授课教师、联席教师、学生

## 2、文本小黑板


“小黑板”是教室中教学工具箱中的一种重要教学工具。它相当于为每一位学生提供了一个小型黑板，便于他们在自己的小黑板上完成老师布置的任务。小黑板提供了两种模式：“画板小黑板”和“文本小黑板”。其中，“文本小黑板”主要用于文本输入和代码编写。当老师分发小黑板后，学生可以在文本小黑板上编写代码或进行其他文本操作。教师也可以随时进入某位学生的小黑板，与学生协同操作，进行指导与交流。

# 六、分组讨论


“分组讨论”是教室中教学工具箱里的一款专业教学工具。旨在帮助教师更好地组织课堂活动。通过设定不同的分组，学生可以在分组教室中进行讨论学习，激发学习兴趣和积极性。教师可以随时调整学生的组别，并与学生互动或隐身监听，了解学习情况和进展，提升教学效果。

# 十四、视频墙


“视频墙”是教室中教学工具箱里的一款专业教学工具。它为教师提供了一个可以实时查看所有学生摄像头画面的功能。如果教师想观察教室中所有学生的摄像头时，但又不想让学生上台展示摄像头，教师可以通过视频墙在课堂上悄悄观察每个学生的上课状态，以便更好地进行教学和管理。此外，“视频墙”还可以在教室中实现监考的功能。

此外，教师还可以在视频墙界面右上角设置“自动翻页”“翻页时间”“布局”，并支持拍照合影。


温馨提示：视频墙建议是双屏使用，把视频墙移动到辅助屏，这样就可以一边上课一边看视频墙

### 什么是“VNC”？


VNC (Virtual Network Console)是虚拟网络控制台的缩写，它是一款优秀的远程控制工具软件。


ClassIn VNC功能集成了VNC客户端，教师只要单独使用一台电脑作为VNC的服务端，即可使用ClassIn中的VNC功能连接该电脑，并可以使用ClassIn中丰富的教学功能来教学。


注：安装VNC服务器的电脑，其需要有公网IP。


教室中的“授课教师”或“联席教师”可以在教室里使用VNC工具。

“化学实验室”是教室中教学工具箱里的一款专业教学工具。化学实验室可以模拟真实的化学实验场景和实验结果，帮助学生在实验室之外进行实验操作和掌握化学实验技能。它通过虚拟现实技术，为用户提供逼真的视觉和听觉体验，并且在操作过程中提供自动化记录和分析数据的功能。该工具广泛应用于大专院校、中学、培训机构等教育领域，为学生提供安全、便捷、低成本的学习方式。


手机和平板没有“化学实验”功能。

# 二十二、尺规工具


“尺规”是教室中教学工具箱里的一款专业教学工具。尺规常被用于绘图和测量。尺规工具中包含直尺、圆规、量角器和三角尺。这些工具能够辅助精确地绘制直线和曲线，以及构建几何图形，如弧线和圆形。在教学中，尺规工具被广泛用于进行几何学和图形的绘制和测量，以及帮助学生理解和应用几何概念。


iPhone、安卓手机、安卓平板没有尺规工具。

# 二十三、几何图形


“几何图形”是教室中教学工具箱里的一款专业教学工具。方便教师直接在该工具中挑选可用的几何图形，并将其添加到黑板中。添加到黑板中的几何图形支持调整图形尺寸、边框颜色和填充色，通过该工具，教师可以轻松地选择不同类型的几何图形，如圆形、矩形、三角形等。

# 二十五、TI小店

商品管理和 TI 小店为增值服务，仅对企业客户开放，如需开通请联系专属客户经理
