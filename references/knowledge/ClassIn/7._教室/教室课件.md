# 教室课件


# 一、如何打开课件


### 适用人员


授课教师、联席教师、被授权的学生


### 使用说明


在classin教室中，可以在云盘中打开用户已经上传的课件，比如打开word文档，播放音频文件（音乐，MP3），视频文件（MP4）等都属于已上传的课件范畴内。


### 注意事项


教室中仅显示本课堂所属学校的组织云盘文件。


### 具体操作步骤


#### 电脑


个人课件：打开云盘，进入 我的云盘 ，打开本地上传的课件


学校/机构授权的课件：打开云盘，进入 组织云盘 ，进入 学校资料 ，打开授权的课件


组织课件：打开云盘，进入 组织云盘 ，打开组织课件


资源中心课件：打开云盘，进入 资源中心 ，进入 我的资源 ，打开从资源中心里获取的课件

# 五、PDF 跳页、缩略图浏览与同步滚动


## 适用场景


以下几种场景适合使用PDF缩略图功能：


在教室中，当用户打开PDF课件时，如果PDF页面数量较多，为了方便用户查看PDF的大纲，可以使用PDF缩略图功能。通过显示PDF缩略图，用户可以快速浏览整个课件的内容结构，并从中选择特定页面进行查看。这一功能使得用户能够更加高效地导航和定位于需要的信息，提升了PDF课件的使用体验。


当用户在教室中展示上一次未完全呈现的PDF课件时，用户可以借助PDF缩略图功能迅速定位到上一次展示的页面。通过点击相关缩略图，用户可以迅速跳转至所需页面，精确呈现之前未完全展示的内容。这一过程有效缩减了查找特定页面所需的时间，提高了教室展示效率。

# 九、课件显示大小调整


授课老师、联席教师、被授权的学生


### 适用场景


当用户在教室中打开课件后，如果感到课件窗口太小或太大，可以对其进行调整。通过调整窗口大小，用户可以更好地浏览课件，享受更佳的视觉体验。其中可调整的课件包括：PDF课件、PPT课件、音频、视频、Word文档、Excel表格等所有教室中支持打开的文件。


#### 电脑：


拖拽调整：将鼠标悬停在课件窗口的边缘，当鼠标指针变为双向箭头图标时，按住并拖拽边框以调整窗口大小。


最大化窗口：如果课件窗口未处于最大化状态，点击窗口右上角的“最大化”按钮，即可将课件窗口最大化。


恢复窗口大小：如果课件窗口已处于最大化状态，点击窗口右上角的“恢复”按钮，即可恢复窗口为非最大化状态。


#### 手机/平板：


双指缩放：将双指放在屏幕上的课件位置，向内或向外滑动，即可实现课件窗口的放大或缩小。
