# 班级公告


英文使用说明


# 一、什么是公告？


当教师有重要通知时可以在班级中发布公告。


# 二、新建「公告」


### 适用人员


班主任、班级教师


### 使用说明


教师可在班级内发布公告，支持将公告同步发布至多个班级（需满足前提：发布公告的教师在这些班级中均拥有教师身份）；若需确保每位同学都查看公告，可在创建公告时开启需要学生确认查收功能；此外，还支持公告置顶操作，置顶后的公告将在班级聊天群顶部显示，方便师生快速查看。


公告内容支持添加文字和文件，其中文件的数量上限为：图片附件15张，视频附件15个，而本地课件、云盘课件和链接的总数也不能超过15个。单个视频和其他文件大小不能超过500M，图片不能超过25M。

# 五、分享「公告」


已发布的公告支持分享到其他软件进行提醒和查看其文字内容，如需查看完整公告附件等需要到ClassIn app中查看；


当教师使用电脑分享公告时，支持复制二维码和保存图片进行分享。


当教师使用手机或平板分享公告时，支持分享到微信好友或微信群、朋友圈等，以及通过复制链接进行分享。
