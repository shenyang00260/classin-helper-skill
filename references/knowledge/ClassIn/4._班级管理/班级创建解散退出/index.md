# 班级创建/解散/退出


英文使用说明


# 一、创建班级


### 什么是班级？


班级是学校为实现特定教育目标而组织的基本教育单位。它包括学生和教师，通过教学和集体活动促进学生的学习与成长。在ClassIn在线教育平台中，班级具有以下几个关键作用和特点：


教学活动的基本单位：班级是展开教学活动的基本单位，教师可以在班级中创建课程、进行在线授课和学习管理。学生可以通过班级获取教学资源和参与讨论。


促进学生的互动和合作：在班级聊天区域中，学生可以通过文字交流、学习讨论和互动等方式与其他成员或教师进行沟通和合作，提高学习效果。


便于管理学习进度和内容：教务老师可以在班级中创建不同系列的课程，从而更好地组织和管理学生在不同课程之间的学习进度和内容。学生可以在班级中找到所需课程，按照自己的学习计划进行学习。


### 适用人群


在ClassIn中，所有用户都可以创建班级，不同版本的ClassIn用户创建的班级可能在设置和功能方面有所不同。主要还是以学校中的教务人员、班主任或课程教师来创建班级，这些用户在ClassIn中有更多权限和功能。


### 适用场景


创建班级是为了更高效地组织和管理学生的学习活动。班级可以将相同或相近年级的学生集中在一起，使他们在统一的环境中学习和交流。通过创建班级，教师能够更好地了解和跟踪学生的学习情况，并及时提供指导和支持。同时，班级为学生提供了一个共同学习的平台，促进了学生之间的合作与互助。


此外，班级还能够培养学生的集体意识和团队合作精神，为学生提供了锻炼综合素质和社会技能的机会。因此，创建班级对于营造有序、积极的学习氛围，提高教育教学质量具有重要意义。


### 使用说明


班级中可以提供多种教学功能，其中包括但不限于以下几个方面：


聊天：在“聊天”中，所有班级成员可以发送文字消息、语音消息（仅限手机和平板）、表情包、图片，以及从电脑本地或云盘中发送文件，促进互动交流。


课程和活动：老师可以在课程下创建单元和活动，活动包括课堂、录播课、作业、测验（试卷）、答题卡、讨论、学习资料和AI口语卡，为老师的授课提供多样化的学习内容和形式。


成绩：在“成绩”中，老师可以查看学生在各项学习活动中的成绩和素养能力，全面了解学生的学习进展和能力水平，进行个别辅导和进一步指导。

# 二、解散班级


### 适用人员


班主任


解散班级指不再继续为该班级提供任何教学支持和指导。班级解散后，该班级仍将保留在用户的ClassIn“课程表”中，用户可以在“课程表”中查找已解散的班级。已经被解散的班级，班级中的成员将无法在班级聊天区域进行任何形式的讨论，班级教师也无法在班级中创建新的教学活动。但是班级用户仍然可以正常观看已解散班级中的课堂回放视频。需要注意的是，如果班级中还有未开始的课堂（课节）或正在进行中的课堂（课节）时，班主任将无法解散该班级，需要等待班级中所有课堂（课节）结束后才可以解散班级。


班级无法被删除或被注销，只能由班级中的班主任在教室设置中将班级解散。


解散班级后学生是否可以继续回看班级中历史课堂的回放视频及课程活动附件内容，取决于班级设置。

# 三、退出班级


班级教师和学生


普通用户创建的班级（免费版）：学生可在任意时间退出班级；


试用版用户创建的班级：教师有未完成课堂时不可退出班级，其他情况可以退出班级；


正式版用户创建的班级：教师和学生在班级下最后一个课堂结束后60天后可退出班级。


点击 班级设置


点击 离开班级
