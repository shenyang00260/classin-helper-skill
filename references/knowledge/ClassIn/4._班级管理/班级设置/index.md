# 二、班级介绍和班级备注


### 适用人员


“班主任”可以设置班级介绍。


“班主任”和“教师”可以设置班级备注。


### 适用场景


班级介绍是为了让教师和学生更全面地了解班级的组成和特点，有助于学生、家长和其他相关人士对班级进行评估和选择。


班级备注用于教师补充班级核心属性与基础管理信息，帮助教师自己快速区分班级。


班主任可以在建立完成的班级中设置班级介绍，最多支持添加400字符。


班主任和教师可以设置班级备注，班级备注仅对自己可见，最多支持添加32字符。

# 三、班级头图


### 什么是班级头图？


班级头图指的是ClassIn客户端中班级群的顶部横幅图片，主要用于突出班级主题、增强班级氛围感和提升视觉冲击力。


班主任可以在建立完成的班级中修改班级头图。

手机暂不支持设置班级头图。

# 四、开放加入班级


当未进班的教师和学生想要通过自主申请加入班级方式报名进班时，班级中的班主任可以开启“开放加入班级”功能。该功能在班级管理中进行设置，这是一种便捷的报名方式，通过这种方式，可以快速地使更多的人员加入班级，以便他们能够参与学习。


班主任开启「开放加入班级」功能后，班级中的教师可以将班级报名入口分享给需要进班的老师和学生。

# 五、分享页面中显示班级报名入口


如果想要提高班级的招生效果，班主任可以在班级管理中开启“分享页面中显示加入班级入口”功能。启用此功能后，在分享的班级作业、测验、网页直播回放等页面中，将展示该班级的加入班级入口，方便有需要的用户自主申请加入班级。


班主任在班级管理中开启「分享页面中显示班级报名入口」功能，开启后，班级教师分享班级作业、测验、网页直播回放等页面中将展示该班级加入班级入口。

# 六、学生修改班级昵称


当班级学生人数较多时，为了提高教师修改班级学生昵称的效率，班主任可以在班级管理中开启“允许学生修改班级昵称”功能。开启该功能后，学生可以自行在班级设置界面中修改自己的班级昵称，以满足个性化需求。


设置的班级昵称会在班级的教室和活动中展示。

# 七、班级成员可互相添加好友


控制班级成员之间添加好友的行为，是一项用于管理班级社交互动的功能。班主任可以根据实际需求开启或关闭“允许班级成员互相添加好友”功能，以便更好地管理班级成员之间的互动行为。当该功能开启时，班级成员可以正常地通过系统添加对方为好友；而当该功能关闭时，班级成员间将无法相互添加为好友，以确保班级管理工作的高效开展，避免不必要的社交互动对正常教学秩序造成干扰。


开启「班级成员可互相添加好友」功能后，老师和学生以及学生和学生之间均可互相添加好友，关闭此功能后相互之间则无法添加好友。班主任加班级成员为好友不受此功能限制。

# 八、允许教师新建课堂


如果想要让班级中所有的教师都具备创建课堂的权限，那么班主任需要在班级管理中开启“允许教师新建课堂”功能。此选项的启用将使得班级中所有教师身份的成员都能够自主在班级里创建新的课堂，并修改自己授课的课堂，以便更为便捷地进行教学和管理班级和有效地分配任务，提升班级教师的自主性和专业性，促进教学工作的顺利开展。


开启「允许教师新建课堂」功能后，班级中教师可以在班级中建立课堂，也可以针对自己是授课教师或联席教师的课堂进行编辑。

# 九、允许离开班级的学生或班级解散后，可查看课程内容


开启此功能后，当出现以下几种情况时，如：“学生被教师移出班级”“班级解散”“学生主动离开班级”，学生仍然可以查看课堂回放、板书课件、录播课视频以及其他活动附件内容。


关闭此功能后，当出现以下几种情况时，如：“学生被教师移出班级”“班级解散”“学生主动离开班级”，学生将无法查看课堂回放、板书课件、录播课视频以及其他活动附件内容。


“后台管理系统”中可以对所有班级设置开启/关闭《允许离开班级的学生或班级解散后，可查看课程内容》
