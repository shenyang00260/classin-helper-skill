# 班级教学评价设置


英文使用说明


# 一、管理评价方案


### 适用人员


“班主任”和“教师”


### 使用说明


在"管理评分方案"中，会提供四个默认方案分别是分数制、百分比、优良评分和ABCD等级制，其中分数制和百分比不支持删除和编辑。此外还支持自定义"评分显示方案"，自定义的"评分显示方案"支持设置其评分名称、百分比和等级，此评分方案应用在录播课、作业、打卡、测验 、答题卡、讨论、AI口语卡等涉及到评分的教学/学习活动中，便于教师更准确的针对教学/学习活动进行评价。

进入 课程管理

# 二、管理能力模型


在"管理能力模型"中，支持自定义能力模型、设置量规等级分值，以便教师更好地完成阶段教学任务，准确地对能力水平等级进行考察评价。能力模型用于有设置评分方案的录播课、作业、讨论、AI口语卡的教学/学习活动中。


#### 电脑

点击左上角 班级名称


在班级设置中进入 课程管理

# 三、成绩计算方式


在"成绩计算方式"中，会提供2种学习活动总成绩计算方式，分别是总分和加权；也可以设置当前课程下的学习活动总成绩评分方案，选的显示方式有：分数制、百分比、优良评分和ABCD等级制及个人自定义评分方案；同时也可以添加成绩类别，成绩类别用在录播课、作业、打卡、测验 、答题卡、讨论、AI口语卡等涉及到评分的教学/学习活动中，学习活动总成绩计算方式如果选择加权，那么会根据学习活动选择的成绩类别进行计算总成绩。
