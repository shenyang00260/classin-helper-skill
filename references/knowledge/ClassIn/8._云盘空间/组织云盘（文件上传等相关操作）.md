# 组织云盘（文件上传等相关操作）


# 一、新建组织盘


## 适用人员


学校/机构主账号


拥有“组织云盘”权限的后台学校/机构管理员


## 适用场景


在教研过程中，教研组的多位老师经常需要共享教研资料，并进行协同工作。为了促进这一过程，使用组织云盘可以满足老师们的需求。除了个人云盘的使用，组织云盘还提供了多人共享和共建的空间。


## 使用说明


学校/机构主账号以及拥有“组织云盘”权限的后台学校/机构管理员可以在ClassIn软件中的“空间”栏目里的“组织云盘”中创建组织盘，创建好组织云后才可以在云盘里添加文件和子文件夹。


## 注意事项


手机不支持在组织云盘里创建组织盘。


## 具体操作步骤


#### 电脑/平板


左侧导航进入 空间


进入 组织云盘


选择 学校/机构


创建组织盘


输入组织盘文件夹名称


点击 创建

# 二、新建/上传文件(文件夹)


针对文件夹拥有可操作/可管理权限的教师


有对应权限的教师、管理员和主账号人员可以在组织盘下新建/上传文件夹和文件，也支持从我的云盘中上传文件到组织盘中。


组织盘下支持新建文件夹，以及新建板书、作业资源、在线PPT、协作文档等文件。平板不支持新建在线PPT。手机仅支持创建文件夹、板书、作业资源和协作文档。只有电脑支持上传文件夹。


组织盘还支持上传文件，比如ppt，word，pdf，图片，音视频等文件。


在ClassIn中对于上传文件的大小有一定的限制：通过电脑上传时，单个文件不得超过 500MB；通过手机或平板上传时，单个文件不得超过 100MB；对于 TXT 文件，建议大小不超过 1MB，以确保在教室内正常使用。此外，通过手机或平板上传的视频文件需注意，分辨率低于 720P 的视频，大小不得超过 100MB； 分辨率高于 720P 的视频，时长不得超过 3 分钟。


文件夹总量限制是2000个，每个文件夹下的文件数量限制是300个。


电脑端支持拖拽文件和文件夹的方式上传至我的云盘。


文件上传后需要一定时间进行转换，建议提前上传。


若需在教室内使用文件，必须在教室内从云盘中打开。


所有格式的文件都可以上传到ClassIn中，但是只有以下所列出的文件格式才能在教室中正常打开使用。

# 三、成员权限


拥有“组织云盘-成员管理”权限的后台学校/机构管理员


针对文件夹拥有可管理权限的教师


有对应权限的教师、管理员和主账号人员可以在组织盘及其下文件夹添加设置成员及管理权限。有对应权限的成员可以查看和使用组织云盘下的文件。


组织盘及其下文件夹-成员管理权限说明：


| 权限 | 说明 |
| --- | --- |
| 可管理 | 文件夹支持：设置成员管理（添加、授权、移除成员）权限，及上传、新建、复制、移动、重命名、删除组织盘下的文件夹；

文件支持：查看、上传、新建、转发、下载、删除、复制、移动、保存至云盘、重命名、打印、在线编辑ppt/edoc操作，及文件权限中设置文件转存载功能。 |
| 可操作 | 文件夹支持：查看成员管理权限及上传、新建、复制、重命名组织盘下的文件夹；

文件支持：查看、上传、新建、复制、重命名、打印、在线编辑ppt/edoc操作；

支持删除自己上传的文件和文件夹。 |
| 仅查看 | 仅支持查看文件夹及其下的文件。 |
| 不可见 | 不可查看文件夹及其下的文件。 |

对于使用“中小学后台管理系统”的学校，老师离职 / 被删除、学生毕业 / 被删除，系统自动解除其组织云盘权限；离职老师复职、学生重新入学，需按新成员流程重新获取组织云盘权限。

# 四、文件权限


拥有“组织云盘-删除”权限的后台学校/机构管理员

# 七、下载文件


拥有“组织云盘-下载”权限的后台学校/机构管理员


电脑支持从组织云盘下载文件，例如图片、板书、PPT、PDF和Word文档。但ClassIn专属文件格式不支持下载或用其他应用打开，如在线幻灯片（eppt格式）和文档（edoc格式）等。


手机和平板支持将图片保存到本地相册，但对于、PPT、PDF等文件没有下载功能，可在组织云盘中打开，再点击分享功能，通过其他应用打开查看使用。
