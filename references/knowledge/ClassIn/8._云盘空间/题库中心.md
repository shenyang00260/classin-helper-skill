# 题库中心


# 一、什么是题库中心？


题库中心为用户提供了丰富的试题资源，包括个人题库、组织题库以及Classin官方题库。用户可以根据需求选择不同类型的试题进行添加，如单选题、多选题、判断题、填空题、问答题和综合题。综合题可以包含多个小题，小题类型包括单选题、多选题、判断题、填空题和问答题。


用户可以单个添加试题，也可以批量添加试题，操作方便。试题添加完成后，用户还可以创建试卷，并将已添加的试题加入到试卷中，或者在创建试卷的过程中添加试题。


机构/学校教师管理中教师可以在ClassInAPP空间-题库中心中显示组织题库，是否可以查看使用组织题库中内容和学段学科的成员管理权限有关。


无论是个人学习还是组织培训，题库中心都能够满足用户的多种需求，提供高效便捷的试题管理和试卷创建服务。

## 2. 添加试题


在我的题库中添加试题时，用户可以根据需求设置知识点或章节，然后将试题添加到相应的知识点或章节下。使用电脑或平板添加试题时，用户可以添加多种类型的试题，包括：单选题、多选题、判断题、填空题、问答题、综合题（综合题中可以包含多个小题，支持的类型有单选题、多选题、判断题、填空题和问答题）；使用手机添加试题时，仅支持添加单选题、多选题、判断题。完成试题添加后，用户可以对已添加的试题进行删除、复制、转发、移动到其他学科等操作。有组织题库权限的老师，还可以将试题添加到其有权限的组织题库中。


当需要添加多个试题时，可以使用电脑中批量添加试题功能。首先，从批量添加试题窗口中下载Excel模板或Word模板，然后按照模板中的格式添加试题。完成后，将已添加好试题的模板上传回批量添加试题窗口。如果您熟悉批量添加试题的格式，也可以直接将符合格式的试题复制粘贴到批量添加试题窗口中的原题录入区域。

## 4. 编辑/删除试题和试卷


对于已添加的试卷，可以对其编辑、删除、复制、转发给 ClassIn 班级和好友、移至其他科目，或添加到机构题库中。

在我的题库中添加试题时，用户可以根据需求设置知识点或章节，然后将试题添加到相应的知识点或章节下。使用电脑或平板添加试题时，用户可以添加多种类型的试题，包括：单选题、多选题、判断题、填空题、问答题、综合题（综合题中可以包含多个小题，支持的类型有单选题、多选题、判断题、填空题和问答题）；使用手机添加试题时，仅支持添加单选题、多选题、判断题。完成试题添加后，用户可以对已添加的试题进行删除、复制、转发、移动到其他学科等操作。主账号、组织题库权限的管理员以及对应学段学科有“可管理”权限的教师，还可以将试题添加到我的题库中。

## 6. 添加试卷


组织题库成员中，拥有“可管理”“可操作”权限的成员


用户可以在组织题库中添加试卷。创建试卷时，您可以从我的题库和组织题库中选择试题加入到试卷中，也可以在添加试卷时直接新增试题，使用电脑时还支持批量添加试题。对于已添加的试卷，支持复制、转发、删除或者将试卷移动到其他学段或学科中，也支持创建测验，使用电脑时可以将试卷下载到电脑本地。主账号、组织题库权限的管理员以及对应学段学科有“可管理”权限的教师，还可以将试题添加到我的题库中。
