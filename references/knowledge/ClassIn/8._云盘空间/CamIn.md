# CamIn


# 一、什么是CamIn？


CamIn 开映，是一款全能易用的视频创作工具。集虚拟摄像头、视频制作与直播推流功能于一身。致力于让每个用户轻松简单地制作精美的视频。


CamIn 提供电脑端的使用支持，在ClassIn软件空间中可查看使用CamIn的云空间。

# 二、上传视频/新建文件夹


### 适用人员


所有人


### 使用说明


用户使用电脑和平板支持在空间中查看和使用Flowin的云空间，在云空间中支持上传视频和新建文件夹，其中视频支持分享、重命名、编辑封面、移动、创建副本、下载、删除等操作；文件夹支持分享、重命名、移动、删除等操作。


### 具体操作步骤


#### 电脑/平板


进入 空间


进入 Camin


点击 新建-上传视频 / 新建文件夹


选择视频或文件夹操作

# 三、团队空间


CamIn 云空间支持团队空间创建与管理，其团队版是适配多人协作的视频制作与管理工具。用户使用电脑和平板在ClassIn 软件的Camin模块中，可切换团队、邀请及管理成员，还能上传视频、新建文件夹。


展开侧面工具栏


切换或新建团队


邀请团队成员


进入 团队空间


进入 空间管理 和 空间设置
