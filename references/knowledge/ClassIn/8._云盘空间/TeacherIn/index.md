# TeacherIn


# 一、什么是TeacherIn？


TeacherIn 是一个面向教育领域的平台，主要提供课程发布、售卖、管理以及教学相关的服务，适用于教师、教育机构及私域运营者。


TeacherIn 与 ClassIn 教学平台深度整合，教师可以将 TeacherIn 上的课程直接应用到 ClassIn 的班级中，实现线上线下教学的无缝衔接。

# 三、应用/分享/发布/删除课程


空间中支持创建课程，课程中支持新建单元和活动。另外


教师可以将课程中活动应用在自己是班主任或教师的班级中。


课程支持分享给其他用户共同编辑使用。


课程支持发布，发布后的课程可以在TeacherIn平台中搜索使用。


课程页面支持用浏览器打开、课程设置（文件下载权限等）、删除等操作。

# 四、应用/分享/发布/删除课程


用户可以在TI中创建组织空间，在组织空间中可以共同管理创建课程，也支持售卖课程等操作。另外空间也有一定的容量大小限制，超过限制无法使用。用户使用电脑或平板时可以在我的空间中切换或建立组织空间，也支持查看个人空间容量大小。
