# AgentIn


# 一、什么是AgentIn？


AgentIn智能体商店，是赋能每一位教育者的AI智能体伙伴。在各种真实的教学场景和流程中，教师可以直接调用AgentIn提供的智能体，来提高工作效率，还可以根据自身的需求来创造智能体，让AgentIn成为真正懂教育的伙伴。


# 二、空间创建AI智能体


### 适用人员


所有人


### 使用说明


用户使用电脑除了可以在班级中创建智能体，还可以在空间内进入 AgentIn 创建智能体。


可以通过选择已有模板进行创建AI应用，创建后可以将该智能体发布到智能体商店中供所有人使用，也可以设置公开范围为仅自己可见。发布后的智能体可以在【我创建的】中查找和使用。发布到智能体商店的智能体，可以在首页中通过筛选发布前设置的商店分类进行查找。已发布的智能体支持在【我创建的】中操作编辑、删除和下架。


也可以导入Flowin、Coze、元器、FastGPT等平台已发布的智能体。已发布的智能体支持在【我创建的】中操作编辑和删除。


另外，用户创建或导入的智能体默认也会展示在【我的收藏】中。


平板不支持创建AI智能体。


手机不支持在空间中进入AgentIn。

# 三、空间收藏AI智能体


用户使用电脑或平板，可在空间内进入 AgentIn，于首页按需筛选并收藏智能体，方便后续快速调用；已收藏的智能体可在【我的收藏】中查找使用，也可以将其取消收藏。


### 具体操作步骤


进入 首页


按需查找智能体后，点击进入该 AI 智能体


进入 体验


点击 收藏


进入 我的收藏 查看并使用收藏的AI智能体

# 四、空间对话AI智能体


用户使用电脑或平板，可在空间内进入 AgentIn，在【我的收藏】、【我的创建】和【首页】中查找并进入智能体与其对话。


在【我的收藏】中与智能体的对话默认是留存的，也支持删除历史对话记录。


平板的空间 - AgentIn中，没有【我的创建】。


#### 电脑/平板


进入 我的收藏/我的创建 与智能体对话；或进入 首页 与智能体对话

# 五、空间应用AI智能体到班级


班主任、班级教师


用户使用电脑或平板，在空间 - AgentIn - 首页查找并进入所需智能体与其对话，可根据其介绍页面中的适用场景（发作业 / 批作业 / 出题 / 学生自学等），将该智能体应用于未结课班级的作业、测验，或添加至对应班级的 AI 应用中。


在介绍页面中，查看 适用于


与智能体对话


智能体回复结束后，点击 应用于

点击 查看 或 删除 自己的历史对话
