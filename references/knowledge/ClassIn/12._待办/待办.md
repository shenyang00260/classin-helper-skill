# 待办


# 一、什么是待办？


“待办”功能旨在帮助用户高效管理和跟踪未完成的学习和教学活动。通过将用户参加的多个班级活动整合到一个界面中，待办模块使用户能够更方便地查看进度并完成任务。此外，待办模块还会提醒用户即将到期的任务，确保重要任务不会被遗漏。无论是学生还是教师，待办功能都能大幅提升任务管理的效率。


# 二、从待办进入任务


### 适用人员


有活动/公告/日程的教师和学生


### 使用说明


当教师在班级中发布学习活动时，如课堂、作业、测验、通知、讨论、答题卡、打卡、录播课和学习资料，学生可以在自己的待办中找到这些待完成的任务。点击相应的待办任务名称，学生即可进入相应的页面进行完成任务。学生完成任务后，活动的教师可以在待办中查看学生的完成情况，并进行批阅。


用户的个人日程以及班级公告，也可以在待办中查看确认。


用户进入【待办】后，可以查看【待处理】、【预计待办】和【已处理】三个板块。


【待处理】：显示所有未完成的任务，按照已过截止时间、一天内、一周内、一周以后、无截止时间分类展示；用户可以通过筛选身份、班级、待办类型来快速查找任务，点击任务名称进入对应项目进行查看完成。


【预计待办】：展示非红点的待办，包括教师或学生暂时不需要处理、暂未开始的任务，按照已过截止时间、一天内、一周内、一周以后、无截止时间分类展示，方便用户查看未来要处理的任务，合理安排时间；用户可以通过筛选身份、班级、待办类型来快速查找任务，点击任务名称进入对应项目进行查看完成。


【已处理】：已完成或手动标记/清空当前分类的任务会自动归类到此板块。


### 注意事项


课堂只有到了上课时间后才会显示在“待处理”中，上课时间之前显示在“预计待办”中。


### 具体操作步骤


#### 电脑/平板


左侧导航进入 待办


选择 待处理 / 预计待办 / 已处理


搜索（活动名称）或筛选待办


点击右侧按钮进入到对应待办任务页面

# 三、标记或清空当前分类


用户可以在【待处理】和【预计待办】中手动将不需要继续完成的待办任务标记或清空当前分类，以更好地管理任务列表。手动标记/清空当前分类的任务会自动归类到【已处理】中。


课堂不支持标记


选择 待处理 / 预计待办


标记 或 清空当前分类

# 四、已处理取消标记


如果不小心误操作将任务标记到已处理中，那么用户可以在【已处理】中找到任务取消标记，取消标记后的任务可以在【待处理】或【预计待办】中查找，或在【已处理】中直接跳转到任务页面继续完成任务。


课堂不支持取消标记


进入 已处理


点击标题左侧对勾取消标记 或 点击右侧进入到对应待办任务页面
