# 设备信息查询：配置，跑分，网速，芯片型号


英文使用说明


# 一、查询设备配置


## 应用场景


查看电脑配置是指检查计算机的硬件和系统规格，以确保其满足特定软件的要求。对于使用电脑的用户，可以参考《电脑配置要求》中的硬件参数，然后对比自己电脑的硬件配置，确认其是否能够运行Classin。


## 具体操作步骤


### （1）Windows电脑


右键单击桌面上 此电脑


选择 属性


在 设备规格 中查看电脑内存容量和CPU型号

# 二、查询网速


为了确保您在使用ClassIn进行在线教学/学习时拥有稳定的网络连接，请在开始前对您的网络状况进行检测。通过提前检测并优化您的网络设置，您可以确保Classin运行流畅，为您的在线教学提供有力支持。


## 使用说明


以下是检测结果中需要关注的几个关键指标：网络上行速度、下行速度、时延、抖动和丢包率。如果您的网络无法同时满足以下要求，您可能会在使用Classin时遇到连接不稳定、频繁掉线等问题，从而影响教学效果。


| 最低网络要求 |
| --- |
| 上行速度：不低于2Mbps |
| 下行速度：不低于2Mbps |
| 时延：中国国内：不超过50ms、国外地区：不超过300ms |
| 丢包率：必须为0% |


最低网络要求


上行速度：不低于2Mbps


下行速度：不低于2Mbps


时延：中国国内：不超过50ms、国外地区：不超过300ms


丢包率：必须为0%


打开 浏览器


在浏览器地址栏中输入：https://www.speedtest.cn


点击 测速

# 三、查询设备跑分


### 应用场景


用户可根据电脑的CPU型号，在指定的CPU跑分查询网站上查询CPU的性能评分，以确定其是否能够支持Classin的正常运行。


### 使用说明


教师电脑的CPU性能评分需达到6000分或以上。


学生电脑的CPU性能评分需达到4000分或以上。


### 具体操作步骤


在浏览器地址栏中输入网址： https://www.cpubenchmark.net/cpu_list.php


在搜索框中输入您的电脑CPU型号


点击 Find CPU 进行搜索


在左侧黄色区域中找到与您电脑CPU型号完全一致的选项


查看右侧 CPU Mark 一栏中的数值，该数值即为您电脑CPU的性能评分

# 四、查询Mac电脑芯片型号


Classin针对Mac电脑提供了两种安装包：一种适用于Intel芯片的Mac，另一种适用于Apple芯片（如M1、M2等）的Mac。因此，用户在下载安装包之前，需要先确认自己电脑的处理器类型，选择相应的安装包进行下载。


将鼠标悬停在桌面左上角 苹果 图标上


在下方点击 关于本机


在关于本机界面中 处理器 一栏右侧查看CPU芯片型号
