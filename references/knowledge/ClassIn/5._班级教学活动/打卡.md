# 打卡


英文使用说明


# 一、什么是「打卡」？


ClassIn中的“打卡”是一种记录学习任务完成情况的工具。通常教师会安排一些需要重复完成的任务例如每天背诵100个英文单词。教师可以在ClassIn上创建一个打卡任务，详细说明具体要求，并发布给学生。学生在完成任务后，可以通过拍照、录视频或录音等方式进行打卡记录。这种打卡机制的主要作用是帮助教师了解学生对课程的掌握程度，从而判断是否需要针对性地加强某些方面的教学。教师可以根据打卡情况调整教学内容和方法，更好地支持学生的学习需求。同时打卡也有助于提高学生的学习积极性和任务完成效率。


# 二、创建「打卡」


### 适用人员


“班主任”和“教师”


### 使用说明


教师在进入班级页面后，可以在课程栏目下布置打卡。“打卡”任务中支持添加多种类型的附件，包括：图片、语音、云盘文件、本地文件和网站链接等，以帮助学生更好地理解和掌握教师讲解的“打卡”内容。需要注意的是，“打卡”活动中附件的数量上限为：图片附件15张、录音附件15条、视频附件15个，而本地文件、云盘文件和链接的总数也不能超过15个，每个视频或文件大小≤500M。电脑上传打卡附件时支持将图片以及文件拖拽到打卡内容中。


创建“打卡”时，可设置的参数请参考下方：

| --- | --- |

| 评分 | 设置打卡的评分制度和分数。 |
| 发布给 | 可以将打卡发布给特定班级、班级中的部分小组成员或部分学生，也可以同时发布给多个班级。 |

# 三、修改「打卡」


打卡支持编辑，但是不同状态下的打卡可编辑的内容有所不同。


草稿和未开始的打卡，“课程”无法修改，其他内容和设置都可以进行修改。


进行中的打卡，内容可以进行修改，设置选项中的“开始时间”和“课程”无法修改。


已结束的打卡，内容不可以再进行修改，设置选项中的打卡周期、打卡提醒、发布给、教师、课程、允许新进班学生提交这几个选项将无法修改。

# 七、分享「打卡」


教师可以将进行中或已结束的打卡活动分享到微信或其他平台中，便于督促学生及时参与打卡活动。


当教师使用电脑分享打卡时，可以将其分享到ClassIn班级或好友中，或者复制链接进行分享，还可以通过手机微信扫描二维码分享给微信好友或微信群中。


当教师使用手机或平板分享打卡时，可以将其分享给ClassIn的班级或好友，也可以将其分享到微信好友或微信群、朋友圈、微博、QQ好友中，以及通过复制链接进行分享。


在其他平台查看教师分享的打卡，其中的图片、录音、视频可直接查看播放，其他文件需唤起 ClassIn 进入班级查看教师布置的打卡内容。

# 九、学生提交「打卡」


教师发布打卡后，学生可以进入ClassIn班级或待办中找到打卡查看并提交。


在打卡页面中，学生选择打卡日期后，可以看到当日打卡剩余时间或补卡截止时间，点击立即打卡或补卡后填写打卡内容，打卡内容中支持上传附件，比如可以上传图片、录音、视频、个人云盘中的文件、本地文件以及链接等。需要注意的是，打卡活动中附件的数量上限为：图片附件15张、录音附件15条、视频附件15个，而本地文件、云盘文件和链接的总数也不能超过15个，每个视频或文件大小≤500M。电脑上传打卡附件时支持将图片以及文件拖拽到打卡内容中。


填写完打卡内容后，学生可以点击提交打卡。


当学生提交打卡后，如果教师还没有对学生的打卡进行点评，那么学生支持对已提交的打卡内容进行修改或删除。如果教师点评学生打卡时需要学生修改打卡内容，那么教师可以将学生的打卡设置为“打回订正”，这样学生就可以对自己已提交的打卡再次进行订正并重新提交，订正后且教师未点评的打卡仍支持学生修改，但不再支持学生删除。

# 十、教师点评「打卡」


学生提交打卡后教师可以进入班级或待办中找到打卡进行查看和点评。


在打卡页面中，教师选择打卡日期后，可以在待批中查看学生的打卡内容并点评，在点评中支持输入文字和录音评语，如果学生的打卡内容表现出色，教师可以将其设为优秀打卡，如果打卡设置了评分，也可以对其评分。


教师点评完成后，可以点击提交点评。如果需要学生重新修改提交打卡，可以勾选打回订正。


另外教师支持在已批栏中修改自己对学生的点评。

# 十一、学生使用「打卡」


“学生”


学生可在待办或班级中查看并提交打卡。进入打卡页面后可看到打卡排名，日历，打卡详情，我的打卡和打卡动态。


在打卡排名中，学生可看到自己的打卡天数和打卡率，也可以看到班级学生的打卡率排名，也支持对自己的打卡排名进行图片分享。


在打卡日历中，学生可点击切换不同日期，查看当日的打卡内容，或进行打卡补卡；日期下面会显示灰色和绿色的对号，以此来区分是否完成打卡。


在打卡详情中，学生可以看到教师发布的打卡内容以及打卡活动的开始和截止时间等信息。


进入打卡记录后，学生可以看到自己的打卡天数、连续打卡天数、获得优秀打卡的次数以及获赞次数，也可以看到每天的打卡内容、教师点评、分数和获赞数量，可以对自己的打卡进行点赞、分享（分享本次打卡或打卡记录）、修改和删除操作（已提交且教师未点评的打卡支持学生修改和删除，订正后的打卡在教师未点评前仅支持修改，不支持删除）；也支持在打卡记录中打卡或补卡。


如果打卡允许学生互相查看内容和成绩，那么学生选择日期后除了可以看到当天自己的打卡外，还可以在打卡动态中看到班级其他学生的打卡内容、分数和获赞数量，也支持对其点赞。


打卡结束后，学生不支持修改或删除自己的打卡内容。

# 十二、教师使用「打卡」


教师在待办或班级中找到进行中或已结束的打卡后，可以进入查看学生打卡并点评。进入打卡页面后可看到打卡排行榜，日历，打卡详情和学生打卡记录。


在打卡排行榜中，教师可看到打卡进度，每位学生的打卡天数、打卡率和排名，也支持对排名进行分享。如果教师使用电脑支持下载学生的打卡分析数据到本地，使用平板支持保存到我的云盘或发送给ClassIn班级/好友，数据包含每位学生的打卡成绩、打卡天数、补卡天数、最长连续打卡天数、获赞次数和获得优秀打卡次数。


在打卡日历中，教师可点击切换不同日期，查看当日学生的打卡情况；日期上会显示当日学生打卡的完成率，如果日期被完整绿色圆环包围说明当日学生全部提交了打卡，另外日期下如果显示红点，表示当日打卡需要教师点评。


在打卡详情中，教师可以看到发布的打卡内容以及打卡活动的开始和截止时间等信息。


教师可以在未交、待批、打回和已批中提醒或查看点评学生的打卡。对于学生已提交的打卡，教师支持对打卡内容点赞、点评、分享（分享本次打卡或打卡记录）和删除，也可以看到学生当前打卡的获赞数量。教师点击学生昵称后面的坚持天数可以进入到该学生的打卡记录中，也可以在排行中点击学生的打卡率或打卡天数进入打卡记录。打卡记录包含该学生的打卡天数、连续打卡天数、获得优秀打卡的次数以及获赞次数，可以看到学生每次的打卡内容、教师点评、分数和获赞数量，也可以对学生的打卡内容进行点赞、分享（分享本次打卡或打卡记录）和删除操作。
