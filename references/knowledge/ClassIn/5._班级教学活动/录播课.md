# 录播课


英文使用说明


# 一、什么是「录播课」？


“录播课”功能指教师预先录制教学视频并上传至“学习活动”中的“录播课”模块，供学生随时观看。此形式不受观看场景和设备的限制，通常用于课前预习、课后复习以及非实时授课类课程。


# 二、「录播课」与「录课回放视频」的区别？


在ClassIn平台中，“录播课”和“录课回放视频”是两种不同的功能。


“录课回放视频”功能指的是授课教师在教室上课过程中使用Classin的“云端录课”功能，从“授课教师”视角将授课过程录制并自动保存为“回放视频”，课后可以在“课程表”中进行回放观看。


“录播课”则是指教师事先录制好教学视频，并将其上传至“学习活动"中的“录播课”模块。这些视频可以供学生随时观看，适用于课前预习、课后复习以及非实时授课的课程。此形式不受观看场景和设备的限制。


# 三、创建「录播课」


### 适用人员

### 使用说明


录播课中支持上传的视频格式有：mp4格式；3gp格式；mpg格式；mpeg格式；3g2格式；avi格式；flv格式；wmv格式；h264格式；m4v格式；mj2格式；mov格式；ogg格式；ogv格式；rm格式；qt格式；vob格式；webm格式。


录播课中最多可以上传15个视频文件，每个视频文件最大不得超过2GB。


电脑上传录播课视频时支持将视频拖拽到录播课内容中。

### 注意事项


录播课视频的可观看时长取决于老师设置的截止时间、允许观看次数和是否允许活动结束后观看。另外如果“学生被教师移出班级”“班级解散”“学生主动离开班级”，已经离开的学生是否可以查看录播课视频取决于班级设置中允许离开班级的学生或班级解散后，可查看课程内容的设置。

# 四、修改「录播课」


草稿和未开始的录播课，“课程”无法修改，其他内容和设置都可以进行修改。


进行中的录播课，内容可以进行修改，设置选项中的“开始时间”和“课程”无法修改。


已结束的录播课，内容无法进行修改，设置选项中的时间、发布给、教师、课程、允许新进班学生观看、解锁条件这几个选项将无法修改。

# 九、查看「录播课」数据


在录播课中，教师可以查看或导出学生的录播课数据，以更好地了解学生的学习情况。针对当前录播课包括：学生参与率、视频完播率、平均观看时长和平均观看进度；


针对学生个人的录播课观看数据包括：观看时长、观看进度、发讨论数、发回复数、获赞数、成绩，如果教师用电脑，支持导出学生参与录播课情况，导出的数据除以上内容外，还包括学生观看次数和观看状态。


此外，教师可以查看讨论区的内容，如果教师用电脑，支持导出讨论区详情数据，导出内容将包含以下信息：讨论数量、画面时间戳节点、发布人、评论内容和评论时间。


手机和平板只能查看学生的录播课数据和讨论区内容，不支持导出。

# 十、学生使用「录播课」


## 适用人员


“学生”


## 使用说明


在老师发布录播课活动后，学生可以看到录播课活动的详情，包含录播课标题名、教师、录播课开始和截止时间以及观看次数（与教师设置录播课参数有关）。


学生观看录播课视频过程中，录播课视频支持播放暂停、切换其他视频、调整录播课视频音量、倍速播放（与教师设置录播课参数有关）、拖动进度条（与教师设置录播课参数有关）、全屏，如果教师开启了讨论区，学生可以针对视频的具体时间戳参与讨论，讨论支持添加文字和图片，也可以看到每个视频的讨论区，且可以在讨论区进行互动，也可以对其他人的讨论进行回复和点赞。如果是自己发布的讨论，自己也可以进行删除。


学生可以看到自己录播课学习的数据，数据包括观看进度，累计观看时长，发评论回复获赞数，成绩。点击查看详情还可以查看具体的视频观看明细，且还支持分享个人观看数据。


## 注意事项
