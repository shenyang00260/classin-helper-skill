# 课堂


英文使用说明


# 一、什么是「课堂」？


在ClassIn平台中，“课堂”是教师向学生传授知识的专属场所，亦称为课节或直播教室。课堂中，教师通过讲解、演示和提问等方式进行教学活动，旨在促进学生的学习与发展。课堂环境的设计与组织对提升教学效果和学生的积极参与至关重要。课堂内常用的功能包括座位席、黑板和各种教学工具，确保教学过程的顺利进行。


# 二、创建「课堂」


### 适用人员


“班主任”


开启《允许教师新建课堂》 功能后的“班级教师”

班级中创建课堂时，支持设置的内容如下：

### 注意事项


在班级内创建课堂时，教师可以选择上台人数。系统会默认使用当前设备上一次创建课堂时的选项。如果更换设备，则系统不会记录上一次的选择，需要重新设置。


如果用户添加课节或课程的时候出现无法添加的情况，可能是当前课程下的课节数量已经达到了上限。


不论课节时长是多少，正式课节的教室拖堂时间都是20分钟，临时教室的拖堂时间是2分钟。


如果“班主任”就是“授课教师”，那么创建课堂时就不需要单独设置授课教师了，因为在客户端中创建课堂时授课教师默认为班主任。


如果“学生被教师移出班级”“班级解散”“学生主动离开班级”，已经离开的学生是否可以在ClassIn查看录课回放视频取决于班级设置中允许离开班级的学生或班级解散后，可查看课程内容的设置。

# 三、修改「课堂」


具有创建课堂权限的授课教师


在ClassIn客户端中，班主任可以修改所有已创建的课堂，具有创建课堂权限的教师只能修改自己是授课教师的课堂。


具体可修改的信息和设置包括：课堂名称、开课日期、开课时间、课堂时长、授课教师、联席教师、上台人数、云端录课、网页直播、网页回放等。请注意遵循下方的相关规则及时间限制。


开课之前的20分钟之前可修改的设置有：课堂名称、开课日期、开课时间、课堂时长、发布班级/成员、OMO站播、上台人数、云端录课（录制ClassIn教室、录制现场）、网页直播、网页回放、展示座位席、学生自动上台。


课堂结束前可随时修改的设置有：更换授课教师、添加/更换联席教师、网页回放。


不受时间限制随时可修改的设置有：AI课堂总结、网页回放、单元主题、允许学生互相查看学习报告。


在正式开课前（包括提前进入教室的时间），课堂可以随时删除。然而，一旦课堂进入正式上课时间，就无法再被删除。


已结束的课堂无法删除。

# 六、设置「联席教师」


## 1、建课时添加联席教师


### 适用场景


如果上课的学生人数较多，班主任或有创建课堂权限的教师可以为课堂添加“联席教师”，联席教师可以在课堂中帮授课教师维护课堂秩序，协助授课教师完成授课。


为课堂添加联席教师前，需要让班主任先将联席教师添加进班级，并在班级中将其设置为班级教师身份，这样班主任或有创建课堂权限的教师才可以在建立课堂时添加班级教师成为联席教师。

# 七、更换「授课教师」


当课堂已经开始后，如果授课教师临时有其他事情，需要临时为本堂课更换授课教师，这时候班主任可以将该教室中的授课老师临时更换为其他人。


为课堂更换授课教师前，需要让班主任先将新的授课教师添加进班级，并在班级中将新进班的老师设置为班级教师身份，这样在更换授课教师的时候才能将这名老师设置为课堂的授课教师。

# 九、班级中观看「录课回放视频」


### 什么是回放视频？


“回放视频”是指在班级中创建课堂时勾选了“录制ClassIn教室”或“录制现场”（即云端录课），授课教师进入正式教室上课录制整个授课过程，这些视频将在下课后约半小时左右自动生成为“回放视频”。


### “录课回放视频”与“录播课”的区别？


“录课回放视频”是指授课教师利用ClassIn云端录课功能，从教师的视角全程录制授课过程的视频。这些视频将在课程结束后约半小时自动生成并可供回放查看。而“录播课”则是指班主任或教师在班级中创建的特定教学活动，教师可以将本地电脑中的视频上传到该“录播课”中，供学生和教师随时观看和学习。


班级教师、课堂学生


授课教师在课堂中录课正常，课后半小时后，班级中的所有教师及课堂学生可在班级中找到课堂观看回放视频。


播放视频时，可根据录课设置，选择播放录制的视频或录制的现场视频，如果回放包含多个视频片段，支持切换片段进行观看；


回放视频支持窗口化播放、倍速播放和调整视频音量，并允许拖动进度条以便灵活观看回放内容；


如果创建课堂时勾选了AI课堂总结中的章节总结和字幕列表，那么在观看回放时也可以查看到AI生成的本堂课总结和字幕列表，且支持复制；


如果创建课堂时勾选了录制ClassIn教室/录制现场，以及AI授课分析，那么授课教师完成录课且声音正常情况下，课后授课教师可以在课堂回放中看到关于本节课的AI授课分析，还能将分析下载为 PDF 保存。


插班生可以观看参与过的课堂“回放视频”。


如需为新进班的学生开通观看班级历史回放视频的权限，请在“后台管理系统”中，启用《新加入课程的学生允许观看历史课节回放视频》功能，启用后再将学生加入到班级中。


如需限制被移出班级的学生无法继续观看历史课节的回放视频，请在“后台管理系统”中启用《移出课程的学生无法观看历史回放视频》功能，启用后，再将学生从班级中移除即可。

# 十、班级中查看「板书课件」


### “板书课件”是什么？


当老师授课过程中在黑板或课件上输入文本或绘画标注时，ClassIn会自动保存老师的板书和课件。被自动保存的板书和课件可以在课后的“课节详情”或“课程表”中查看。老师和学生可以方便地回顾和复习课堂内容。


「板书课件」中包括「课堂板书」和「带批注课件」。「课堂板书」是正式教室中在黑板上书写的板书内容，「带批注课件」是正式教室中打开的课件。「课堂板书」或「带批注课件」只要生成其一，课后就会显示「板书课件」入口。


「板书课件」的生成有一定条件：


首先，需要在www.eeo.cn管理后台开启板书课件留存功能；


其次，正式教室老师授课过程中，在黑板或在打开PPT/PDF/Word(docx/doc)/Epub课件（包括被授权允许转存载的课件）上输入任意数量文本时，或画笔标注超过10个元素（PPT课件需单页画笔标注超过10个元素）时，ClassIn课后会自动保存带有批注的整个板书内容和整个课件。


班级的老师和课堂学生课后可以在班级中查看这节课的「板书课件」，另外这节课的授课教师、联席教师以及课堂学生还可以在课程表中查看这节课的「板书课件」。


这节课的授课教师和联席教师支持分享「板书课件」到ClassIn班级/好友、微信，使用手机或平板还支持分享到微博、QQ 等中，也可以复制链接发给他人；另外也支持将生成的「课堂板书」以jpg图片保存到ClassIn的个人云盘中。


黑板上或课件上满足保存条件后，对板书操作Ctrl+n或一键清屏后，会留存清屏前的板书内容；


「带批注课件」最后以PDF格式保存，预览缩略图是课件的第一页；


只有正式教室满足条件课后才会生成「板书课件」。

# 十一、班级中查看「教学报告」


正式教室的授课教师或联席教师正常出勤的前提下，课堂结束半小时后，会生成本堂课教师的「教学报告」，课堂的授课教师、联席教师和班主任可以在ClassIn班级中查看「教学报告」，课堂的授课教师和联席教师也可以在课程表中查看。


「教学报告」默认显示的是新版教学报告，最上方也支持切换到旧版报告中使用；支持切换12种语言查看其内容；显示课堂信息，包含授课教师的头像和班级昵称、课堂开始时间和上课时长；


「教学报告」数据内容包括考勤情况、课堂参与率、课堂活动时间分布、课堂记录和课堂成就5部分。


考勤情况：可查看到本堂课学生的出勤、迟到、早退人数数据；查看明细还可以看到学生具体出勤情况、上课时长和学生的学习报告，如果使用电脑还支持导出考勤数据。


课堂参与率：显示学生在教室内使用答题器、小黑板、互动消息（教室中聊天框的聊天）和抢答器的参与率；查看明细中可以具体查看学生的参与详情，如果使用电脑还支持导出明细数据。


课堂活动时间分布：显示教师在教室中使用教学工具的时间分布情况，分为过程性评价（包括答题器、小黑板、抢答器）、教学互动（包括随机选人、举手、分组讨论、互动消息、计时器、定时器等）、课堂激励（分发给学生奖杯）四部分；查看明细中可以更直观地看到工具的使用时长和次数。


课堂记录：包括教室中的课堂板书、精彩瞬间（监课图片）和课堂笔记。


课堂板书是黑板上书写的板书内容，生成课堂板书图片的条件是黑板上画笔标记元素大于10或包含文本输入内容，另外教师Ctrl+n或操作一键清屏也会保存之前满足生成条件的板书图片；


精彩瞬间（监课图片）是教师教室窗口自动保存的图片，可用于回顾教室内讲解知识的过程；


课堂笔记是使用电脑的教师在教室中使用课堂笔记功能后保存的笔记图片。


如果教师使用手机或平板查看课堂记录中的图片时，支持长按保存图片到本地相册中。


课堂成就：包括学生作答答题器（随堂练习）的正确率排名以及得到教师发放奖杯的排名数据；查看明细中可以看到学生的数据情况，如果使用电脑还支持导出明细数据。

## 3. 学生课后修改评价


关闭正式教室后，课堂的学生默认会弹出一个评价窗口，在评价窗口中，学生可以对教师的表现进行星级评分和文字评价。如果课后学生需要填写或修改评价，那么进入课堂详情的课后评价中，点击评价教师即可查看或修改评价，评价支持多次修改提交。


本堂课的授课教师/联席教师对学生的评语操作屏蔽后，该学生将无法修改对教师的评语内容，只能修改星级评分。

# 十五、班级中查看「AI授课分析」


Classin AI授课分析基于课堂的录课视频，分析老师的表达及互动情况，对老师上课过程进行再现，并针对授课表现的多个维度进行AI评价和评分。


eeo后台管理人员在“学校设置-增值服务”中打开AI授课分析开关，然后在班级中创建课堂时勾选了录制ClassIn教室/录制现场，以及AI授课分析，那么授课教师完成录课且声音正常情况下，课后授课教师可以在课堂回放中看到关于本节课的AI授课分析，还能将分析下载为 PDF 保存。


AI授课分析报告中包含课堂概况、课堂再现（教学时间分配、时间分配时序图、课堂关键词云、课堂高频词表）、课堂数据详情（语言表达效果分析、教学语言逻辑与结构分析、互动分析）、课堂整体AI评价及优化建议。AI授课分析报告的内容仅支持中文，不支持多语言。
