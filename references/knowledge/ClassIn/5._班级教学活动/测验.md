# 测验


英文使用说明


# 一、什么是「测验」？


ClassIn中的“测验”功能，本质上就是“试卷”，旨在评估学生在特定学科或领域中的知识和技能掌握情况。考试不仅是检验学生学习成果的重要手段，还能帮助学校和教育机构评估教学质量和课程设置的有效性。此外，测验能够帮助学生自我检测学习进展和薄弱环节。通过参加测验，学生能够了解自己在各个学科中的掌握程度，并及时发现不足。这有助于学生明确学习重点，提高学习效率，迅速弥补知识缺陷。


# 二、创建「测验」


## 1、创建测验活动


### 适用人员


“班主任”和“教师”


### 使用说明


进入测验中的添加试题界面：


教师可以从“题库选题”中添加“ClassIn题库”提供的试题或试卷，支持通过关键词或条件，筛选试题与试卷；


如果教师自己提前录入过试题，可以在“我的题库”中调用提前录入的试题与试卷；


如果学校给教师授权了试题试卷资源，教师可以在“组织题库”中调用被授权的试题试卷资源。


将需要的试题与试卷加入试题，在试题篮中可以针对已加入的试题进行删除，再添加到当前试卷即完成选题。


教师还可以在“添加/批量添加试题”中手动添加单题或批量添加试题完成试题的录入，由于批量添加试题须符合特定格式才能被ClassIn识别，因此批量添加试题时，需要教师先下载模板，根据模板和录入说明进行添加试题，将添加好的试题保存后再进行上传即可。


教师使用电脑和平板可以基于教学目标或特定的知识点，由AI智能体出题。


添加好的试题支持设置分数，移动试题位置，删除试题，编辑，复制等操作。支持预览，可以查看电脑端和手机端学生查看试卷的效果。还支持保存试卷资源到我的题库，如果教师使用电脑也支持下载试卷到电脑本地。

## 2、批量添加试题


### 适用场景


当教师需要将大量试题导入ClassIn时，可以使用“批量添加试题”功能。


在批量添加试题-上传试题中，可以将本地电脑中满足录入说明的Word或Excel文档中的试题直接复制并粘贴到“批量添加试题”窗口中。也可以将符合录入模板的本地Word或Excel文档拖拽到该窗口中使用。


在批量添加试题-编辑试题中，支持对试题进行编辑删除，也支持将其保存到我的题库中。


#### （1）单选题格式


题号格式：


题号必须为阿拉伯数字


题号后必须加标点符号或空格，例如：1. 或 1、或 1空格


题干格式：


题干可以包含文字和图片，文字最多15000字符


选项区域格式：


选项序号必须为大写字母


每个选项后必须加标点符号或空格，例如：A. 或 A、或 A空格


选项数量为2-26个，用A-Z按顺序标记


选项内容可以包含文字和图片，文字最多200字符


答案区域格式：


格式必须使用右侧的格式，例如：答案：A


解析区域格式（非必填项）：


解析区域格式必须使用右侧的格式，例如：解析：为什么是A


解析内容可以包含文字和图片，文字最多15000字符

# 三、修改「测验」


测验发布后，测验中的试题、标题不支持修改。


草稿和未开始的测验，“课程”无法修改，其他内容和设置都可以进行修改。


进行中的测验，测验内容以及设置选项中的开始时间、答题限时、课程、填空题手动批阅将无法修改。


已结束的测验，测验内容以及设置选项中的时间、答题限时、发布给、教师、课程、填空题手动批阅、允许新进班学生提交、解锁条件这几个选项将无法修改。


草稿、进行中、已结束的测验均支持保存试卷资源到我的题库。

# 七、分享「测验」


老师可以在学生未登录ClassIn或未开启ClassIn通知功能的情况下，将布置的测验分享到微信或其他平台中，便于老师及时督促学生学习。


当教师使用电脑分享测验时，可以将测验分享到ClassIn班级或好友中，或者复制链接进行分享，还可以通过手机微信扫描二维码分享给微信好友或微信群中。

# 八、学生写「测验」


找到测验：在班级中课程栏目下找到教师布置的测验。


进入答题界面：点击测验右侧的“开始作答”按钮，学生将进入答题界面。在这个界面的顶部，学生可以看到试题数和总分数。同时，在界面左下角，还会显示答题剩余时间。这些信息对学生进行答题和控制时间至关重要。


答题过程：学生现在可以开始答题了。根据试题，学生在界面中选择正确的答案或填写答案。一旦学生完成了一个试题，系统会自动在右侧的答题卡上会标记已回答的试题，以帮助追踪已经回答和未回答的试题，右侧的答题卡也支持点选，便于学生快速跳转到对应题目处。。


提交测验：当学生完成答题后，可以点击界面右下角的“提交测验”按钮。这将意味着学生已经结束了答题过程，试卷将被自动收回。


学生提交测验后，不支持对测验答案进行修改，所以请学生在提交测验之前仔细检查答案。


对于限时测验，中途退出也会继续计时，剩余时间结束后系统会自动交卷。

# 九、批阅「测验」


“批阅测验”是指教师对学生的测验试卷进行评估和评分的过程。如果学生实际上是在线下完成的测验，教师也可以进行批阅。在批阅测验时，教师会根据测验中的题目要求和标准答案，对学生的答案进行详细的分析和评价。教师会考虑学生的理解和解题能力，准确度和完整度，语言表达和逻辑推理等方面，从而给予相应的得分或评级。


当学生提交测验后，教师可以在班级页面的课程栏目下找到对应的测验，并点击右侧的“去批阅”按钮。这时教师将进入“待批阅”栏目，可以针对每道题目逐一判断其正误。除了对错外，多选题和问答题也支持半对。教师可以根据学生的答案，与试题的答案解析，在自动批阅的基础上进行人工判断，对于系统已自动批阅的判断题、选择题等客观题，也支持教师修改批阅结果。此外，右侧的答题卡也支持点选，便于教师快速跳转到对应题目处。


批阅完成后，教师可以点击“完成”或“保存并批阅下一个”，即完成当前学生的批阅或继续进行下一位学生的批阅。如果教师选择“保存并批阅下一个”，系统将自动跳转到下一位学生的答卷页面。


学生提交的测验教师可多次进行批阅，不会留存中间的批阅记录，以最终记录为准。

# 十、查看下载「测验数据」


在后台管理系统中，教师可以下载测验数据。此外，在软件中，教师还能够直接查看或下载测验数据。教师可以根据这些数据进行分析和评估学生的学习情况，以及进行个性化教学和提供针对性的反馈。


查看测验数据时，可以按照每道题目进行查看，并支持按题型进行筛选。点击某一道题目后，教师可以看到该题的正确答案、当前题目的正确率和总正确率、每个选项的选择人数以及各选项的选择比例等详细信息。


电脑导出学生的测验明细数据后，数据将以Excel格式保存到本地计算机中。表格中包含的学生信息如下：学生班级昵称、是否提交测验、学生是否补交过测验、学生提交测验的时间、学生的答题时长、学生的测验是否已被批阅、学生的测验分数、学生的正确率、学生答对的题目数量、学生答错的题目数量、学生部分答对的题目数量、学生未作答的题目数量、学生每道题的得分、学生每道题的所选答案，上述数据将针对每位学生进行区分。并附加以下信息：班级ID、班级名称、测验标题、单元ID、单元名称、试题数、测验满分、批阅人。


手机和平板只能查看测验数据，无法导出学生的测验明细数据。
