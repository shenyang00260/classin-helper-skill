# 讨论


英文使用说明


# 一、什么是「讨论」？


“讨论”指的是在班级中创建一个讨论组。在这个讨论组中，班级内的所有学生和教师都可以自由地提出问题、分享观点和意见、解释概念或发表感想。这样的讨论组营造出一种开放和包容的氛围，鼓励学生主动思考和表达，同时也为教师提供了了解学生思维方式和学习需求的重要机会。此外，学生还能够通过与其他同学交流，分享彼此的知识和经验，促进共同学习。这样的互动与合作有助于激发学生的学习热情和动力，培养他们的批判性思维和解决问题的能力。


# 二、「讨论」与「分组讨论」的区别？


ClassIn中有两个讨论功能，教室外的班级中有一个讨论功能，这个讨论创建后班级教师和学生都可以参与。另一个讨论功能是在教室中，名为“分组讨论”，该功能主要是将学生以小组形式分成多个小组，小组之间教室中讨论，并且也支持教师参与各组学生的讨论。


# 三、创建「讨论」


### 适用人员


“班主任”和“教师”


### 使用说明


在新建讨论时，教师支持在讨论内容中输入文字和上传各类附件，如图片、录音、云盘文件或本地文件、链接。需要注意的是，附件的数量上限为：图片附件15张、录音附件15条，而本地文件、云盘文件和链接的总数也不能超过15个，每个文件大小≤500M。


电脑上传讨论附件时支持将图片以及文件拖拽到讨论中。


创建“讨论”时，可设置的参数请参考下方：


| 设置 | 功能 |
| --- | --- |
| 时间 | 设置讨论的开始和结束时间。 |
| 评分 | 设置讨论的评分制度和分数。 |
| 发布给 | 可以将讨论发布给特定班级、班级中的部分小组成员或部分学生，也可以同时发布给多个班级。 |
| 教师 | 指定活动教师，接收相关通知。 |

# 四、修改「讨论」


讨论支持编辑，但是不同状态下的讨论中可编辑的内容有所不同。


草稿和未开始的讨论，“课程”无法修改，其他内容和设置都可以进行修改。


进行中的讨论，内容可以进行修改，设置选项中的“开始时间”和“课程”无法修改。


已结束的讨论，内容不可以再进行修改，设置选项中的时间、发布给、教师、课程、允许新进班学生参与、解锁条件这几个选项将无法修改。

# 八、分享「讨论」


教师可以将进行中或已结束的讨论活动分享到微信或其他平台中，便于督促学生及时参与讨论。


当教师使用电脑分享讨论时，可以将其分享到ClassIn班级或好友中，或者复制链接进行分享，还可以通过手机微信扫描二维码分享给微信好友或微信群中。


当教师使用手机或平板分享讨论时，可以将其分享给ClassIn的班级或好友，也可以将其分享到微信好友或微信群、朋友圈、微博、QQ好友中，以及通过复制链接进行分享。


在其他平台查看老师分享的讨论，其中的图片、录音、视频可直接查看播放，其他文件需唤起 ClassIn 进入班级查看。


此外教师在分享讨论时，如果已有用户发表了评论，那么打开分享的讨论中也会显示最热tab下第一个一级评论内容。

# 九、学生参与「讨论」


“学生”


学生可在待办或班级中查看、参与讨论。


评论区分为最热和最新tab，评论区中


支持点赞、发表和回复（自己或他人）评论，评论中支持输入文字、上传图片录音和音视频文件，需要注意的是，附件的数量上限为：图片附件9张、录音附件9条，音视频文件的总数也不能超过9个，每个文件大小≤500M，学生使用电脑也支持拖拽图片文件到评论中。


支持在自己或他人评论后点击只看自己自己或此人的评论，支持删除自己回复或发表的评论，使用手机时还支持复制评论。


如果学生被教师操作禁言，那么学生在发表回复评论时会提示你已被禁言或禁言中，禁言状态下不支持点赞、发表回复评论。


学生区中，学生可看到自己的评论、回复、获赞数量和成绩，如果使用电脑或平板也可以分享自己的讨论报告；如果教师设置允许学生互相查看成绩，那么学生也可以看到其他同学的讨论情况。


讨论结束后，学生不支持发表回复、点赞和删除评论。

# 十、老师参与「讨论」


教师在待办或班级中找到进行中或已结束的讨论后，可以进入查看、参与讨论及评分。


支持点赞、发表和回复（自己或他人）评论，评论中支持输入文字、上传图片录音和音视频文件，需要注意的是，附件的数量上限为：图片附件9张、录音附件9条，音视频文件的总数也不能超过9个，每个文件大小≤500M，使用电脑时还支持拖拽图片文件到评论中。


支持在自己或他人评论后点击只看自己或此人的评论；班主任支持删除所有人的评论，如果是班级教师支持删除自己和学生的评论（不支持删除班主任评论）；使用手机时还支持复制评论。


支持禁言学生，学生被教师禁言后在发表回复评论时会提示已被禁言或禁言中，禁言状态下学生不支持点赞、发表回复评论。


在学生区中，教师可以看到对应学生的评论、回复、获赞数量和成绩，使用电脑或平板也可以分享对应学生的讨论报告，另外使用电脑还支持下载讨论内容。


如果讨论设置了评分，那么教师在讨论区的学生评论后和学生区中均支持对其评分。
