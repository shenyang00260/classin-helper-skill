# AI口语卡


英文使用说明


# 一、什么是「AI口语卡」？


在英语教育培训行业中，听、读、说、写四大重要学习系统中，阅读和口语的客观评分或评价一直是个难题。因为它们涉及到发音、词汇、语调、语法和流畅度等多个因素，人工评分不仅工作负荷大，对评分者的体力和脑力也是巨大挑战。此外，由于评分者的状态不同，评分结果也会有波动，从而增加机构的运营成本。AI口语评测可以全面提升学生的口语能力，同时为教师的批阅工作带来高效和省时的解决方案。

# 二、创建「AI口语卡」


### 适用人员


“班主任”和“教师”


### 使用说明


教师使用电脑或平板支持在班级中新建AI口语卡。


新建口语卡界面中老师可以点击添加卡片来布置题目，每道题目可以输入一个单词或一个句子（限150字），按回车键会自动识别生成音频和释义，如果输入的是单词还会生成音标，最多可以添加100道练习题。老师还能通过点击“批量添加”添加题目，批量添加时可以输入卡片内容，用换行方式隔开每道题，如果老师使用电脑还可以用Excel导入添加。除此之外还可以通过AI智能添加，在AI界面中输入具体要求后会点击生成，可以将生成内容应用到口语卡中。


当老师录入口语卡时，系统会自动生成录入单词或句子对应的音频、音标和释义，如果对识别出的音频、音标、释义不满意，可以将鼠标悬浮在文本框上进行删除。删除后，老师可以点击自动生成重新生成内容或手动输入内容，如果是音频还支持重新录音、上传音频或重新自动生成。另外音标和释义为选填项，不填写不会影响正常发布和评分。


题目可以通过拖拽方式调整顺序，方便老师按需排列。

# 三、修改「AI口语卡」


教师使用电脑或平板支持对AI口语卡编辑，但是不同状态下的AI口语卡可编辑的内容有所不同。


草稿和未开始的AI口语卡，“课程”无法修改，卡片内容和设置都可以进行修改。


进行中的AI口语卡，设置选项中的“开始时间”和“课程”无法修改，针对卡片内容是否修改，取决于AI口语卡中的练习题目学生是否有朗读，如果学生有朗读，那么教师不支持修改卡片内容，学生未朗读，那么教师支持修改卡片内容。


已结束的AI口语卡，卡片内容不可以再进行修改，另外设置选项中的时间、发布给、教师、课程、允许新进班学生提交、解锁条件这几个选项也无法修改。


进入 课程


找到AI口语卡，点击「···」 ，然后选择 编辑

# 六、复制「AI口语卡」


草稿、未开始、进行中和已结束的AI口语卡活动均可复制到本班级或其他班级。复制后的AI口语卡中，除开始时间和截止时间外，内容和参数默认和原AI口语卡一致。


教师只有使用电脑和平板才支持复制AI口语卡活动。


找到AI口语卡，点击「···」，选择 复制活动

# 七、分享「AI口语卡」


教师可以将进行中或已结束的AI口语卡分享到微信或其他平台中，便于督促学生及时朗读提交AI口语卡。


当老师使用电脑分享AI口语卡时，可以将其分享到ClassIn班级或好友中，或者复制链接进行分享，还可以通过手机微信扫描二维码分享给微信好友或微信群中。


当老师使用手机或平板分享AI口语卡时，可以将其分享给ClassIn的班级或好友，也可以将其分享到微信好友或微信群、朋友圈、微博、QQ好友中，并且也支持复制链接分享给其他人。


在非ClassIn APP中查看分享内容时，支持播放老师发布的口语卡音频等内容，如果学生需要朗读提交仍需唤起ClassIn进入班级使用。

# 九、学生提交「AI口语卡」


教师发布AI口语卡后，学生可以进入ClassIn班级或待办中找到AI口语卡查看并朗读提交。


学生在AI口语卡朗读界面中，可以查看题目，在每道题中可以播放系统音频，可以自行朗读录音，朗读完其中的某道题目后，系统会自动给出星级评价，并用颜色标出读得不准确的词汇（红色代表较差或漏读，黄色代表良好，绿色代表优秀，参考下图），也支持播放自己朗读的音频，另外学生还可以选择重新录制或继续录制下一道题目。

学生完成所有朗读后可以点击提交，如果设置了评分，提交后会显示成绩。

# 十一、批阅「AI口语卡」


学生提交AI口语卡后，教师可以在班级或待办中找到AI口语卡进行批阅。


如果AI口语卡设置了评分方案，系统会根据学生朗读结果自动生成评分（参考下图，学生的每一道题目系统都会自动给出星级评价，并用颜色标出读得不准确的词汇），那么教师可以在AI口语卡的已批中再次批阅学生的口语卡；如果AI口语卡没有设置评分方案，那么教师可以在AI口语卡的待批中批阅学生的口语卡。

在批阅过程中，教师可以播放学生每道题的录音，根据录音进行点评。在点评中支持输入文字和录音评语，如果学生的朗读表现出色，教师可以将其设为优秀，如果设置了评分，也可以对其口语卡评分。


教师批阅完成后，可以点击提交。如果需要学生重新朗读提交，可以点击打回订正。


教师可多次批阅学生提交的口语卡，在学生的历史记录中会留存学生提交、系统批阅和教师批阅记录，如果不需要学生看到部分批阅记录，教师可以在学生的历史记录中操作删除。

# 十二、查看下载「AI口语卡」数据


教师进入AI口语卡的详情页面，可以查看到当前口语卡整体的朗读分析和学生数据，如果教师使用电脑也支持下载学生数据明细。


朗读分析：教师点击柱状图可以跳转到具体题目卡片，方便老师定位班级普遍存在的问题，在手机上，朗读分析位于活动详情页的“朗读内容与数据”部分。


学生数据：包含当前AI口语卡学生的总提交率、最高分、平均分及优秀数量的统计。


教师使用电脑支持在学生数据后下载全部数据即学生数据明细到电脑本地，数据包含每位学生提交、批阅、得分情况等。


手机和平板只能查看朗读分析和学生数据，不能下载学生数据明细。
