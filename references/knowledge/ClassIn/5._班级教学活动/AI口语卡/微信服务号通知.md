# 微信服务号通知


# 一、绑定微信公众号


### 适用人员


所有人


### 适用场景


当用户绑定ClassIn微信服务号后，即使未登录ClassIn应用或未开启ClassIn软件通知，也能通过微信接收上课通知和班级中LMS学习活动的通知，确保不会错过任何重要信息。


### 使用说明


用户绑定微信服务号后，ClassIn将在开课前通过微信服务号向老师和学生发送提醒消息。具体推送时间如下：


（1）付费版ClassIn上课通知：


教师：将在开课前20分钟收到上课通知。


学生：将在开课前10分钟收到上课通知。


（2）LMS活动通知：


当老师发布LMS学习活动，或对学生已提交的LMS学习活动进行批阅、催交或订正操作后，学生将会收到微信消息推送。


当学生提交LMS学习活动后，老师将会收到微信消息推送通知。


通过这些精准的消息推送，ClassIn确保所有师生都能及时获取重要的上课和学习活动信息。


### 注意事项


免费版教室，老师，联席教师，学生，旁听生均为开课前5分钟。


微信服务号每天最多有100万次推送，超过将被拦截。


课程或课节名字超过24个字也会被拦截。


### 具体操作步骤


微信右上角，点击 放大镜


搜索栏中输入 ClassIn服务号 ，在搜索结果中点击 ClassIn服务号


进入ClassIn服务号


点击 关注公众号


点击 绑定账号


点击 确认绑定


输入 手机号 或 邮箱 或 ClassIn账号密码 绑定


当有开课或LMS活动提醒时，会在微信服务号中提示
