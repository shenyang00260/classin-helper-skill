# 答题卡


英文使用说明


# 一、什么是「答题卡」？


在ClassIn平台中，“答题卡”是一种常用于考试的答题工具。相较于传统“测验”中复杂的题目录入方式“答题卡”无需录入试题内容，只需上传试卷图片并添加对应的答题选项即可，大大降低了录题成本。“答题卡”支持单选题、多选题、判断题、填空题和问答题等多种题型。班主任和教师均可创建答题卡，为学生提供便捷的答题体验。


# 二、创建「答题卡」


### 适用人员


“班主任”和“教师”


### 使用说明


教师在进入班级页面后，可以在课程栏目下建立答题卡。在试卷文件和答案文件中，对于上传附件的数量，有一定的限制。例如，最多上传15张图片、最多录制15条录音、最多上传15个本地或云盘pdfword音频文件。电脑上传支持拖拽。


在答题卡中，可以将已添加好的答案选项保存为答题卡模板，也支持从已保存的模板中进行选择使用。


答题卡中支持添加的试题类型有：单选题、多选题、判断题、填空题、问答题。最多支持添加200道题的答案选项。支持批量添加试题，也支持添加结构说明。


教师使用电脑和平板可以基于试卷文件，由AI解析生成答题卡，支持解析的是图片、pdf、word的文件格式。

# 三、修改「答题卡」


草稿和未开始的答题卡，“课程”无法修改，其他内容和设置都可以进行修改。


进行中的答题卡，不支持增删答题卡试题类型和选项，设置中的开始时间、答题限时、课程也无法修改。


已结束的答题卡，不支持增删答题卡试题类型和选项，设置中的时间、答题限时、发布给、教师、课程、填空题手动批阅、允许新进班学生提交、解锁条件这几个选项也无法修改。


草稿、进行中、已结束的答题卡都支持保存答题卡模板，也都支持修改选项中设置的答案。


手机暂不支持修改答题卡。

# 八、学生使用「答题卡」


找到答题卡：在班级中课程栏目下找到教师布置的答题卡。


进入答题界面：点击答题卡右侧的“开始作答”按钮，学生将进入答题界面。在这个界面的左侧是试卷文件页面，支持对试卷界面进行隐藏和展开，试卷界面支持切换文件图片、放大缩小还原文件图片，左下角支持播放录音音频文件。在这个界面的右侧是答题卡界面，学生可以看到试题数和总分数。同时，在界面左下角，还会显示答题剩余时间。这些信息对学生进行答题和控制时间至关重要。


答题过程：学生现在可以开始答题了。根据试卷文件，学生在界面中选择对应试题的正确的答案或填写答案。一旦学生完成了一个试题，系统会自动在右侧的答题列表中会标记已作答的试题（平板会显示在左侧，手机会显示在右上角），以帮助追踪已经回答和未回答的试题，试题列表中的试题也支持点选，便于学生快速跳转到对应题目处。


提交答题卡：当学生完成答题后，可以点击界面右下角的“提交”按钮。这将意味着学生已经结束了答题过程，试卷将被自动收回。


学生提交答题卡后，不支持对答题卡答案进行修改，所以请学生在提交答题卡之前仔细检查答案。


对于限时答题卡，中途退出也会继续计时，剩余时间结束后系统会自动交卷。

# 九、批阅「答题卡」


“批阅答题卡”是指教师对学生的试卷进行评估和评分的过程。在批阅答题卡时，教师会根据答题卡的试卷和答案，对学生的答案进行详细的分析。教师会考虑学生的理解和解题能力，准确度和完整度，语言表达和逻辑推理等方面，从而给予相应的得分或评级。


当学生提交答题卡后，教师可以在班级页面的课程栏目下找到对应的答题卡，并点击右侧的“去批阅”按钮。这时教师将进入“待批阅”栏目，可以针对每道题目逐一判断其正误。单选题、多选题、判断题根据答题卡选项设置的答案系统自动批阅，不支持批阅过程中对其结果进行修改（如某一选项答案设置错误，建议教师修改答题卡中的选项答案）；填空题和问答题支持人工手动批阅，除了对错外，也支持半对。此外，右侧的试题列表也支持点选，便于教师快速跳转到对应题目处。


批阅完成后，教师可以点击“完成”或“保存并批阅下一个”，即完成当前学生的批阅或继续进行下一位学生的批阅。如果教师选择“保存并批阅下一个”，系统将自动跳转到下一位学生的答卷页面。


学生提交的答题卡教师可多次进行批阅，不会留存中间的批阅记录，以最终记录为准。

# 十、查看下载「答题卡」数据


## 适用场景


在ClassIn软件中，教师还能够直接查看或下载答题卡数据。教师可以根据这些数据进行分析和评估学生的学习情况，以及进行个性化教学和提供针对性的反馈。


查看答题卡的数据统计时，可以按照每道题目进行查看，并支持按题型进行筛选。点击某一道题目后，教师可以看到该题的正确答案、当前题目的正确率和总正确率、每个选项的选择人数以及各选项的选择比例等详细信息。


电脑导出学生的答题卡明细数据后，数据将以Excel格式保存到本地计算机中。表格中包含的学生信息信息如下：学生班级昵称、是否提交答题卡、学生是否补交过答题卡、学生提交答题卡的时间、学生的答题时长、学生的答题卡是否已被批阅、学生的答题卡分数、学生的正确率、学生答对的题目数量、学生答错的题目数量、学生部分答对的题目数量、学生未作答的题目数量、学生每道题的得分、学生每道题的所选答案，上述数据将针对每位学生进行区分。并附加以下信息：班级ID、班级名称、答题卡标题、试题数、答题卡满分、批阅人。
