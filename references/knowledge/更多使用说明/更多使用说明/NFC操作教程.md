# 常见问题


1.贴卡后系统没反应


检查第一步的设置信息是否存在，绑定第三方 NFC 卡片，必须设置机构信息


软件确认是最新版


以上两步没有问题时，可以联系硬件同事确认读卡器是否正常工作


2.支持哪些设备


理论上凡是带 NFC 芯片的卡或电子设备都可以


3.手机 NFC 刷卡绑定后，还是提示需要扫码绑定认证


手机 NFC 需要绑定一张虚拟卡（公交卡、银行卡、电子门禁卡），作为绑定卡片使用，安卓系统一般可以自建空白虚拟卡。


如果不绑定任何卡片，手机 NFC 模块每次都会自动生成新的临时虚拟卡号，所以会提示每次都需要绑定
