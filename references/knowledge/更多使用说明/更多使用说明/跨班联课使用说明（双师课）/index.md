# 跨班联课使用说明（双师课）


# 一、什么是双师课？


双师课是一种创新的教学模式，通过ClassIn平台将不同班级的学生整合在一起，共同参与学习。这种方式不仅能够突破传统班级的界限，还能够显著提高教学和学习的效率与效果。通过打破空间的限制，双师课可以将名师课堂无限复制，让更多学生受益于优质教育资源。


# 二、适用场景


教学管理：名师可以同时面向多个小班进行直播授课，大幅提升名师效能，同时保证了优质教学资源的完成留存


课程效果：各小班的助教老师既可以与学生进行班内互动，也能够在课前，课后批改作业，发放资料灯，实现个性化管理


服务运营：满足家长加强课堂跟盯及学生学习行为把控的需求，教学系统全程记录全体学生的表现数据与进步情况


招生续费：大班的规模和价格，却带来小班的体验与效果，助力机构树立差异化优势，打造机构口碑并实现成倍增收


# 三、使用说明


双师教室：由一个主教室与多个子教室组成。每个主教室最多可关联100个子教室。


影像与声音：在授课过程中，主教室的“授课教师”和“被连麦的学生”影像和声音将传输到所有教室，除此之外，子教室中的学生和助教，无法看到或听到主教室中其他人员（包括助教老师）的影像与声音，也无法看到或听到其他子教室中的学生和助教的影像与声音。同时主教室中的授课教师、助教和学生也无法看到或听到其他子教室中的学生和助教的影像与声音。


文字聊天：在主教室中，所有教师和学生均可通过聊天工具查看其他所有子教室中助教或学生发送的文字消息。而在子教室中，助教和学生只能看到本教室内成员发送的文字消息以及来自主教室“授课教师”的文字信息，无法看到其他子教室的消息，也无法看到主教室中除“授课教师”外其他人员（包括助教）发送的文字消息。


一键重置功能：在开课前，子教室中的学生可能会被授予在黑板上书写和绘画的权限。为了确保主教室和子教室内容的一致性，“授课教师”在进入主教室后，可以在讲课前使用“一键重置”功能。具体操作为：点击右上角的四个点图标，选择“重置”选项。该功能将执行以下操作：清空所有教室的板书内容、关闭所有课件和工具、使所有教室的助教和学生全部下台、清除所有学生的授权状态、重置奖杯数和学生上台次数。这样可以确保课程开始时，所有教室的状态一致，为教学活动的顺利进行提供保障。


主教室可用教学工具：鼠标模式、选择/移动工具、画笔、橡皮擦、文本录入、截图、激光笔、云盘、课堂笔记、聊天工具、花名册、打开本地文件、保存/分享板书、投具、屏幕共享、骰子、计时器、定时器、浏览器、物理/化学实验、奖励排行榜(仅显示主教室排行榜)、教学素材库、辅助摄像头、拖拽激光笔、尺规工具、几何图形、双师答题器、连麦、一号双端（仅支持PC/iPad，建议使用PC作为主设备）。


主教室禁用工具与功能：LMS、小黑板、分组讨论、VNC、直播聊天、随机选人、抢答器、视频墙（包括单独观看主教室学生与观看所有子教室学生）、学生屏幕共享、协作文本、多向浏览器。


在正式开课前，需要进行一次全面的模拟演练，包括排课、备课、上课、维持课堂秩序、录课以及学生课后复习等所有环节。确保各个环节无误后，方可正式开课。


# 四、教务必看


## 1. 如何开通双师课服务


如需开通双师课程服务，请联系您的客户经理。


## 2. 后台建课


注意：双师课程无需单独创建班级，可以使用现有的班级进行授课。如果确实需要新建班级，只需按照普通班级的建课流程进行创建即可。


### 2.1 创建主课堂


在主班级中选定一个课程（注意，不是整个班级），并将其作为主课堂的基础，创建它的子课堂。如果班级中尚未添加任何课堂，可以参考下方图示进行创建。（创建路径：登录后台管理 ＞ 班级管理 ＞ 进入班级 ＞ 添加课堂）

### 2.2 添加子课堂


进入到班级里，勾选已创建的课堂作为主课堂，点击【批量新建子课堂】。在添加子课堂的过程中，您需要指定子课堂所属的班级。可以参考下方图示进行创建。（选择所属课班级时，可以通过上方搜索栏搜索班级名称或者按照标签搜索，效率更高。）


注意事项：


（1）每个主课堂最多可创建100个子课堂。（单次最多创建50个，超过50个时，需要创建2次）


（2）若距离主课堂开课时间不足5分钟或已过开课时间，则无法创建子课堂。


（3）可以批量为所有子课堂添加联席教师，或者为每个子课堂单独添加联席教师。（联席教师为可选项，非必需。如果不小心添加错误，可以使用“全部移除”功能移除联席教师）


（4）如果联席教师在课堂上需要同时查看多个班级的聊天情况，可以将该联席教师安排为 主课堂 和 多个子课堂 的联席教师，方便联席教师在主课堂里查看整体的聊天区。


（5）创建完成后，子课堂的非公共部分的信息，可以在相应班级中进行修改。


（6）在录课方面，可以选择【使用主课堂的录课回放】或【云端录课（各子课堂分别录课）】。需要注意的是，若选择子课堂云端录课，每个录课的子教室都需要有一名助教进入教室进行全程录制。

### 2.3 编辑与删除


ClassIn APP（客户端）和后台界面上，均可以编辑和删除 主课堂 ，且子课堂的相关内容会自动同步更新。（软件中修改课堂）（管理后台修改主课堂）


ClassIn App（客户端）上，无法直接编辑或删除 子课堂 ，用户只能在后台进行相关操作。用户可以进入子课堂对应的班级进行常规编辑和删除（修改或删除子课堂），还可以通过点击主课堂最右侧的“操作”按钮，选择“在线双师”，然后进行删除操作

## 3. 课后服务素材下载


（1）教师在主教室答题器中的截图可以作为助教老师沟通的抓手，本截图留存于后台-课堂管理-课堂数据（主课堂）-授课分析-答题器数据中。


（2）点击查看图片可查看本节课的所有答题器截图，右上角的“下载图片”可以将所有图片打包下载，方便课后服务的老师和学生/家长进行沟通。


（3）部分机构助教老师无后台权限，需要教务老师统一帮忙下载。


（4）课后服务的老师，可以通过这些答题器的截图和学生的答题数据，为学生提供个性化的辅导。


# 五、教师必看


## 1. 主讲教师设备


主讲老师只能用电脑和iPad上课（推荐优先使用电脑，功能更加完整），不支持使用手机和安卓Pad上课。


## 2. 教室重点工具


连麦


连麦是主讲老师和学生沟通的重要渠道


（1）在双师课的主教室中，主讲教师通过“打开教学工具连麦”或者”右键点击聊天消息“，可以选择某一个学生进行连麦（仅能同时连麦一个人），被连麦的学生，其音视频会同步到主班和所有的子班


（2）连麦结束后，可以鼠标悬停在视频窗口上选择“关闭连麦”或者“打开连麦列表，选择其他学生连麦”


（3）连麦视频窗口支持自由缩放和移动，清晰度和学生座位席的清晰度一致，显示逻辑和学生座位席无关，即连麦和座位席可以同时展示一个学生


（4）主讲教师的电脑和iPad支持连麦功能

聊天


聊天区是主讲老师和子教室学生沟通的重要渠道（给子教室学生授权、发奖杯等功能不支持）。


（1）子教室学生的发言，名字前会带有班级名称。


（2）主教室的聊天工具可以在上方选择查看【全部教室的聊天】或查看【某一个子教室的聊天】


（3）主教室和子教室聊天区，不支持【问题】功能。如果学生有问题，学生可以在【聊天】窗口中进行提问。

双师答题器


答题器是主讲教师收集学情的主要途径。


（1）双师教室发送答题器的操作与普通答题器相同。


（2）答题过程中教师可以看到全部班级的答题正确率以及各个选项的选择人数。


（3）答题器上方可以筛选班级，查看指定班级的答题情况。


（4）结束答题后，可以给答对的学生或参与学生分发奖杯。这是主讲教师给子教室学生分发奖杯的唯一途径。

一键重置功能


开课前，子教室的学生可能在某些情况下被授权，有权限在黑板上写写画画等等。为确保主教室和子教室内容一致，主讲教师进入教室后，讲课前可使用教室重置功能（右上角四个点-重置）：


清空所有教室的板书


关闭所有课件和工具


所有教室的助教和学生全部下台


清除所有学生的授权状态、奖杯数和上台次数

# 六、助教必看


## 1. 助教基础职能


课中：


1. 在后台的监课管理页面，检查未出勤的学生，联系学生出勤


2. 维持课堂秩序，教室聊天区有不适宜的内容时，在子教室中及时制止或者删除


3. 通过子教室视频墙，实时查看学生的学习状态，对于学生进行提醒


4. 解决学生的软件使用问题


5. 主讲老师授课之外，对各自班进行补充指导


禁止事项：


1. 主教室里的联席教师不要对课件和板书随便操作，操作的结果不会同步到子教室（后续版本会改进）


2. 子教室里的联席教师，不要和主讲教师同时操作同一个功能（板书、课件、教学工具），否则可能导致主教室和各个子教室不同步


课后：


1. 批阅作业、测验等课前课后活动


2. 家校沟通，利用教师上课时的答题器截图等等，进行完善的个性化学情分析和辅导。


## 2. 聊天区的使用


排课时，教务老师将助教（联席教师）同时排到主课和他所负责的所有子课里。


联席教师默认在主教室里，观看聊天区学生的留言信息，如果发现异常（例如有学生不文明发言或者提问），联席教师根据发言者的班级信息，进入子教室，打开聊天区进行处理，可以删除留言、禁言或者直接回复。


目前不支持助教在主教室直接处理异常，未来会支持。


子教室的学生，只能看到本班的聊天内容和主讲教师在主教室发布的聊天信息。


## 3. 课后沟通素材


除作业、测验等常规沟通内容外，主讲教师在主教室答题器中的截图也可以作为助教老师沟通的抓手，本截图留存于后台-课程管理-课堂数据-答题器数据中。


点击查看图片可将本节课的所有答题器截图打包下载。


无后台权限的助教老师可以请教务老师统一帮忙下载。
