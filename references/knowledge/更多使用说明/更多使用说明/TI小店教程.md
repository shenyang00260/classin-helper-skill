# TI小店教程


# 一、开通TI小店服务


商品管理和TI小店目前为增值服务，仅支持企业客户，如需开通，请联系您的专属客户经理。


# 二、后台管理添加商品/上下架商品


教务老师在后台，可以通过课堂的商品管理添加商品。


在授课时，所有的教师、助教和后台教务人员，在同一时刻只能上架一个商品，如果有人上架新的商品，原上架的商品会被自动的直播下架


后台的直播上架和回放上架的设置，可以随时开启或者关闭，但直播上架设置仅在直播期间生效，回放上架仅在课后回放期间生效


![](https://cofile.eeo.cn//newbee/assets/emoji-15/OpenFileFolder.png)


从商品库中选择现有商品，每堂课最多支持同时选择6个商品

# 三、教室中上架/下架商品


直播期间，教师或助教可在 ClassIn 直播教室的 “教学工具 - TI 小店”，或教务老师在后台通过 “课堂商品管理设置”，将商品设为 “直播上架” 状态，学生观看直播时可直接购买。

学生在直播或者回放时，可以购买处于“直播上架”或者“回放上架”的商品
