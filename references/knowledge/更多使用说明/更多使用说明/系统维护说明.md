# 系统维护说明


## 系统维护说明


尊敬的客户，为了给您提供更好的产品和服务，系统定期会进行例行维护。English version is at the bottom.


系统维护期间，部分业务可能无法使用。请您在排课的时候，确保课节时间（含课前20分钟和课后20分钟），与以下的维护时间无重叠：

如果您的机构开通了API服务，在系统维护期间的所有API请求，都会返回 901 的状态码。


在某些特殊情况下，可能会在其他时间进行系统维护。如果您发现在非上述例行维护时间内系统不可用，请您联系客服了解详情。


给您造成的不便，敬请谅解。

## System Maintenance Description


Dear customer, in order to provide better products and services, the system will be regularly maintained.


During the maintenance, some service would be unavailable. Please make sure the time of the lessons (include 20 minutes before and 20 minutes after the lesson), do not overlap with the following system regular maintenance time：


![](https://cofile.eeo.cn/res-store%2F39dcd198696de92b6c6d62d62d43c61ab78aca32e651808526afa49cf2537d96_71472?q-sign-algorithm=sha1&q-ak=AKIDJQGsEOK2TfbFdEZifMnxrWx85mpdHj6H&q-sign-time=1776009600;1776614400&q-key-time=1776009600;1776614400&q-header-list=host&q-url-param-list=&q-signature=0a679e5ebbd8e4a52686d07e87d67e2dfbf2ef86)


If your institution open the API service, all the API requests will receive status code 901 during the maintenance period.


In some special conditions, the system maintenance would happen in other time. If the system is unavailable out of above regular time, please contact the customer service for more details.


We apologize for any inconvenience caused and appreciate your understanding.
