# 班级学生管理（后台）


# 一、添加班级学生


### 适用人员


“管理后台”主账号


已开通“班级管理”权限的子账号


### 使用说明


在“管理后台”中，教务老师可以在班级下添加任何学生，不管这个学生是否已经注册了ClassIn。如果学生已经注册了ClassIn并且已经被添加到“管理后台”的学生列表中，教务老师只需要搜索学生的名字或手机号即可轻松找到并将学生添加到班级中。如果学生没有被添加到“管理后台”的学生列表中，或者学生还没有注册ClassIn账号，则教务老师可以通过代注册的方式将学生添加到班级中并帮学生完成ClassIn账号的代注册。

# 二、复制其他班级学生


## 适用人员


## 适用场景


在“管理后台”中，当某个班级中的课堂已达到上限时，教师若需继续新增课堂，则必须创建一个新的班级。为避免在新创建的班级中再次手动添加所有学生，系统允许教师将原班级中的所有学生直接复制到新班级，从而节省时间和精力，提高工作效率。


## 注意事项


如果您的班级中已经有学生，复制其他班级的学生名单时，将会替换掉当前班级中的所有原有学生。因此，请在操作时务必谨慎。
