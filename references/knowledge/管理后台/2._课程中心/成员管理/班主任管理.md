# 班主任管理


# 适用人员


“管理后台”主账号


已开通“课程管理”权限的子账号


# 适用场景


当班级需要更换班主任时，教务老师可以在后台管理系统中找到相应的课程，然后将现任班主任替换成其他人员。


# 相关文章


《管理后台添加子账号》


# 具体操作步骤


### （1）修改班主任


登入管理后台


左侧导航进入 班级管理


找到班级后，点击 班级名称，进入到班级中


进入 成员


点击班主任右侧 ···


选择 更换班主任

### （2）批量修改班主任


进入 班级管理


勾选多个班级


点击 批量操作


选择 批量设置班主任
