# 标签管理


# 适用人员


“管理后台”主账号


已开通“课程管理”权限的子账号


# 适用场景


在“管理后台”中，标签的主要作用是为了方便教务老师更加便捷地管理课程、课节和教师学生成员。通过使用标签，教务老师可以更快捷地查找课程、课节或成员，根据标签对其进行分类和定位。这将使教务老师在管理工作中更加高效和方便。


# 使用说明


管理后台中，标签总数量上限2000个，每个课程与课节，每位教师与学生的个人标签数量上限10个。

# 具体操作步骤


### （1）添加/修改课程标签信息


登入管理后台


左侧导航进入 课程管理


点击右侧 创建课程


下方点击 选择课程标签


新建标签或选择已有标签


点击 确定 保存
