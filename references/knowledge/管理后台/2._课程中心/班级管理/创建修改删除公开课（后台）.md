# 创建/修改/删除公开课（后台）


# 一、创建公开课


### 适用人员


“管理后台”主账号


已开通“公开课”权限的子账号


### 适用场景


如果老师想要创建一个没有班级的课堂时。比如班会、家长会、名师讲座、教师研讨会等，那么可以考虑创建一个公开课。这样可以更好地满足多种场景下的教学需求。


### 使用说明


当用户在“管理后台”中创建了公开课后，软件中也会相应地生成一个与之对应的公开课；同样的如果用户在软件中创建了公开课，在“管理后台”中也会生成一个对应的公开课课程。这两个公开课没有任何区别。


### 具体操作步骤


登入管理后台


左侧导航进入 公开课


点击 创建公开课


设置参数

# 二、修改公开课


已开通“班级管理”权限的子账号


查找公开课


鼠标悬停在需要修改的参数上，当右侧出现笔状图标时，点击该图标修改此位置参数
