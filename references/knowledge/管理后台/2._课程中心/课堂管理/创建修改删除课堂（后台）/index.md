# 创建/修改/删除课堂（后台）


# 一、创建课堂


### 适用人员


“管理后台”主账号


已开通“班级管理”权限的子账号


### 使用说明


添加课堂的时候，都有哪些参数可以设置，请参考下方表格


| 设置 | 说明 |
| --- | --- |
| 课堂名称 | 最多50个字符 |
| 开课日期 | 可以创建当天的课堂或其他日期的课堂 |
| 开课时间 | 最近开课时间可设置5分钟之后开课 |
| 课堂时长 | 最少课堂时长可位置为15分钟，最长课堂时长可设置为23小时55分钟 |
| 授课教师 | 添加授课教师（主讲教师），每堂课只能添加一位 |
| 联席教师 | 每课堂最多可添加6位联席教师，也可以一位都不添加 |
| 教学模式 | “在线教室”更合适纯线上教学，“智慧教室”适合线上下线下混合式教学 |
| 自动上台 | 开启“自动上台”模式，学生进入教室后，无论学生的摄像头是否打开，学生的摄像头框都会显示在台上，关闭“自动上台”模式，学生进入教室后则在台下听课，学生的摄像头不会自动出现在台上 |
| 上台人数 | 最低1v0，最高1v12。1v0代表台上仅显示教师的摄像头，无法展示学生的摄像头。1v12代表台上除了教师摄像头，还可以同时展示12名学生的摄像头。 |
| 课堂云盘资源 | 仅在本堂课教室中可以打开的公共课件资源 |
| 云端录课 | 支持同时开启“录制教室”和“录制现场”，也可单一选择一种录课模式，或开启任何录课模式 |
| 直播/回放 | 支持同时开启“网页直播”和“网页回放”权限。开启“网页直播”权限后，用户可使用教师分享的“网页直播/回放链接”观看教室中的实时上课转播。开启“网页回放”功能后，用户可使用教师分享的“网页直播/回放链接”观看上课时的录课回放视频 |

### 注意事项


创建课堂的时候，如果设置了“课堂授权资源”，那么老师在上这堂课的时候，将无法使用“班级云盘资源”中授权的课件，只能使用“课堂云盘资源”中授权的课件和个人云盘中自己上传的课件资源。

# 二、修改课堂


## 适用人员


## 使用说明


修改“课堂”的相关信息和设置时，请注意以下时间限制：


（1）课堂名称、开课日期、开课时间、课堂时长：


修改时间：可在正式开课前20分钟之外进行修改。


限制：正式开课前20分钟之内、正式开课后以及课堂结束后均无法修改。


（2）授课教师和联席教师：


修改时间：可在开课前或开课后进行添加或修改。


限制：课堂结束后无法再进行修改。


（3）上台人数（例如：将 1v1 改为 1v6 ）：


修改时间：可在正式开课前的20分钟之外进行修改。


（4）云端录课功能和网页直播功能：


（5）网页回放功能：


修改时间：随时可更改，无论是开课前、上课中还是课堂结束后均可修改网页回放的观看权限。


（6）删除课节：


修改时间：可在正式开课前的任意时段删除课节。


限制：一旦课堂正式开始后将无法删除课节。

# 三、删除课堂


已经开始上课的课堂和已经结束的课堂无法被删除，只能删除未开始的课堂。


#### （1）删除单个课堂


找到需要删除的课堂，点击右侧 操作 ，选择 删除课堂
