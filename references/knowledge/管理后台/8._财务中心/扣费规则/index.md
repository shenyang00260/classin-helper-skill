# 扣费规则


### 什么是扣费规则？


扣费规则是指在学校或企业在使用ClassIn过程中，当使用到特定的功能时需要扣除一定的费用的规则。这些规则由ClassIn指定标准，根据服务的类型和使用方式的不同，扣费规则也有所不同。


### 适用人员


“后台管理系统”主账号


拥有“财务中心”权限子账号


### 使用说明


ClassIn的收费是基于预充值机制的。学校或企业事先在他们的ClassIn后台管理系统中充值，然后用户在使用不同的ClassIn功能时，系统会从学校或企业的ClassIn账户中扣除相应的费用。需要注意的是所有费用都从学校或企业的ClassIn账户余额中扣除。ClassIn不会向任何个体教师或学生收取费用。


| 项目名称 | 扣费规则 |
| --- | --- |
| 课堂(课节) | 一堂课的扣费总额=学生扣费+联席教师扣费（没有可忽略）+录课费用（没有可忽略） |
| 学生 | 学生功能的扣费计算公式：学生扣费=台上人数每小时单价×有效出勤学生和旁听生人数×课节时长

（1）台上人数每小时单价：台上人数每小时单价根据创建课节时设置的台上人数来计算，具体单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）有效出勤学生和旁听生人数：课堂学生和旁听生必须在教室中累计听课10分钟以上才会被认定为有效出勤

（3）课节时长：根据创建课节时设置的课节时长来计算，如果课节被提前结束，计费时按实际上课时间计算。 |
| 联席教师 | 联席教师扣费计算公式：联席教师实际扣费=联席教师单价×课节时长

（1）联席教师单价：联席教师的具体单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看。不同清晰度的“台上人数”课节，教室中的联席教师单价也有所不同。（联席教师在教室中的累计上课时间满10分钟就会开始扣费，不足10分钟则不会扣费）

（2）课节时长：根据创建课节时设置的课节时长来计算，如果课节被提前结束，计费时按实际上课时间计算。（需要注意的是：计费最小单位是半小时，不足半小时的按半小时计算） |
| 录课 | 录课扣费计算公式：录课实际扣费=录课每小时单价×录课时长

（1）录课每小时单价：录课每小时单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）录课时长：按实际生成的回放视频时长为准（最小单位为分钟，不足一分钟按一分钟计算）。“回放视频时长”小于“课节消费的计费时长”时，按回放视频时长计算。“回放视频时长”大于“课节消费的计费时长”时按课节消费的计费时长计算。 |
| 网页直播 | 网页直播流量扣费计算公式：网页直播流量实际扣费 = 流量单价 × 直播流量

（1）流量单价：流量每GB单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）网页直播流量：每课节100MB以内直播流量免费。 |
| 网页回放 | 网页回放流量扣费计算公式：网页回放流量实际扣费 = 流量单价 × 回放流量

（1）流量单价：流量每GB单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）网页回放流量：用户观看网页回放时后台所产生的流量，和下载课节视频录像时产生后台所产生的流量

（3）需要注意的是：每月1号对上月的回放流量进行扣费 |
| 存储费用 | 存储扣费计算公式：存储实际扣费=（课节资源+课程附件+班级文件+收费组织云盘+教学方案-存储上限）×存储单价

（1）每月1号对存储进行扣费。实际存储量在机构存储上限内不计费，超过的按存储单价计费。实际收费项以合同约定为准。

（2）课节资源包含收费的录课视频、收费的监课图片以及板书课件。其中系统生成的录课视频和监课图片前3个月免费，超过3个月继续存储则需收取存储费用，在3个月内删除则后续不再扣费。上传的课节视频不享受前3个月免费的优惠政策，当月月末会统计存储费用。

（3）监课图片存储按200KB/张换算，1GB（约5242张）为1个计费单位，不足1GB的部分不计费。

（4）课程附件不享受前3个月免费的优惠政策，当月月末会统计存储费用。

（5）班级文件不享受前3个月免费的优惠政策，当月月末会统计存储费用。

（6）组织云盘不享受前3个月免费的优惠政策，当月月末会统计存储费用。

（7）教学方案不享受前3个月免费的优惠政策，当月月末会统计存储费用。 |
| 短信通知 | 短信扣费计算公式：短信实际扣费 = 中国大陆短信单价 × 实际发送中国大陆短信条数 + 中国港澳台及国际短信单价 × 实际发送中国港澳台及国际短信条数

（1）中国大陆短信单价：中国大陆短信单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）中国港澳台及国际短信单价:中国港澳台及国际短信单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（3）实际发送短信条数：70字内（含70）按一条发送，超过70字按67字/条发送(不区分中英文，每个汉字、字母、数字、空格、符号等均计为1个字) |
| 语音电话 | 语音电话扣费计算公式：语音电话实际扣费 = 中国大陆语音电话单价 × 中国大陆实际通话分钟数

（1）中国大陆语音电话单价：中国大陆语音电话单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）中国大陆实际通话分钟数：单次通话时长不足1分钟的按1分钟计算 |
| 智能班主任 | （1）“智能班主任”功能年费可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）开启“智能班主任”功能后，系统会在您的账户中立即扣除一年的功能开通费用。

（3）如果您在到期最后一天23:59:00时未将此功能关闭，系统将自动为您续费一年。如果您不想继续使用此功能，请您在最后一天的23:59:00之前关闭此功能。

（4）需要注意的是：使用“智能班主任”功能时会产生短信和语音电话的费用，这些费用会根据您的使用量独立结算。具体费用请参考“短信实际扣费”和“语音电话实际扣费”的说明。 |
| 超级客服 | 超级客服扣费计算公式：超级客服扣费 = 中国大陆语音回呼单价 × 中国大陆实际通话分钟数

（1）中国大陆语音回呼单价：中国大陆语音回呼单价可以在后台管理中的“财务总览”页面的右上方的“扣费规则”里查看

（2）中国大陆实际通话分钟数：主叫人和被叫人分别按照其实际通话时长单独扣费，单次通话时长不足1分钟的按1分钟计算。 |
