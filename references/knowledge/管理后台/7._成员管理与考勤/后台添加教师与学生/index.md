# 后台添加教师与学生


# 一、后台添加教师


## 适用人员


“后台管理系统”主账号


拥有“教师管理”权限子账号


## 使用说明


要在“后台管理系统”中添加一名教师进入系统时，需要提供以下基本信息：教师的手机号或邮箱（随便选一个填写即可），教师的姓名。除此之外还可以选择上传教师的照片、为教师打上标签、决定是否允许教师使用公共课件资源，以及填写教师的简介（不填也可以）。最后，点击“确定”按钮就可以完成添加了。


要注意的是，如果被添加的教师还没有在注册过ClassIn账号，就需要添加时勾选“代注册新教师”的选项。这样系统会为这名教师自动注册账号并发送一条随机密码的短信，这样教师就可以用手机号和密码登录系统了。

# 二、后台添加学生


拥有“学生管理”权限子账号


## 适用场景


当教务老师想要为课程添加学生时，只能从当前“后台管理系统”中已有的学生中进行选择。如果要添加的学生不在当前“后台管理系统”中，教务老师需要先在“学生管理”添加该学生，再将该学生添加为当前“后台管理系统”的学生身份后，方可将其添加到该课程中。


要在“后台管理系统”中添加一名学生时，需要提供以下基本信息：学生的手机号或邮箱（随便选一个填写即可），学生的姓名。需要注意的是，如果被添加的学生还没有在注册过ClassIn账号，就需要在添加的时候勾选“代注册新学生”的选项。这样系统会为这名学生自动注册账号并发送一条随机密码的短信，这样学生就可以用手机号或邮箱和密码登录系统了。
