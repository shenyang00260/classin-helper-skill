# 教师/学生停用、隐藏及恢复


# 适用人员


“后台管理系统”主账号


拥有“教师管理”或“学生管理”权限子账号


# 适用场景


停用教师：当教师成员离职后，教务老师可以在“后台管理系统”中的“教师管理”栏目下停用该名教师，成功停用该名教师后，该名教师将不会出现在排课时的可选教师列表中。


恢复已停用的教师：如果教务老师想让被停用的教师再次在排课时可被选择，可以登入“后台管理系统”，然后进入“教师管理”栏目。在“教师管理”栏目下找到被停用的教师后，取消教师的停用状态，这样该教师就会重新出现在排课时的可选教师列表中了。这个情况一般多用于教师重新回岗后将教师重新纳入到可安排授课的人选中。


隐藏学生：教务老师可以在“后台管理系统”中的“学生管理”栏目下隐藏某位学生，成功隐藏该名学生后，该名学生将不会出现在排课时的可选学生列表中。


恢复已隐藏的学生：如果教务老师想让被隐藏的学生再次在排课时显示，可以登入“后台管理系统”，然后进入“学生管理”栏目。在“学生管理”栏目下找到被隐藏的学生后，取消学生的隐藏状态，这样该学生就会重新出现在排课时的可选学生列表中了。


# 注意事项


如果被停用的教师拥有“后台管理系统”中的子账号身份，那么将无法停用该名教师，需要先在“权限管理”中将该名教师的子账号身份删除后才可以对其进行停用操作。


如果被停用的教师名下还有未上完的课或正在进行中的课，那么将无法停用该名教师。

# 具体操作步骤


### （1）后台停用教师


登入后台管理系统


左侧导航进入 教师管理


查找教师


点击右侧 停用
