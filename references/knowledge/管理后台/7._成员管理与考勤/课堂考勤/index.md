# 课堂考勤


### 适用人员


“后台管理系统”主账号


拥有“财务中心”权限子账号


### 适用场景


当教务老师想要查看某一个时间端内的课堂考勤明细时，可以从后台管理系统中的“统计下载”模块里将课堂考勤明细以表格形式下载电脑本地中。从表格中能获取到的信息有：考勤时间范围、班级ID、班级名称、课堂ID、课堂名称、开课时间、结束时间、时长（分钟）、台上人数、课堂学生数、授课教师、授课教师昵称、授课教师手机号、授课教师邮箱、课节标签、身份、姓名、昵称、手机号、邮箱、出勤、实际上课时长（分钟）、迟到、早退、第一次进入教室、最后一次离开教室、班级标签、教师标签、学生标签、班主任姓名、班主任手机号、班主任邮箱。


### 使用说明


每次最多可下载6个月。


可同时下载多个任务


申请下载的任务会显示在《任务中心》。用户可以前往任务中心完成下载操作。

### 具体操作步骤


登入后台管理系统


左侧导航进入 统计下载


筛选日期


点击 课堂考勤 右侧 下载 ，等待下载任务生成后，在新的弹窗中点击 去查看
