# 接收教室求助信息并回复


# 一、接收教室求助信息


## 适用人员


“管理后台”主账号


已开通“监课管理”权限的子账号


## 适用场景


当教室里的老师或学生需要帮助时，他们可以使用教室中的求助功能向“管理后台”发送求助信息。教务员可以登录“管理后台”，在“监课管理”页面中找到相应的课节，并点击课节下方的问题信息，查看教室中提出的问题。教务可以回复这些问题，给予帮助。


## 相关文章


《管理后台添加子账号》


## 具体操作步骤


登入管理后台


左侧导航进入 监课管理


找到课节


在课节左下方，查看 求助消息

# 二、向教室发送信息


“后台管理系统”主账号


当教务老师在教室里进行监课时，由于其身份是旁听者，因此无法与教室中的任何成员进行互动。如果发现教室中某个用户出现了异常问题，教务老师可以从后台管理系统中给教室内的某个成员发送文字消息以告知该用户如何解决问题。


点击课节下方 联系


选择 联系教师 或 联系学生


勾选联系对象，在 描述问题 中输入内容，点击 发送
