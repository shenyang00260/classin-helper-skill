# 进入监课教室


### 适用人员


“管理后台”主账号


已开通“监课管理”权限的子账号


### 适用场景


当教务老师想要旁听学校中某位老师的授课过程时，可以从后台管理系统中的监课管理页面中进入到对应的教室进行旁听


当教室中的授课老师因网络不稳定或设备问题导致的云端录课中断无法再正常继续录课，并且这堂课的教室中没有联席教师的情况下，教务老师可以从后台管理系统中的监课管理页面中进入到对应的教室进行本地录课


### 使用说明


当教务老师需要进行监课时，需要在“管理后台”中的“监课管理”页面点击“进入教室”按钮。此时网页会提示是否唤起相关应用，需要教务老师点击“允许”或“打开ClassIn”等类似的按钮才能成功进入监课教室。如果按照操作流程未能成功唤起ClassIn软件，则需要关闭教务老师电脑中的杀毒软件，然后在ClassIn软件的设置中修复注册表。完成修复后重新操作一遍唤起软件即可。需要注意的是教务老师从后台管理系统中进入教室监课是免费的，不会产生费用。


当教务老师进入教室监课时，他会以一个“隐身人”的身份进入教室。此时，教室中的老师和学生无法察觉到教务老师的存在，教务老师也不能与教室中的任何人互动。然而要注意一点是如果教务老师使用的后台管理系统账号在教室中具有联席教室或学生身份，则教务老师在进入教室监课时其身份将自动转换为教室中对应的身份。此时教务老师可以与教室中的成员互动，教室中的成员也可以察觉到教务老师的存在。一旦教务老师不是以“隐身人”的身份进入的教室，则将按照正常教室中的身份扣除后台管理系统中的费用。


### 注意事项


监课管理中最多展示1000个教室。
