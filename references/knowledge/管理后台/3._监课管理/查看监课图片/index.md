# 查看监课图片


### 适用人员


“管理后台”主账号


已开通“监课管理”权限的子账号


### 使用说明


教务老师进入监课详情页面后可以看到教室中的监课图片，监课图片是ClassIn系统以授课教师的视角自动进行的截图并且每分钟都会自动保存一张截图。


### 注意事项


监课管理中最多展示1000个。


### 相关文章


《进入监课教室》


《管理后台添加子账号》


### 具体操作步骤


登入管理后台


左侧导航进入 监课管理


查找课节


点击图片进入详情中查看 监课图片
