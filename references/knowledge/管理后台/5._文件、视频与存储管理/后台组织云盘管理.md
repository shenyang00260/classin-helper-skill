# 后台组织云盘管理


# 一、创建文件夹/上传课件


## 什么是组织云盘？


“后台管理系统”和ClassIn软件中的组织云盘功能是相互同步的。用户在ClassIn中创建组织云盘并上传课件后，这些课件会自动同步到学校的后台管理系统中的组织云盘。反之，教务老师在后台管理系统上传课件并添加组织成员后，成员登录ClassIn，进入学校的组织云盘即可看到这些课件。


管理员可以在组织云盘中创建各种《文件夹》和《子文件夹》，并添加成员和设置成员权限。这样，这些文件夹就成了一个个小型的共享和协作空间，比如“英语组”或“一年级组”。这种方式不仅方便了文件的共享和管理，还提升了学校或机构在文件授权和管理方面的效率。


通过组织云盘，教师们可以更方便地共享、协作和管理教研资料，充分满足协同教学的需求。这一功能大大提升了文件管理的便捷性和教学工作的效率。

| 权限 | 说明 |
| --- | --- |
| 可管理 | 可以添加、授权、移除成员;可以对文件进行上传、删除、重命名、转发、使用等操作 |
| 可操作 | 可以上传、新建、使用文件，不能删除 |
| 仅查看 | 能预览文件和在教室使用文件，不能上传、新建、删除 |
| 不可见 | 不可查看文件 |
