# 学校机构名称与LOGO


### 适用人员


“管理后台”主账号


已开通“学校设置”权限的子账号


### 使用说明


教务老师上传的图片支持固定比例裁剪，建议上传的图片比例为260px*90px，图片大小不超过1MB。在“管理后台”中设置的学校名称与学校logo会在“网页直播/回放页面”和“教学报告”中展示。

### 具体操作步骤


登入管理后台


左侧导航进入 学校设置


上方进入 基本设置


在 品牌名称与LOGO设置 中，点击 编辑


设置 品牌名称 和 品牌logo


点击 确定
