# 学校/机构简介


### 适用人员


“管理后台”主账号


已开通“学校设置”权限的子账号


### 使用说明


简介内容支持添加学校文字和图片介绍、报名电话、报名链接等等。(图片要求宽度不超过690px，高度不限，单张大小不超过800KB；建议上传图片不要过多过大，以免影响页面加载速度）。在“管理后台”中设置的学校简介会在“网页直播/回放页面”和“公开课报名页面”以及“学习报告”中展示。
