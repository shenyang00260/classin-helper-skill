# 后台管理员


# 适用人员


“后台管理系统”主账号


拥有“权限管理”的子账号


# 使用说明


后台管理系统是一个用于管理学校课程、教师和学生等信息的系统。主账号老师可以在后台管理系统中为系统添加管理员（子账号），让管理员可以登录后台管理系统，并授予管理员不同的权限。不同的权限可以让管理员管理不同的后台区域。以下是对应权限可管理的后台区域：


| 课程中心 | 班级管理/课堂管理 | 管理员可以创建班级、修改班级、班级结课，以及为班级添加学生。还可以创建、修改、删除课堂。 |
| --- | --- | --- |
| 公开课 | 管理员可以创建公开课、修改公开课、删除公开课、以及为公开课添加学生。 |
| 教师管理 | 管理员可以添加、修改、停用、启用教师，并为教师设置授权资源及标签。 |
| 学生管理 | 管理员可以添加、修改、隐藏、显示、删除学生，并为学生设置授权资源及标签。 |
| 存储管理 | 数据概览 | 管理员可以查看后台管理系统的存储情况。 |
| 组织云盘 | 管理员可以创建文件夹、上传课件以及复制、移动课件。其中还包括删除、下载和成员管理三个分支选项，勾选这些分支选项后，可以管理对应的功能。 |
| 题库中心 | 管理员可以添加，修改，删除组织题库的试题试卷。 |
| 课堂资源 | 管理员可以查看每节课的【录课视频】【监课图片】【板书课件】的存储量，也可以对其进行锁定操作，以防止其被误删除。同时，管理员也有权利手动删除【录课视频】【监课图片】【板书课件】。 |
| 班级资源 | 管理员可以查看班级的【班级文件】【课程附件】【共创文档】的存储量，也可以对其进行锁定操作，以防止其被误删除。同时，管理员也有权利手动删除【班级文件】【课程附件】【共创文档】。 |
| 资源中心 | 管理员可以在资源中心添加资源商品和管理资源订单，也可以查看主账号获取到的资源。 |
| 教学管理 | 直播回放 | 管理员可以查看修改网页直播回放的个性化设置。 |
| 教学数据 | 管理员可以修改课后评价，查看教学报告，学习报告，课堂数据中的授课分析和学习分析。 |
| 作业管理 | 管理员可以查看/下载作业数据。 |
| 测验管理 | 管理员可以查看/下载测验数据。 |
| 教务管理 | 监课管理/账号速查 | 管理员可以旁听正在上课的教室，查看教室中老师和学生的出勤情况，并查看老师或学生上课时的网络情况和设备使用情况。 |
| 财务中心 | 财务中心 | 管理员可以查看收费规则，订单详情，发票等信息。 |
| 我的学校 | 我的学校 | 管理员可以查看修改机构的个性化配置。 |
| 权限管理 | 管理员可以为学校后台添加更多管理员并支持修改或删除管理员。 |
| 客户端权限 | 创建公开课、班级 | 管理员可以在ClassIn软件中创建公开课或班级。 |

# 注意事项


修改管理员时，账号或邮箱无法修改，只能修改备注姓名和相关权限。


# 具体操作步骤


### （1）添加管理员


登入后台管理系统


左侧导航进入 权限管理


点击右侧 添加子账号


添加子账号 基础信息


设置子账号 管理权限


点击 确定
