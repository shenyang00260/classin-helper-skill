# 登录/退出《管理后台系统》


# 适用人员


“管理后台”主账号


“管理后台”子账号


# 适用场景


每所使用ClassIn的学校都有一个专门为该校定制的管理后台系统。在后台中，教务老师可以进行排课、设置、监控班级课堂，查看教学数据以及财务账单等等，这些操作都会直接影响到软件中班级、教师、学生和其他相关功能的使用。如果教务老师需要在管理后台中进行排课或设置等操作，就需要先登入到“管理后台”中才可进行设置。


# 使用说明


每个学校的“管理后台系统”中都有一个主账号，主账号拥有“管理后台”中的所有权限。如果需要多个教务老师协助使用“管理后台”，主账号可以在“管理后台”添加多个子账号，并设置每个子账号所能使用的不同权限。这样可以方便地实现多人协作。


“管理后台”中支持显示的语言有：英语、简体中文、韩语、越南语、日语、阿拉伯语。用户可以在后台管理系统首页右上角切换语言


当教师在多个学校的“管理后台”中拥有“子账号”或“主账号”身份时，教师可以先登入到“管理后台”中，然后在右上角进入到其他学校的管理后台中
