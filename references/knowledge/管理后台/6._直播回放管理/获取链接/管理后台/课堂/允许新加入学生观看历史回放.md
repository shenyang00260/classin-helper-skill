# 允许新加入学生观看历史回放


### 适用人员


“后台管理系统”主账号


拥有“学校设置”权限子账号


### 使用说明


教务老师可以在后台管理系统中打开或关闭新加入课程（班级）的学生观看课程（班级）中的历史回放视频的功能，开启该功能后，新加入该课程的学生可以观看加入该课程之前，该课程下已经上过的课程所生成的录像回放视频。


| 功能状态 | 说明 |
| --- | --- |
| 未开启时 | 新加入课程的学生，将成为历史课节的调出生，学生端不显示历史课节，不能观看历史课节的回放视频。 |
| 开启后 | 新加入课程的学生，将成为历史课节的正式学生（缺勤状态），学生端显示历史课节，且可以观看回放视频。 |


功能状态


说明


新加入课程的学生，将成为历史课节的调出生，学生端不显示历史课节，不能观看历史课节的回放视频。


未开启时


新加入课程的学生，将成为历史课节的正式学生（缺勤状态），学生端显示历史课节，且可以观看回放视频。


开启后


### 注意事项


该操作产生的学生在历史课节下的身份变化不可逆，请谨慎操作。若对使用此设置有疑问，请先咨询客户经理或客服后再选择使用。


该设置不支持课程的旁听生转为正式学生的场景，课程的旁听生将永远不能观看历史课节的回放视频，请悉知。

### 具体操作步骤


登入后台管理系统


左侧导航进入 学校设置


上方进入 课程回放


在 课程回放观看权限高级设置 中，开启或关闭 新加入课程的学生，允许观看历史课节的回放视频
