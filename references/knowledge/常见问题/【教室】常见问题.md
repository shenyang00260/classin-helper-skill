# 【教室】常见问题


# 1. 点击“上课”按钮没反应，进不去教室


现象描述：用户在ClassIn中的消息列表里找到教室并点击“上课”按钮后，没有任何反应，无法正常进入教室。


解决方案：


电脑用户，请尝试下方任意一种方法以解决此问题：


重新登录ClassIn：关闭ClassIn程序，然后重新启动并登录。


关闭杀毒软件和防护程序：退出电脑中的杀毒软件和防护类程序，例如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。


关闭防火墙：在电脑设置中关闭防火墙。


手机/平板用户，请尝试下方任意一种方法以解决此问题：


重新启动设备：将手机或平板关机，然后重新开机，重新登录ClassIn。


检查设备配置：性能较低的手机或平板可能无法正常进入，请确认所使用的设备是否满足ClassIn的配置要求。


# 2. 上课过程很卡


现象描述：在使用ClassIn进行上课时，用户可能会遇到卡顿现象。例如，教室中的画面播放不流畅，一帧一帧地显示，或者画面突然卡住不动，需要等待一段时间才能恢复。另外，课堂中其他人的声音可能会出现断断续续、不连贯的情况。


关闭其他程序：仅运行ClassIn，关闭除ClassIn外的其他已打开的软件。


退出防护软件：退出电脑中的杀毒软件和防护类程序，例如：360安全卫士、360杀毒、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。


重启电脑：重启电脑。


连接电源：使用笔记本电脑的用户，请确保连接充电线（电量不足时，电脑会自动降低功耗，可能导致软件卡顿）。


使用散热设备：使用笔记本电脑的用户，请增加散热设备。


重启设备：重启手机或平板。


关闭其他程序：关闭手机或平板后台的所有程序，仅运行ClassIn。


更换设备：如果上述两步操作后问题仍存在，可能是设备性能不足，建议临时使用其他设备上课（例如其他手机、平板或电脑）。


# 3. 进入教室后，一直提示：正在同步教室信息


现象描述：在进入教室后，系统持续显示“正在同步教室信息”的提示，导致用户无法正常进入教室。


重新登录ClassIn：关闭教室和ClassIn应用程序，然后重新登录。


关闭安全软件：退出电脑中的杀毒软件和防护类程序，例如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。


关闭防火墙：暂时关闭电脑中的防火墙。


重启路由器：断开路由器电源，等待几秒钟后重新连接电源并启动路由器。


切换网络：如果使用笔记本电脑，可以尝试连接有线网络或使用手机分享的5G热点。


优化教室黑板内容：如果教室中黑板内容较多，可以请教师删除部分板书或提前保存并清空所有板书，然后再让学生重新进入教室。


靠近路由器：将手机或平板设备靠近路由器以确保信号强度。


切换网络：临时连接其他网络。例如，如果设备已连接Wi-Fi，可以断开Wi-Fi并连接手机自身的移动网络。


重启设备：关机重启手机或平板。

# 4. 上课过程中，频繁提示：正在同步教室信息


现象描述：在使用ClassIn上课时，您可能会不时收到提示信息：“正在同步教室信息”。


关闭杀毒软件和安全防护程序：退出或关闭电脑中的杀毒软件和防护类程序，如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。


关闭防火墙：关闭电脑中的防火墙，以确保网络通信顺畅。


重启路由器：将路由器断电，然后重新接通电源，以重启路由器。


优化网络连接：如果使用笔记本电脑，可以尝试连接有线网络或手机共享的5G热点，以获得更稳定的网络连接。


靠近路由器：将手机或平板设备靠近路由器，以增强网络信号强度。


切换网络：临时更换网络连接。例如，如果您使用手机上课且已连接WiFi，可以断开WiFi并连接手机的移动数据网络。

# 5. 上课时，教室画面卡住不动了


现象描述：用户使用电脑在ClassIn中上课时，教室画面突然卡住彻底不动了。


解决方案：尝试下方任意一种方法以解决此问题


仅运行ClassIn：关闭除ClassIn以外的所有其他软件，以确保系统资源集中用于ClassIn。


关闭杀毒软件和防护程序：退出电脑中的所有杀毒软件和防护类程序，如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家和联想电脑管家等。


关闭防火墙：暂时关闭电脑中的防火墙，以排除防火墙对网络连接的干扰。


连接充电线：对于使用笔记本电脑的用户，请确保连接充电线，因为电量不足时，电脑会自动降低功耗，从而导致软件卡顿。


增加散热设备：对于使用笔记本电脑的用户，建议增加散热设备，以防止因过热导致的性能下降。


# 6. 教室画面显示不全


现象描述：在使用电脑进入教室上课时，遇到了教室界面显示异常的问题。具体表现为教室画面不完整，工具栏无法显示，有时左侧画面缺失，有时右侧画面缺失。


解决方案：请将电脑的分辨率调整至 1280 x 720 或更高，以确保界面的正常显示。

# 7. 平板无法全屏


现象描述：使用平板电脑进入ClassIn教室后，教室画面无法全屏显示，ClassIn画面居中停留在平板屏幕中间位置。


解决方案：按照下方步骤进行操作


进入平板电脑的“设置”。


在“设置”中找到并点击“应用程序”。


在应用程序列表中找到“ClassIn”并点击进入。


检查是否有“横屏模式”选项。如果有，请选择“自适应”。


如果没有“横屏模式”选项，请下拉平板电脑顶部的通知栏，并启用“自动旋转”功能。

# 8. 使用手机上课，每堂课会消耗多少流量？


问：使用手机的移动流量上课时，每堂课大约会消耗多少流量？


答：每堂课的流量消耗范围在100-200MB之间，若视频质量较高或有多位参与者开启高清视频，流量消耗可能会上升到数百甚至上千MB。这些流量消耗会从使用移动数据上课的教师或学生的手机套餐中扣除。


# 9. “联席教师”可否提前开课？


问：授课教师尚未进入教室时，联席教师是否有权限提前进入教室并开始授课？


答：可以，联席教师有权限提前进入教室并开始授课。然而，联席教师无法进行云端录课。如果课程启用了云端录课和直播功能，必须等到主讲教师进入教室并启动录课后，使用直播链接的观众才能通过链接观看课程直播。


# 10. 不下载ClassIn，可否在网站上授课？


问：老师是否可以不下载ClassIn，直接在网站上进行授课？


答：老师必须下载并安装ClassIn客户端才能进入教室进行授课。


# 11. 教室中打开工具后，工具瞬间消失


现象描述：在教室中打开某些工具后，工具窗口会瞬间消失，特别是那些带有提示弹窗的工具。


解决方案：这种现象通常是由于360安全卫士的弹窗拦截功能所致。建议暂时退出360安全卫士以解决此问题。


# 12. 一号多用


问：一个账号可以提供给多名学生上课使用吗？


答：一个账号是否可以供多名学生使用取决于上课时间安排。如果多名学生的上课时间重叠，那么一个账号无法同时满足多个教室的需求，因为一个账号只能进入一个教室上课。然而，如果多名学生的上课时间不重叠，理论上可以共用一个账号。


尽管如此，仍然不推荐多个学生共用一个账号。这样做可能会导致教学数据和学生信息（如名字等）的混淆，从而影响教学质量和学生的个性化学习体验。
