# 【课件】常见问题


# 1. PPT自动翻页（自动放映）


现象描述：在教室中打开PPT课件后，PPT课件会自动翻页或自动播放动画效果，而并非人为操作。


解决方案：按照下方步骤进行操作


在本地电脑上打开PPT文件。


取消勾选“自动换片”功能。


点击“全部应用”以确保设置生效。


保存修改后的PPT文件。


将更新后的PPT文件重新上传至ClassIn云盘。

# 2. PPT课件中文字错位/重叠


现象描述：用户在教室中打开PPT课件后，发现文字出现错位或重叠的情况。


解决方案：


中文字体选择：建议在PPT课件中使用系统自带的中文字体，如“宋体”、“黑体”或“微软雅黑”，以确保兼容性和显示效果。


英文字体选择：建议使用“Arial”字体，以避免显示问题。同时，尽量避免使用“斜体”、“下划线”、“加粗”等字体效果，以减少潜在的文字错位或重叠现象。


文件保存与上传：在本地修改完PPT课件后，保存文件并重新上传到云盘，以确保在不同设备上打开时能够正常显示。

# 3. 打不开txt文件


现象描述：用户在教室中无法打开txt格式的文件。


解决方案：如果txt文件的大小超过1MB，将无法在教室中正常打开。建议用户将txt文件中的内容复制到Word文档中，然后重新上传到ClassIn的云盘中。这样可以更有效地解决文件过大导致的无法打开问题。


# 4. 打开课件后，课件中不显示内容


现象描述：用户在教室中打开课件后，课件内容未显示，课件呈现全白或全黑。


解决方案：尝试下方任意一种方法以解决此问题


重新打开：关闭课件并重新打开。


清理缓存：清理ClassIn缓存。


重新登录：退出教室并关闭ClassIn应用，然后重新登录ClassIn。


# 5. PPT无法全屏显示


现象描述：用户在教室中的云盘工具里打开PPT课件，并将其放大全屏显示后，发现PPT课件的画面两侧仍然存在黑边，无法完全填满屏幕，未实现真正的全屏效果。


解决方案：建议在本地电脑上调整PPT文件的宽度和高度，以适应教室屏幕的比例。具体调整如下：


常规屏幕模式：


宽度：31.75厘米


高度：13.851厘米


或


宽度：63.5厘米


高度：27.7厘米


大屏模式或隐藏座位席时：


高度：33.4厘米


调整完成后，重新上传PPT到ClassIn云盘中进行使用，即可实现真正的全屏显示效果。

# 6. 报错提示：页面图片过大无法正常显示


现象描述：用户在教室中使用iPad或iPhone打开云盘中的PDF课件时，遇到提示“页面图片过大无法正常显示”。


解决方案：该问题是由于PDF文件中的图片元素过大，导致iPad或iPhone设备内存超载。尝试下方任意一种方法以解决此问题


减少图片数量：PDF课件编辑者可在电脑上减少单页中的图片数量，然后将修改后的PDF重新上传到教室云盘中使用。


缩小图片尺寸：PDF课件编辑者可在电脑上缩小PDF中的图片尺寸，然后将缩小后的PDF重新上传到教室云盘中使用。

# 7. 超过500MB的课件如何上传


问：超过500MB的课件该如何上传？


答：可以将课件拆分后，以多个课件的形式进行上传。
