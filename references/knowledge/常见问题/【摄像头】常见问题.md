# 【摄像头】常见问题


# 1. 摄像头画面显示“白色斜杠相机”


现象描述：用户在教室中打开摄像头后，摄像头窗口中不显示自身画面，仅显示一个“白色斜杠相机”图标。


解决方案：尝试下方任意一种方法以解决此问题。


如果用户的电脑中安装了“电脑管家”软件，可能会导致摄像头无法正常调用。请尝试临时关闭电脑管家，以排除其干扰。


用户的笔记本电脑可能禁用了本地摄像头，导致摄像头在教室中无法正常使用。可以按以下步骤重新启用摄像头：


对于联想牌笔记本：前置摄像头旁边有一个滑动开关，将其打开即可重新启用摄像头。


对于其他品牌的笔记本：同时按下键盘上的FN键和带有相机图标的功能键，即可重新启用摄像头。


如果以上操作均无效，请联系您的电脑售后服务，安装最新版的摄像头驱动程序。

# 2. 摄像头画面显示“ClassIn头像”


现象描述：用户在教室中打开摄像头后，摄像头窗口中不显示自己的画面，而仅显示一个圆形的ClassIn个性头像。


解决方案：出现这种情况是因为用户禁用了自己的摄像头。请用户在教室中启用摄像头，以恢复正常显示。

# 3. 摄像头画面显示“绿色摄像头标“


现象描述：用户在教室中打开摄像头后，摄像头窗口中不显示自己的画面，仅显示一个绿色的摄像头标志。


解决方案：


对于电脑用户，请尝试下方任意一种方法以解决此问题：


关闭杀毒软件和防护类程序：请退出电脑中的杀毒软件和防护类程序，例如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家。


重启路由器：断开路由器电源，等待数秒后重新连接电源，重启路由器。


切换网络连接：如果您使用的是笔记本电脑，请尝试连接有线网络或使用手机热点（建议选择5G热点）。


对于手机和平板用户，请尝试下方任意一种方法以解决此问题：


靠近路由器：将手机或平板设备靠近路由器，以增强网络信号。


切换网络：临时使用其他网络连接。例如，断开当前的WiFi网络，改为使用移动数据网络。


关闭视频硬件解码器：如果您使用的是安卓系统设备，请在ClassIn设置中关闭“开启视频硬件解码器”功能。

# 4. 摄像头画面全黑


现象描述：用户在教室中打开摄像头后，摄像头窗口不显示自己的画面，窗口呈现全黑状态。


检查遮挡物：检查摄像头前方是否有遮挡物。


安装最新驱动：联系电脑售后服务，安装最新版摄像头驱动程序。


联系小助手：如以上操作无效，请在售后服务群中联系ClassIn小助手，或拨打客服电话：400-077-1585 获取进一步帮助。

# 5. 摄像头画面显示：摄像头可能被占用，请关闭其他软件


现象描述：用户在教室中打开摄像头后，摄像头窗口中不显示自己的画面，仅显示一段提示信息：“摄像头可能被占用，请关闭其他软件。”


关闭其他软件：请确保电脑上除ClassIn之外，没有其他正在调用摄像头的软件。


退出防护程序：退出电脑中的所有杀毒软件和防护类程序，例如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家。


启用相机权限（Windows 10）：如果您使用的是Windows 10系统，请开启“允许应用访问你的相机”功能。（路径：点击桌面左下角菜单 → 进入左下角齿轮图标 → 选择“隐私” → 点击“相机” → 开启“允许应用访问你的相机”。）


授权摄像头权限（Mac）：如果您使用的是Mac电脑，请授权摄像头权限，并重启电脑。已授权的用户可以尝试重新授权后重启电脑。


调整ClassIn设置：在ClassIn设置中，将摄像头模式切换为“默认模式”或“高级模式”。


修复摄像头（Windows）：如果您使用的是Windows系统的电脑，可以尝试在ClassIn教室中使用修复摄像头功能。

# 6. 摄像头画面昏暗


现象描述：用户在教室中打开摄像头后，画面显示较为昏暗。


调整环境光线：确认教室环境光线较暗时，建议打开房间内的照明设备，以增加光线。


调整摄像头参数：登录QQ，在设置选项中调整摄像头的亮度参数，以改善画面效果。

# 7. 打开摄像头后，自动关闭摄像头


现象描述：用户在教室中打开摄像头后，摄像头又自动关闭。


解决方案：对于使用Windows系统电脑上课的用户，可以在教室中尝试修复摄像头。如果修复后问题仍然存在，可能是由于摄像头驱动程序损坏或版本过低所致。建议联系电脑售后服务部门，安装最新版本的摄像头驱动程序。


# 8. 摄像头画面倒置


现象描述：用户在教室中打开摄像头后，发现摄像头画面是倒置的。


电脑用户：可以通过调整摄像头的角度来纠正画面倒置的现象。


手机/平板用户：可以尝试将设备上下旋转，以恢复正常的画面显示。

# 9. 摄像头画面花屏


现象描述：用户在教室中打开摄像头后，摄像头画面出现了花屏现象。


修复摄像头：如果您使用的是Windows系统，请在教室中尝试修复摄像头。


更换设备：对于外接摄像头，尝试重新插拔设备或更换USB接口连接。


更新驱动：确认摄像头无物理损坏的情况下，请联系电脑售后服务，更新并安装最新版摄像头驱动程序。（为确认摄像头是否损坏，可以在其他软件（例如：微信、QQ等）中使用视频功能检查摄像头画面。如果在这些软件中摄像头画面也出现花屏，可能表明摄像头已经损坏。）

# 10. 摄像头画面全灰


现象描述：用户在教室中打开摄像头后，摄像头画面呈现灰色。


解决方案：这种情况通常是由于用户在笔记本电脑中禁用了本地摄像头，导致其无法正常工作。以下是针对不同品牌笔记本电脑的解决方法：


联想笔记本电脑：前置摄像头旁边有一个滑动开关，将其滑动至打开位置，即可重新启用摄像头。


其他品牌笔记本电脑：同时按住键盘上的FN键和带有相机图标的按键，即可重新启用摄像头。

# 15. 摄像头画面显示不全

# 16. 摄像头画面显示“ClassIn头像+禁用标志”


现象描述：用户在教室中打开摄像头后，摄像头画面中仅显示用户自己的ClassIn头像和一个禁用图标。


解决方案：这一问题是由于用户的安卓设备性能较低，无法支持正常的上课视频功能。为了保证课堂流畅度，系统优先减少视频流的使用，因此用户只能看到自己和教师，而无法看到其他学生的画面。

# 17. 摄像头虚拟背景不可用


现象描述：用户在教室中使用摄像头虚拟背景时，提示不可用。


解决方案：按照下方步骤进行操作。


确认设备系统版本：


Mac系统需为10.15及以上。


Windows系统需为10及以上。


系统版本确认无误后，请进行驱动程序升级（具体操作方法可参见下方附录）。


如驱动升级后问题仍未解决，请联系技术支持小助手进行进一步排查。

# 19. 是否可以将学生的摄像头设置为默认开启状态


问：老师是否可以将学生的摄像头设置为默认开启状态，并且禁止学生自己关闭自己的摄像头？


答：不可以，老师无法将学生的摄像头设置为默认开启状态，并且也无法禁止学生自己关闭自己的摄像头。
