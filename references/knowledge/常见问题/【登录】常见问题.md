# 【登录】常见问题


# 1. ”登录“按钮呈灰色，无法点击


现象描述：在ClassIn登录界面输入账号和密码后，发现“登录”按钮始终呈灰色，无法点击。


解决方案：尝试下方任意一种方法以解决此问题


稍候片刻：可能是由于网络连接速度较慢，建议稍等片刻后再尝试。


关闭杀毒软件和防护类程序：请退出电脑中的防护软件（如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。）


关闭防火墙：请临时关闭电脑中的防火墙，以排除其对网络连接的影响。


重启路由器：断开路由器的电源，然后重新连接电源，以重启路由器。


尝试其他网络连接：如果您使用的是笔记本电脑，可以尝试连接“有线网络“或”手机共享的5G热点“

# 2. 一直显示“登录中”


现象描述：在ClassIn登录界面点击“登录”按钮后，程序一直显示“登录中”状态，无法完成登录流程。


稍等片刻：可能是网络连接速度较慢，稍等一会儿再尝试。


关闭杀毒软件和防护程序：退出电脑中的杀毒软件和防护类程序（如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。）


关闭防火墙：暂时关闭电脑中的防火墙，以排除防火墙设置导致的问题。


重启路由器：断电重启路由器，然后重新连接网络。


尝试其他网络连接：如果使用的是笔记本电脑，可以尝试连接”有线网络“或”手机分享的5G热点“。


更改服务器设置：在ClassIn登录界面右下角点击“更多设置”，尝试切换到AUTO选项下的前三个服务器中的任意一个重新登录。


# 3. 登录窗口无响应


现象描述：使用苹果电脑的用户在打开ClassIn登录界面后，登录窗口出现卡顿现象，无法进行任何操作，点击登录窗口中的任何按钮都没有反应，用户也无法在登录窗口中输入任何内容。


解决方案：按照下方步骤进行操作


打开电脑中的“系统偏好设置”。


在系统偏好设置中选择“辅助功能”。


在辅助功能界面中，点击左侧菜单中的“VoiceOver”。


取消勾选“启用VoiceOver”选项。


详细操作步骤请参考以下链接的具体教程：https://zhidao.baidu.com/question/1604057812342981067.html


# 4. 报错提示：网络不给力，请检查网络设置


现象描述：在打开ClassIn登录界面后，界面提示：“网络不给力，请检查网络设置”。


退出防护软件：请关闭电脑中运行的防护类程序（如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。）


关闭防火墙：暂时关闭电脑中的防火墙以排除其对网络连接的干扰。


重启路由器：将路由器断电数秒钟后，再重新接通电源并启动路由器。


更换网络连接：若您使用的是笔记本电脑，建议尝试连接”有线网络“或”手机分享的5G热点“，以确保网络稳定性。


# 5. 报错提示：密码错误


现象描述：在ClassIn登录界面输入账号和密码后，点击“登录”按钮，系统提示“密码错误”。


检查密码输入：确保ClassIn密码输入正确，并注意字母的大小写区分。


核对账号输入：确保ClassIn账号输入无误，特别注意账号前后是否有不必要的空格。


重置密码：在ClassIn登录界面中，点击“忘记密码”选项。按照提示步骤进行密码重置。


# 6. 报错提示：账号不存在


现象描述：在ClassIn登录界面输入账号和密码后，点击“登录”按钮，系统提示“账号不存在”。


核对账号输入：请确认输入的ClassIn账号是否正确，确保账号前后没有空格。


核对账号是否已注册：请检查该账号是否已经注册。如果尚未注册ClassIn账号，可以在登录窗口中点击“立即注册”进行注册。


修改区号：请确认账号左侧的“区号”选择是否正确。例如：中国大陆地区的区号为+86，其他国家和地区的区号请参考此链接：https://www.zou114.net/qh.asp


# 7. 报错提示：C盘空间不足，请您清理磁盘后重新打开ClassIn


现象描述：启动ClassIn时，系统提示：“电脑C盘空间不足，请清理磁盘后重新启动ClassIn。”


清理C盘空间：删除不必要的文件和应用程序，以释放磁盘空间。


重新安装ClassIn：将已安装的ClassIn卸载，然后在官方网站重新下载安装包，并将ClassIn安装至非C盘的其他磁盘中。

# 9. 登录窗口乱码


现象描述：启动ClassIn登录界面后，登录窗口中的所有文字均显示为乱码。


右键单击ClassIn图标。


选择“以管理员身份运行”。


# 10. 报错提示：无权限


现象描述：在ClassIn登录界面选择左上角的“二维码”登录方式，使用手机扫描二维码时提示“无权限”。


解决方案：请将ClassIn客户端升级至最新版，按照下方步骤进行操作


使用账号密码登录ClassIn客户端。


完成登录后，进行客户端升级操作。


如果无法正常升级，建议卸载当前版本的ClassIn客户端，然后访问官网：www.eeo.cn，下载并安装最新版的ClassIn客户端。


# 11. 打开ClassIn，不显示任何内容


现象描述：在手机或平板上启动ClassIn应用程序后，界面仅显示白屏，没有任何内容加载。


关闭设备电源，进行重启。


重启后，再次运行ClassIn应用程序。

# 12. 报错提示：获取Web服务器失败


现象描述：在使用手机或平板登录ClassIn时，系统提示“获取web服务器失败”。


检查网络连接：请确保您的设备已连接至互联网。您可以尝试在手机或平板上打开任意网站，确认网络连接是否正常。


重启设备：尝试关机并重新启动您的手机或平板。


靠近路由器：将您的手机或平板尽量靠近无线路由器，以确保信号强度。


重启路由器：关闭路由器电源，等待几秒钟后重新接通电源并启动路由器。


切换网络：断开设备的无线网络连接，尝试使用手机自带的移动数据网络。如果使用的是平板，可以尝试连接手机分享的5G热点。


检查联网权限：确保您的手机或平板已允许ClassIn应用访问网络。


# 13. 报错提示：无法获取服务器地址


现象描述：使用手机或平板登录ClassIn时提示“无法获取服务器地址”。


检查网络连接：确保您的网络正常运行。您可以尝试打开任意网站，以验证网络连接是否正常。


重启设备：尝试重启您的手机或平板，以排除临时性故障。


靠近路由器：移动设备至路由器附近，以确保信号强度。


重启路由器：断开路由器电源，然后重新接通电源以重启路由器。


切换网络：断开当前的无线网络，改用手机自带的移动数据网络。如果您使用平板，可以尝试连接手机分享的5G热点


# 15. 报错提示：暂时无法登录，请稍后再试


现象描述：在手机或平板上登录ClassIn时，系统提示“无法获取服务器地址”。


检查网络连接：请确认您的设备是否已连接到互联网。您可以通过打开任意网站来验证网络连接的正常性。


切换网络：断开当前的无线网络连接，改用手机自带的移动数据网络。如果您使用的是平板设备，可以尝试连接手机共享的5G热点网络。


# 16. 报错提示：请关闭代理后重试


现象描述：在使用手机或平板登录ClassIn时，系统提示“请关闭代理后再试”。


首先，请检查您的手机或平板设备是否启用了VPN功能。


如果已启用VPN，请先将其关闭，然后再次尝试登录ClassIn。


若关闭VPN后问题仍然存在，请联系ClassIn客服以获得进一步的帮助。


# 19. 报错提示：时间限额


现象描述：用户在苹果手机或平板上打开ClassIn登录界面后，系统提示“时间限额”。


解决方案：此提示是由于用户在苹果手机或平板的设置中为ClassIn应用启用了时间限制。用户可以按照下方步骤进行操作


在提示页面的最下方点击“忽略限额”。


选择“今日忽略限额”，即可正常进入ClassIn应用。


另外，用户也可以通过以下方法彻底关闭时间限制：


打开手机或平板的“设置”。


进入“屏幕使用时间”选项。


选择“App限额”。


找到ClassIn应用，关闭其限额功能。
