# 【云盘】常见问题


# 1. 报错提示：上传失败，请稍后重试


现象描述：在尝试将本地课件上传到ClassIn云盘时，上传过程无法顺利完成，上传到一半时失败，并提示“上传失败，请稍后再试”。


解决方案：尝试下方任意一种方法以解决此问题


重启路由器：上传过程中，如果网络出现不稳定的情况可能会导致上传失败。用户可以尝试重启路由器，即先将路由器断电，然后重新接通电源并启动路由器。


更换网络连接：如果用户使用的是笔记本电脑，可以尝试切换到有线网络，或者使用手机的5G热点进行连接，以确保网络的稳定性。

# 2. 课件显示叹号


现象描述：在将课件上传至ClassIn云盘后，课件图标上出现红色“叹号”。


解决方案：红色“叹号”表示课件转换失败，未能成功转换的课件将无法在教室中展示。请参考“课件制作标准”对本地课件进行修改，然后重新上传至ClassIn云盘。如果修改后重新上传的课件仍然显示红色“叹号”，请通过以下方式联系ClassIn客服团队解决问题：1. 在售后服务群中联系ClassIn小助手、2. 拨打客服电话：400-077-1585。


# 3. 云盘中不显示课件


现象描述：在教室中打开“云盘”工具后，云盘内不显示任何课件且没有任何提示信息。


重新加载：用户可以点击教室云盘窗口右上角的“重新加载”按钮，尝试刷新页面。


清理缓存：关闭教室，清理ClassIn缓存，然后重新进入教室。


切换服务器：关闭教室并退出ClassIn，重新登录ClassIn。在登录窗口右下角点击“更多设置”，选择Auto下方前三个服务器中的任意一个进行登录。
