# 清理缓存


### 适用人员


所有人


### 适用场景


在使用ClassIn时，有时可能会遇到以下问题：


打开作业或测验栏目页面时，页面显示为空白，无任何内容。


在教室中打开课件后，显示为全黑或全白，无法显示课件的原有内容。


在教室中打开云盘工具后，一直提示“加载中”，无法正常显示云盘中的课件。


针对上述任意一种情况，用户可以尝试使用软件设置中的“清理缓存”功能来解决问题。


### 具体操作步骤


#### 电脑


点开左下角 头像


进入 文件传输


点击 清理缓存
