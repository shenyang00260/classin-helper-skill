# 【麦克风】常见问题


# 1. 说话没有声音（麦克风显示为灰色）


现象描述：当用户在教室台上发言时，其他人无法听到用户的声音。如果用户使用电脑上课，可以看到摄像头左下角的麦克风图标呈现为灰色。如果用户使用手机或平板上课，摄像头下方的音频指示条会呈现为灰色。


解决方案：这是由于用户自行禁用了麦克风。请用户在教室设置中重新启用麦克风即可解决问题。

# 2. 说话没有声音（麦克风显示为红色）


现象描述：学生在教室台上发言时，其他人听不到其声音。如果学生使用电脑上课，其摄像头左下角的麦克风图标显示为红色；如果学生使用手机或平板上课，其摄像头下方的横条显示为红色。


解决方案：此情况是由于教师将学生的麦克风设为静音。教师可以在花名册中或通过学生摄像头下方的快捷键将学生的麦克风解除静音。

# 3. 说话没有声音（麦克风显示为绿色，说话时没有声音波动）


现象描述：用户在教室中打开了麦克风，但其他人听不到用户的声音。使用电脑上课的用户在发言时发现摄像头左下角的麦克风图标没有任何声音波动；使用手机或平板上课的用户在发言时发现摄像头下方的音频指示条没有任何声音波动。


解决方案：


Windows电脑用户，请尝试下方任意一种方法以解决此问题：


开启自动调整麦克风音量：在教室设置中启用“自动调整麦克风音量”功能。


切换麦克风设备：如果连接了多个麦克风设备，在教室设置中尝试切换使用其他麦克风。


硬件修复：在教室设置中使用“硬件修复”功能，修复麦克风。


麦克风隐私设置：右键单击桌面右下角的“喇叭”图标，选择“打开声音设置”，进入“麦克风隐私设置”，勾选“允许应用访问麦克风”。


关闭空间音效：右键单击桌面右下角的“喇叭”图标，关闭“空间音效”。


取消独占控制：右键单击桌面右下角的“喇叭”图标，选择“声音”，进入“录制”，右键单击正在使用的麦克风，选择“属性”，进入“高级”，取消勾选“允许应用程序独占控制该设备”。


切换麦克风模式：在ClassIn设置中，将麦克风模式切换为“传统模式”。


苹果电脑用户，请尝试下方任意一种方法以解决此问题：


授权使用麦克风：在电脑设置中授权ClassIn使用麦克风，完成授权后重启电脑。如果已经授权过，可尝试重新授权并重启电脑。


手机/平板用户，请尝试下方任意一种方法以解决此问题：


重启设备：关机重启手机或平板。


使用耳机：尝试佩戴耳机（如果已经佩戴耳机，可尝试拔掉耳机）。


# 4. 打开麦克风后，麦克风自动禁用


现象描述：用户在教室中打开麦克风后，麦克风设置会自动切换回禁用状态。


解决方案：尝试下方任意一种方法以解决此问题


修复硬件：用户可以尝试通过教室设置中的“修复硬件”功能来修复麦克风驱动。


更新声卡驱动：如果问题仍未解决，建议用户联系电脑的售后，并安装最新版本的声卡驱动程序。


# 5. 说话有杂音


现象描述：当老师或学生在教室中讲话时，其他人听到的声音中可能会出现杂音。这些杂音类型可能包括：滋滋声、嘈杂声、电流声等。


保持环境安静：让产生杂音的用户在较为安静的房间中进行上课。


开启降噪模式：在教室设置中，开启“麦克风降噪”功能。


使用耳机：建议产生杂音的用户尝试使用耳麦或耳机。


避开干扰：将手机等电子设备尽量远离电脑和音频设备，以避免电流声的干扰。


# 6. 说话声音模糊


现象描述：教师或学生在台上发言时，声音不清晰。


更换设备：临时更换一个麦克风设备，确保其正常工作。


使用耳机：如果未佩戴耳麦，可以尝试佩戴耳麦，以提高声音的清晰度和传输效果。
