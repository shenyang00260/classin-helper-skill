# 【录课】常见问题


# 1. 云端录课中断


现象描述：在使用电脑上课时，教师视角中教室右上角的云端录课状态图标呈现为红色，表示录课中断。


解决方案：为确保录课过程顺利进行，请注意以下几点：


避免插拔音频设备：请勿在录课过程中插拔任何音频设备（包括耳机、耳麦、音响、蓝牙耳机）。特别是蓝牙耳机，由于其连接稳定性较差，可能会导致录课中断。


关闭防护类程序：退出电脑中的所有杀毒软件和防护类程序，例如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等，以免干扰录课服务的正常运行。


关闭防火墙：关闭电脑中的防火墙设置，以确保其不会阻碍录课服务的正常运作。


优化网络连接：如果您使用的是笔记本电脑，请尝试连接有线网络。 或者连接手机分享的5G热点。


联系小助手：如果以上方法均无效，请在ClassIn售后群中联系小助手，或拨打售后客服电话：400-077-1585，寻求进一步的技术支持。


# 2. 开始录课后，教室窗口变小


现象描述：教师在使用ClassIn进行录课时，每当录制开始后，教室窗口会自动缩小，并且无法调整窗口大小。


解决方案：教师按照下方步骤进行操作


打开ClassIn客户端，进入设置选项。


在设置菜单中找到并选择“录制画面范围”。


勾选“当前尺寸教室窗口”选项。


保存设置后，重新进入教室并开始录课。


# 3. 开始录课时，提示：该功能需要开启录屏权限


现象描述：教师在使用Mac OS 10.15及以上系统的苹果电脑进行录课时，系统提示需要开启录屏权限。


解决方案：尝试下方任意一种方法以解决此问题


设置录课范围：进入ClassIn设置，在“录制画面范围”选项中选择“当前尺寸教室窗口”，然后重新进入教室进行录课。


开启录课权限：在Mac电脑的“系统偏好设置”-“安全性与隐私”-“屏幕录制”中，勾选并授权ClassIn进行屏幕录制。

# 4. 老师或学生可以下载录课回放视频吗？


现象描述：老师或学生可以下载录课回放视频吗？


解决方案：具有后台管理下载权限的教师可以下载课堂回放视频，而所有学生都无法下载这些视频。
