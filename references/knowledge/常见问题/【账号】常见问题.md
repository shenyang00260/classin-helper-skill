# 【账号】常见问题


# 1. 提示：无权限


现象描述：在注册ClassIn账号时，点击“获取验证码”按钮后收到提示信息：“无权限”。


解决方案：此问题通常是由于用户电脑的系统时间不准确导致的。请按照以下步骤重新设置电脑的系统时间，然后再次尝试获取验证码：


右键单击电脑右下方的“时间”。


进入“调整日期/时间”。


打开“自动设置时间”和“自动设置时区”


重新启动电脑以确保时间设置生效。


重新尝试注册ClassIn账号并获取验证码。


# 2. 账号多人使用


问：一个ClassIn账号是否可以支持多名学生同时使用进行上课？


答：使用同一个ClassIn账号进行多名学生同时上课会带来一些限制。因为一个账号在同一时间只能在一个教室内活动，如果该账号已经在某一设备上进入了教室，其他设备使用该账号尝试进入不同教室时，将导致原先设备被迫退出。因此，如果多名学生需要使用同一账号进行上课，必须确保他们的上课时间不发生冲突，以避免无法正常进入课堂的情况。


# 3. 国外注册ClassIn


问：国外用户能否使用国外手机号注册ClassIn账号？


答：国外用户可以使用其国外手机号注册ClassIn账号。只需确保该手机号能够正常接收验证码短信，即可完成注册。
