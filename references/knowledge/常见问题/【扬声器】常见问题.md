# 【扬声器】常见问题


# 1. 听不到声音（没有声音）


现象描述：用户无法听到教室中的任何声音，包括音视频课件声音和其他人的说话声。


解决方案：


Windows电脑用户，请尝试下方任意一种方法以解决此问题：


调高教室设置中的扬声器音量：检查并调高教室应用程序中的扬声器音量设置。


调高电脑系统中的扬声器音量：通过系统音量控制面板调高电脑的扬声器音量。


修复声卡驱动：在教室设置中使用“修复硬件”功能修复可能存在的问题。


切换声音输出通道：如果使用笔记本连接了外部显示器，点击桌面右下角的“喇叭”图标，切换到正确的声音输出通道。


苹果电脑用户，请尝试下方任意一种方法以解决此问题：


重启电脑：直接重启Mac电脑，音视频功能将自动恢复正常。


切换声音输出通道：如果使用笔记本连接了外部显示器，点击桌面右上角的“喇叭”图标，切换到正确的声音输出通道。


手机/平板用户，请尝试下方任意一种方法以解决此问题：


调整音量键：使用设备侧面的音量按键调高音量。


临时使用耳机：尝试佩戴耳机或拔掉已经连接的耳机，检查声音是否恢复。


# 2. 教室中声音小


现象描述：用户在教室中听到其他人说话的声音非常小或完全听不清楚。


电脑用户，请尝试下方任意一种方法以解决此问题：


调整教室设置：检查并提高教室设置中的扬声器音量。


调整系统音量：检查并提高电脑系统中的扬声器音量。


调整音量按键：使用手机或平板侧面的实体音量按键来增加音量。


使用耳机：尝试临时佩戴耳机。如果已经佩戴耳机，可以尝试拔掉耳机，以排除耳机问题。


# 3. 教室中有回音（回声）


现象描述：用户在教室中听到回音


解决方案：当教室中有多名学生在台上发言时，教师可以采取以下步骤来排除回音源：教师要求所有学生暂时保持沉默，然后教师说一段话。如果此时教师听到了自己的回音，可以观察各位学生的麦克风状态，寻找出现波动的麦克风。确认产生回音的学生后，可以在该学生不发言时将其麦克风静音，以防止回音干扰。对于产生回音的学生，尝试下方任意一种方法以解决此问题。


拉开设备距离：如果学生使用的是音响和麦克风上课，可以让其调整设备位置，使麦克风和音响朝向相反，并增加两者之间的距离。


佩戴耳机：建议产生回音的学生佩戴耳机上课，这样可以有效避免回音问题。
