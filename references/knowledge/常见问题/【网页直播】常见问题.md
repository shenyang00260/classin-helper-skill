# 【网页直播】常见问题


# 1. 提示：老师正在赶来的路上


现象描述：用户在网页上观看ClassIn直播时，画面和声音突然中断，并显示提示语：“老师正在赶来的路上，请稍等。”


解决方案：尝试下方任意一种方法以解决此问题


如果授课教师尚未进入教室，请耐心等待教师进入教室并开始授课后，观看者可以按F5刷新网页。


如果授课教师已进入教室但尚未开始授课，请联系教务人员，让其通知授课教师在教室中的录课倒计时提示中点击“立即录课”。


如果因授课教师网络不稳定或设备问题（如插拔扬声器）导致录课中断，请查看录课状态，并在ClassIn售后服务群中联系小助手，或拨打客服电话：400-077-1585。

# 2. 提示：课节未公开


现象描述：用户在浏览器中打开ClassIn直播链接后，未能看到任何画面和声音，页面提示“课节未公开，不可观看”。


解决方案：该问题是由于课堂未开启“网页直播”权限所致。教师在创建课堂时需勾选“网页直播”功能，以便通过浏览器访问的用户能够观看。如果课堂已经创建完毕，也可在开课前20分钟以上，由具有建课权限的教师修改课堂设置，勾选“网页直播”功能。

# 3. 提示：老师暂时离开教室


现象描述：用户在网页上观看ClassIn直播时，突然出现画面和声音中断的情况，并且网页提示“老师暂时离开教室，请稍等”。


解决方案：该问题可能是由于授课教师在上课过程中手动退出教室或网络不稳定导致的掉线。请耐心等待授课教师重新进入教室并开始授课后，用户可以按F5键刷新网页以继续观看直播。

# 4. 观看网页直播时，直播画面全黑，但可以听到声音


现象描述：用户通过网页链接观看课堂直播时，只能听到教室内的声音，看不到画面。


解决方案：按照下方步骤进行操作


授课教师先暂时离开教室；


打开ClassIn设置，依次进入“系统设置”-“高级设置”，在“录课编码”选项中勾选“传统模式”；


授课教师重新进入教室并开始录课；


观看直播的用户刷新网页以重新加载直播内容。
