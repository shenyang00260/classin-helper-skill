# 【课表】常见问题


# 1. 找不到教室（没课、 不显示课）


现象描述：学生登录ClassIn后，无法找到已加入的教室。


解决方案：尝试下方任意一种方法以解决此问题


核对账号：请点击ClassIn左上角的“头像”图标，进入个人信息页面，核对当前登录的ClassIn账号是否与教师安排课程时使用的账号一致。（手机或平板用户请点击左上角头像，在“账号与安全”栏目中核对登录账号是否一致。）


清除数据文件：如果确认账号登录无误，请在ClassIn设置中选择“清除数据文件”功能进行操作。请注意，消息列表中仅显示24小时内即将开始的课程，若课程开始时间超过24小时，请在左侧“课程表”栏目中按日期或班级进行查找。（手机或平板用户请在ClassIn设置中选择“重新同步信息”功能。）


# 2. 消息列表一直显示“收取中...”


现象描述：登录ClassIn后，ClassIn主页的消息列表无法加载任何班级和课堂，消息列表中持续显示“收取中”。


解决方案：


电脑用户，请尝试下方任意一种方法以解决此问题：


重新登录：关闭ClassIn应用程序，然后重新登录。


等待片刻：网络连接可能存在短暂延迟，请稍等片刻再试。


关闭杀毒软件和防护程序：退出电脑中的杀毒软件和防护类程序（例如：360安全卫士、360杀毒软件、腾讯电脑管家、2345安全卫士、微软电脑管家、联想电脑管家等。）


关闭防火墙：暂时关闭电脑中的防火墙，确保网络连接不受限制。


重启路由器：断开路由器电源，然后重新连电启动路由器。


尝试其他网络连接：如果您使用的是笔记本电脑，可以尝试连接有线网络或通过手机分享的5G热点进行连接。


手机/平板用户，请尝试下方任意一种方法以解决此问题：


重启设备：重启手机或平板设备，然后重新登录ClassIn。


靠近路由器：将手机或平板设备靠近路由器，以增强网络信号。


切换网络：临时连接其他网络。例如，如果当前使用的是WIFI网络，可以断开WIFI，改用手机自身的移动网络。

# 3. 上课时间错误


现象描述：在ClassIn中显示的课堂开始时间与教师实际安排的时间不一致。


解决方案：请将电脑的本地时区设置为当前所在地区的正确时区。


# 4. Classin课表能同步到手机日历中吗？


问：ClassIn的课堂时间能否同步到手机日历中？


答：目前，ClassIn中的课堂时间无法直接同步到手机日历。


# 5. 学生未上课，是否能看回放视频？


问：学生未能上课，未进入教室，课后是否可以观看课堂回放视频？


答：可以的。即使学生未能进入教室上课，课后仍可观看课堂回放视频。前提条件是：学生必须是该课程的正式学生，并且该课程已启用云端录课功能，同时授课教师在课时内进行了正常录课。
