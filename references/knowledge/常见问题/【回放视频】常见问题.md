# 【回放视频】常见问题


# 1. 回放视频分段


现象描述：在观看某课堂回放视频时，视频被分割成多个片段，每个片段包含特定的课堂内容。用户需要依次观看这些片段才能获取完整的课程体验。


解决方案：


避免中途退出教室：授课教师在上课期间如果中途退出教室，会导致录课过程中断。重新进入教室并开始录制会导致回放视频被分割，影响学生的观看体验。因此，教师在上课期间应尽量避免退出教室，以确保录课的连续性。


优化网络稳定性：上课期间网络信号不稳定会导致课程录制过程中断，从而引起视频分割。这是因为在重新连接网络的过程中，录制系统可能需要重新启动，导致视频被分成多个片段。建议教师在课前优化网络环境，确保上课期间网络连接的稳定性。


控制录制时长：当录制视频时长达到2小时后，系统会自动分段生成回放视频。教师可根据课程内容合理安排每个录制片段的时长，避免单个录制片段过长而被系统自动分段。


已生成的录课视频无法进行更改。为确保下次录制时效果更佳，请参考以上建议。

# 2. 提示：课节未公开，不可观看


现象描述：用户通过网页链接观看课堂回放时，页面显示“课节未公开，不可观看”的提示，导致无法正常播放回放内容。


解决方案：此问题通常是由于用户未开启观看网页回放的权限所致。请按照以下步骤在“后台管理系统”中进行设置：


登录网站：www.eeo.cn


进入“课节管理”模块


找到并打开“网页直播权限”选项

# 3. 提示：无回放视频


现象描述：用户通过客户端观看课堂回放时，无法正常播放视频内容，并出现提示“无回放视频，请联系你上课的学校”。


录课失败：如果老师由于网络或其他原因未能成功录制课程，请联系ClassIn售后服务团队。您可以通过售后服务群联系小助手，或拨打客服电话：400-077-1585。


后台权限设置：如果老师已经成功录课但仍无法观看回放，请检查后台设置是否允许客户端观看回放。请按以下步骤操作：


登录后台管理系统（www.eeo.cn）


进入“学校设置”菜单


查找并开启“课程回放”选项


课程未实际进行：如果老师没有按计划上课，导致没有回放视频，请直接联系授课老师了解具体情况。


# 4. 提示：该视频正在转码中


现象描述：用户在客户端观看课堂回放时，无法正常播放回放视频，并收到提示信息“该视频正在转码中，请稍后再试”。


解决方案：课堂结束后，视频需要经过一段时间的转码处理，通常约半小时左右才能生成回放。因此，出现该提示是因为视频尚未完成转码。请用户耐心等待片刻，再尝试播放回放视频。


# 5. 回放中，PPT字体重叠


现象描述：用户在观看课堂回放时，发现回放画面中教室内展示的PPT出现重叠现象。


解决方案：该问题通常是由于老师在授课过程中将课件拖拽到教室边缘之外所引起的。为避免此类问题，建议老师在授课时尽量避免将课件拖拽至教室边缘之外的区域。

解决方案：该问题可能是由于用户当前网络信号不稳定所致。建议用户优化网络连接后再尝试观看。

# 7. 软件中，没有回放按钮


现象描述：在软件课程日历中，已结束的课节没有显示课堂回放的播放按钮。


解决方案：出现此问题的原因是当前“课堂（课节）”未启用录课功能，因此无法生成可供回放的视频，自然也就没有回放按钮。为避免再次出现此问题，排课老师需要在排课时选择启用“云端录课”功能，并勾选任意一种录课模式。完成这些设置后，课程结束后系统将自动生成回放视频，您就能看到播放回放的按钮了。


# 8. 回放视频画面显示不完整


现象描述：用户在观看回放视频时，教室画面仅显示一半，无法完整呈现。


教师在教室内进行云端录课时，请确保教室页面完全显示在主显示器的桌面范围内，避免页面部分超出桌面显示区域。


如果教师的电脑连接了多个显示器，请确保教室页面始终停留在主显示器上，不要将其拖拽至其他显示器，以确保录制画面的完整性。

# 9. 回放画面全黑，但可以听到声音


现象描述：用户在查看回放视频时，画面显示为全黑，但可以听到录制的声音。


如果用户的电脑连接了多个显示器，请确保教室窗口未被拖拽到其他显示器上，以避免显示问题。


在录制课程时，若需要使用屏幕共享功能，建议尽量避免本地播放PPT，以防止出现画面黑屏的情况。

# 10. 回放视频中，录到了桌面内容


现象描述：在查看录制回放时，发现录制内容不仅包含教室画面，还包括了电脑桌面的内容。


解决方案：为了避免录制到与课堂无关的画面，建议录课教师在录课前进行以下设置


打开classin客户端，点击左下角的"设置"按钮。


进入"系统设置"页面，选择"教室设置"选项。


在"录制教室画面范围"栏目下，勾选【当前尺寸教室窗口】。


请注意：如果在录制过程中使用了屏幕共享功能，录制内容将会包含整个电脑桌面。


# 11. 回放视频中，录到了其他软件声音


现象描述：用户在观看课堂回放时，发现视频中录制了教室外的所有声音，例如视频网站的声音和通讯软件的消息提示音等。


解决方案：为了避免录制到与课堂无关的声音，建议录课教师在录课前进行以下设置


打开ClassIn客户端。


点击左下角的“设置”按钮，进入系统设置界面。


选择“高级设置”选项。


在“录课模式”栏目下，勾选“软件声音”选项。


# 12. 回放视频画面模糊


现象描述：用户在观看课堂回放视频时，发现录制的画面质量不佳，呈现模糊状。


解决方案：录课教师在开始录课前，打开ClassIn客户端，并点击左下角的“设置”。进入“系统设置”页面后，选择“高级设置”选项。在“录制课程画幅大小”栏目下，勾选“全高清”选项。此操作将显著提升后续录制课程的画面清晰度。


# 13. 提示：很遗憾，视频中断


现象描述：用户在观看回放视频时，出现无法正常播放的提示信息：“很遗憾，视频中断，请稍等……”。


录课老师在上课时应避免将教室窗口最小化，并确保其他窗口不会完全遮挡住教室窗口。


如果录课老师上课时未最小化教室窗口，请临时在软件设置中的【录制画面范围】选项下，勾选【整个电脑桌面】。此设置会将老师在桌面上的所有操作录制进去。


勾选【整个电脑桌面】后，上课时老师在桌面上的所有操作都会被录制进去。

# 15. 回放画面闪烁


现象描述：观看课堂回放时，画面偶尔会出现闪烁的情况。


录课教师在上课期间应避免将教室窗口拖拽出电脑屏幕范围。


若使用Mac触屏电脑进行授课，请避免使用【三指切换屏幕】功能；若使用Windows电脑，请避免使用【多桌面切换】功能。


# 16. 回放视频中，能听到老师声音，听不到学生声音


现象描述：在观看录课回放时，可以清晰听到老师的讲话声音，但学生的讲话声音要么听不到，要么非常微弱。然而，实际授课时学生的声音是正常的。


解决方案：出现该问题的原因是因为录课时选择的是“系统声音”作为录音源。在这种情况下，如果录课老师的系统扬声器音量较低，就会导致录制到的学生声音非常小甚至完全录不到。为了避免这个问题，录课老师在录课前进入软件设置中，将录音源切换为“软件声音”。这样可以确保录制的音频包括所有参与者的声音，从而提高回放质量。


# 17. 回放视频中，声音时有时无


现象描述：用户在观看课堂回放时，偶尔会遇到没有声音的情况，但实际上，整个授课过程中的声音都是正常的。


解决方案：此问题的出现是由于录课老师的录音模式为“系统声音”。当老师在上课期间临时关闭了系统扬声器时，系统声音录制功能将无法正常工作，导致声音未能录制成功。因此，建议老师在进入教室进行录课前，需在ClassIn客户端左下角点击“软件设置”，进入“系统设置”-“高级设置”，然后在“录课模式”中将录音方式切换为“软件声音”。这样可以确保录制过程中声音不受系统扬声器状态的影响，从而避免无声音的情况发生。


# 18. 回放视频中，画面全灰色


现象描述：用户在观看课堂回放视频时，无法正常看到上课的回放画面，呈现的是灰色屏幕。


解决方案：该问题通常是由于在录制过程中，教室屏幕和窗口切换时，Mac操作系统弹出录屏权限通知，而录课老师未及时响应，导致回放画面出现灰色屏幕。为避免此问题，录课老师在录课前，需在本地电脑的系统设置中，提前授权ClassIn进行屏幕录制。具体操作步骤如下：


打开Mac电脑的“系统偏好设置”。


选择“安全性与隐私”。


点击左侧栏中的“屏幕录制”。


在右侧应用列表中，找到并勾选ClassIn。

# 19. 提示：加载失败，刷新尝试


现象描述：用户在使用客户端观看课堂回放时，遇到无法正常播放回放视频的问题，并收到提示信息：“加载失败，请刷新尝试”。


解决方案：此问题通常由部分安卓设备不支持查看ClassIn课堂回放文件引起。建议用户可以临时使用电脑或其他兼容设备进行观看，以确保正常访问回放视频。

# 20. 回放视频中，画面全绿色


现象描述：用户在使用iPad或iPhone设备观看课堂回放时，视频画面出现绿色屏幕。


解决方案：该问题是由于iOS系统16.1版本的兼容性问题导致的。建议用户将设备的操作系统升级至最新版本，以解决此问题。

# 21. 回放视频不完整


现象描述：课堂回放视频不完整。


网络不稳定：授课教师的网络连接可能不稳定，导致录制过程出现中断。建议授课教师在下次上课前检查并优化网络连接。


插拔设备：在录课过程中插拔耳机或耳麦可能导致录制中断。请避免在录课过程中进行此类操作。


其他原因：如果上述措施无效，授课教师可在ClassIn售后群中向小助手反馈问题，由小助手协助查找并解决具体问题原因。
