# 名词注释


| ClassIn词汇 | 解释 |
| --- | --- |
| IM | 全称Instant Messaging，即时通讯区域。用户登录ClassIn后进入的主界面即为IM界面，包含个人头像、即时消息列表、待办事项、联系人、空间、课程表、黑板和设置等模块。 |
| 个人头像 | 位于ClassIn主界面左上角，点击可设置个人信息，包括头像、名称、名片分享、二维码分享、手机号绑定、邮箱绑定、密码修改、性别、出生日期、地区和个性签名等。 |
| 即时消息列表 | ClassIn主界面中的一个模块，用于接收班级和好友的消息。电脑用户通过左侧导航栏进入，手机和平板用户通过下方导航栏进入。在此列表中，可接收班级和好友的消息，进行置顶、免打扰、删除等操作。 |
| 联系人 | 用户可以在ClassIn的左侧（电脑）或下方（手机、平板）导航栏进入“联系人”模块，进行添加好友、查看班级和好友列表、处理好友申请、查看和管理好友详细信息等操作。 |
| 授权资源 | 学校授予特定教师使用的公共教学资源，如课件、试卷等。通过授权，教师可以使用标准化的教学资源，确保教学内容的统一性和质量。 |
| 台上 | 在教室中，教师通过“花名册”工具邀请学生展示摄像头的位置称为“台上”。台上的学生可以与教师和其他台上学生进行语音交流，所有教室成员都能看到其摄像头影像。 |
| 上台 | 教师在“花名册”工具中邀请学生到“台上”展示摄像头的操作。 |
| 回到台下 | 教师让学生停止展示摄像头的操作，称为“回到台下”。 |
| 台下 | 未在“台上”展示摄像头的教室成员，均处于台下听课状态。 |
| 授权 | 教师在“花名册”工具中授予学生使用教室工具的权限，包括画笔、选择/移动工具、点击工具、黑板擦、文本输入工具、截图工具和云盘等。 |
| 空白 | 用户打开ClassIn某个界面后，界面中没有显示任何内容，称为“空白”。常见情况包括打开作业、测验、云盘或课件界面后为空白。 |
| 白屏 | 用户打开ClassIn某个界面后，界面中没有显示任何内容，称为“白屏”。常见情况与空白类似。 |
| 黑屏 | 用户打开ClassIn某个界面后，界面中没有显示任何内容，只有纯黑页面，称为“黑屏”。常见情况包括打开云盘或课件界面后黑屏。 |
| 排课 | 用户在ClassIn主界面中点击“创建班级”，然后在创建好的班级中进入“课程”栏目，点击“新建”按钮，选择“课堂”进行排课。 |
| 开课 | 用户在ClassIn主界面中点击“创建班级”，然后在创建好的班级中进入“课程”栏目，点击“新建”按钮，选择“课堂”进行开课。 |
| 建课 | 用户在ClassIn主界面中点击“创建班级”，然后在创建好的班级中进入“课程”栏目，点击“新建”按钮，选择“课堂”进行建课。 |
| 视频墙 | 一种教学工具，教师通过视频墙实时监视班级中所有学生的摄像头画面，方便观察学生的上课状态。若学生未开启摄像头，教师只能看到其视频框。 |
| 在线教室、本地教室、临时教室 | 本地教室：ClassIn主界面的“黑板”功能，供用户熟悉教室功能和制作板书，无时间限制。   在线教室：ClassIn中的“课堂”，教师和学生在此上课。   临时教室：教师和学生可以随时创建，免费使用。 |
| 教学活动 | 班级中所有以教学为目的和内容的活动，包括课堂、录播课、作业、测验、打卡、答题卡、讨论和学习资料等。 |
| 多班测验 | 教师在创建测验时，在测验设置中的“班级”一栏勾选多个班级，使多个班级的学生都能参与该测验。 |
| 回音 | 教室中扬声器的声音被麦克风捕捉并传回教室，导致回音。通常是麦克风与扬声器距离太近所致。 |
| 专注模式 | 一种提升学生上课专注度的功能。开启后，电脑用户的教室界面将全屏显示，无法切换其他界面；手机和平板用户在ClassIn隐藏到后台时会自动退出教室。 |
| 修复硬件驱动 | ClassIn中的硬件修复功能，用于解决麦克风、扬声器、摄像头等设备的故障。 |
| edb | ClassIn中独有的电子板书文件格式，用于保存教师在教室中制作的板书，便于随时使用和共享。 |
| 黑板 | 主界面黑板功能：指的是本地教室，供教师和学生熟悉教室功能。   教室中的黑板：教师和学生在教室中的互动工具。   教室工具箱中的小黑板工具：用于教学过程中的作答和互动。 |
| 大屏一体机 | 一种将显示屏和计算机主机集成在一起的设备，通常用于展示、演示和信息发布。 |
| 大屏一体机与大屏模式 | 大屏一体机：集成显示屏和主机的设备。大屏模式：ClassIn教室中的操控界面模式。 |
| 课程 | 软件中班级里的“课程”模块，可添加单元主题和学习活动。   后台管理系统中的“课程”，与软件中的“班级”意思相同。 |
| 后台管理系统 | ClassIn的管理平台，教务老师可在此创建课程与课节，管理教师与学生，监控课堂数据，并进行个性化设置等。 |
| 时间单位 | 半小时=30分钟，一小时=60分钟。 |
| 流量单位 | 1GB=1024MB，1MB=1024KB，1KB=1024字节。 |
| 手机 | 一种便携式电子通信设备，支持打电话、发短信、上网等功能，常见系统包括iOS、Android和HarmonyOS。 |
| 平板 | 一种便携式电子设备，采用触摸屏和虚拟键盘，常见系统包括iOS、Android和HarmonyOS。 |
| 上课的前端 | 即ClassIn客户端。 |
| 网页回放 | 课堂结束半小时后通过分享链接观看的网页版课堂回放视频，需开启云端录课功能。 |
| 助教 | 即联席教师。 |
| 教室 | “教室”是指进行教学活动的场所，包括：正式课堂、班级临时教室、好友临时教室、本地教室 |
| 空间 | 空间，又称为云盘，是指一种提供在线存储服务的功能。通过使用空间，用户可以将本地设备中的文件上传至云端，从而实现文件的远程存储和访问。用户只需使用任意设备连接互联网，登录ClassIn软件并进入空间栏目，即可在线查看、分享、下载或编辑自己的文件。   空间的优势在于其高度可靠性和极佳的跨设备性。用户的文件数据被存储在云端服务器上，不仅保证了文件数据的完整性和安全性，还使得用户可以方便地从任意地点使用任意设备远程访问和管理文件。同时，云空间也提供了多种在线编辑工具，用户可以不用下载本地编辑软件，直接在线对文件进行编辑和处理。 总之，云空间是一种高效、便捷且安全的在线文件存储和管理方式，为用户带来了极大的便利和保障。   在ClassIn中可以从两个不同的位置进入到“空间”中，一个是ClassIn主界面导航栏位置可以进入到空间中，另一个是教室中教学工具里“空间”工具可以进入到“空间”中这两个位置的“空间”里的数据是互通的。空间的主要功能是为用户提供一个在线存储课件的地方。用户可以在空间中授权使用“资源中心”和“授权资源”的课件，并且还可以在“ClassIn题库”和“ClassIn试卷库”中自定义试卷。另外，用户还可以直接访问Teacherln。 |
| 前端 | 指的是ClassIn客户端 |
| 1v1 | 1v1指的是课堂上同时显示摄像头影像的人数，即1名教师和1名学生。ClassIn平台支持的上台人数有多种配置，包括1v1、1v2、1v3、1v4、1v5、1v6、1v7、1v8、1v9、1v10、1v11、1v12、1v13、1v14、1v15。 |
